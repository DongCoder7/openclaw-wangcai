## 一、数据中心光交换机（OCS）行业概览

### 1.1 什么是光交换机（OCS）？
| 特征 | 说明 |
|:---|:---|
| **定义** | Optical Circuit Switch，光路交换机，直接在光层进行信号交换 |
| **核心优势** | 无需光电转换（O/E/O），超低时延（ns级），超低功耗 |
| **对比电交换机** | 电交换需逐包解析、缓存排队，时延2-5μs/跳；光交换直接光路切换 |
| **核心价值** | 解决大规模AI集群内部互联的"尾延迟"问题，提升GPU利用率 |

### 1.2 OCS vs 传统电交换机
| 对比维度 | 电交换机 | 光交换机（OCS） |
|:---|:---|:---|
| **交换原理** | 电信号逐包解析、缓存转发 | 光信号直接路由，无缓冲 |
| **单跳时延** | 2-5 μs | <100 ns |
| **整网时延** | 百微秒量级（7-9跳） | 微秒量级 |
| **功耗** | 高（需光电转换） | 低（无O/E转换） |
| **适用场景** | 通用数据中心 | 大规模AI训练集群 |
| **代表厂商** | Cisco、Arista、华为 | 谷歌（自研）、华为、光迅科技 |

> "当大模型参数突破万亿、GPU集群规模迈向万卡，只有把交换平面搬到光层，让连接关系随作业流量实时重构，才能把GPU利用率从60%提升到90%以上。"
> ——DCS-W OCS技术白皮书

### 1.3 市场规模与增长
| 指标 | 数据 | 来源 |
|:---|:---|:---|
| 2025年全球OCS市场规模 | 7.8亿美元 | C&C产业研究 |

## 二、液冷需求：从"可选"到"必选"

### 2.1 液冷成为刚需的驱动力
| 驱动因素 | 具体表现 |
|:---|:---|
| **芯片功耗暴增** | NVIDIA B200 TDP达1000W，B300将达1400W，Rubin将达2300W |
| **谷歌TPUv7** | 单芯片功耗980W，**100%要求液冷** |
| **单机柜功率** | 从30kW迈向50kW、100kW，风冷触及物理极限 |
| **政策要求** | 中国新建大型数据中心PUE必须<1.15（2026年起） |
| **PUE对比** | 风冷PUE 1.5+，液冷PUE 1.1-1.2 |

### 2.2 液冷市场规模预测
| 时间 | 市场规模 | 渗透率 | 来源 |
|:---|:---|:---|:---|
| 2024年 | ~14%渗透率 | AI数据中心液冷 | TrendForce |
| 2026年 | **165亿美元**（~1162亿人民币） | ~30-40% | 国海证券 |
| 2026年 | 液冷市场千亿级起点 | - | 华源证券 |
| 2028年 | 55亿美元（液冷占33%） | - | Omdia |

> "2026年将成为液冷市场的千亿产值起点"
> ——华源证券

### 2.3 光交换机与液冷的协同
| 协同点 | 说明 |
|:---|:---|
| **光模块液冷** | 800G/1.6T光模块功耗18W+，需液冷散热 |
| **交换机液冷** | Arista、Cisco推出液冷光交换机，降低系统功耗5-10% |
| **OCS液冷** | 华为DC808光交换机，浸没式液冷降低PUE至1.04 |
| **CPO+液冷** | 共封装光学（CPO）与液冷技术结合，是未来主流 |

## 四、核心投资逻辑

### 4.1 液冷投资主线
```
AI芯片功耗暴增 → 液冷从可选变必选 → 2026年千亿市场启动
    ↓
重点关注：
1. 已切入NVIDIA、谷歌、华为等头部客户供应链的厂商
2. 技术壁垒高、单品价值量大的环节（微通道冷板、CDU）
3. 从部件供应商升级为全栈系统解决方案商的企业
```

### 4.2 OCS投资逻辑
```
大规模AI集群（万卡级） → 电交换时延瓶颈 → OCS全光交换需求
    ↓
重点关注：
1. 光通信核心器件（MEMS微镜、硅光芯片）供应商
2. 与谷歌、华为等OCS先行者有合作的厂商
3. 光模块向OCS配套转型的龙头
```

## 五、风险提示

| 风险类型 | 说明 |
|:---|:---|
| **技术路线风险** | 冷板式 vs 浸没式，若技术切换快于预期，现有产品价值量假设偏差 |
| **客户验证风险** | 海外大客户验证/导入节奏低于预期（尤其英维克的Google/NVIDIA订单） |
| **竞争加剧** | 液冷赛道玩家增多，价格战风险 |
| **光交换机落地进度** | OCS技术仍在早期，大规模商用进度不确定 |
