# 美股复盘SOP（标准操作程序）

> 版本：v1.0  
> 适用：每日美股收盘后复盘  
> 时间：北京时间每日21:00-22:00（美股收盘后）

---

## 一、复盘流程

```
Step 1: 大盘扫描（5分钟）
    ↓
Step 2: 板块分析（10分钟）
    ↓
Step 3: 中概股检查（5分钟）
    ↓
Step 4: 关键事件梳理（5分钟）
    ↓
Step 5: 对A股影响评估（10分钟）
    ↓
Step 6: 输出复盘报告（5分钟）
```

---

## 二、详细检查清单

### Step 1: 大盘扫描

**必看指数**：
- [ ] 道琼斯指数（趋势判断）
- [ ] 标普500（整体情绪）
- [ ] 纳斯达克（科技股强弱）
- [ ] 罗素2000（小盘股情绪）
- [ ] VIX恐慌指数（风险偏好）

**记录模板**：
```
大盘表现：
- 道指：XXX点（+/-X%）
- 标普：XXX点（+/-X%）
- 纳指：XXX点（+/-X%）
- 成交额：放大/萎缩/正常
- 市场情绪：恐慌/贪婪/中性
```

### Step 2: 板块分析

**核心板块**（必须跟踪）：

| 板块 | 代表ETF | 必看标的 | 关键指标 |
|:-----|:--------|:---------|:---------|
| **科技** | XLK | AAPL/MSFT/GOOGL | 收入增长 |
| **AI算力** | SMH | NVDA/AMD/AVGO | 数据中心收入 |
| **半导体** | SOXX | NVDA/AMD/INTC | 订单/库存 |
| **中概互联** | KWEB | BABA/PDD/JD | 业绩/政策 |
| **新能源** | TAN/ICLN | ENPH/SEDG | 装机量 |
| **生物医药** | XBI | LLY/NVO/REGN | 管线进展 |

**记录模板**：
```
板块表现：
- 科技：领涨/领跌，原因
- AI算力：英伟达+/-X%，催化剂
- 半导体：设备/材料/设计分化
- 中概：资金流入/流出情况
```

### Step 3: 中概股检查

**核心持仓**（如有）：
- [ ] 阿里（BABA）- 电商+云
- [ ] 拼多多（PDD）- 电商+Temu
- [ ] 京东（JD）- 电商+物流
- [ ] 蔚来/小鹏/理想 - 电动车
- [ ] 贝壳（BEKE）- 地产服务

**检查项**：
- [ ] 涨跌幅 vs 纳指
- [ ] 成交量变化
- [ ] 盘后交易情况
- [ ] 是否有新闻/公告

### Step 4: 关键事件梳理

**宏观经济**：
- [ ] 美联储官员讲话
- [ ] 经济数据发布（CPI/PPI/非农）
- [ ] 美债收益率变化（10Y/2Y）
- [ ] 美元指数（DXY）

**公司事件**：
- [ ] 财报发布（重点关注）
- [ ] 并购/分拆消息
- [ ] 产品发布
- [ ] 管理层变动

**地缘政治**：
- [ ] 中美关系动态
- [ ] 俄乌冲突进展
- [ ] 中东局势

### Step 5: 对A股影响评估

**传导路径**：
```
美股表现 → 中概股 → 港股ADR → A股情绪
```

**关键映射**：
| 美股板块 | A股映射 | 影响方向 |
|:-----|:--------|:---------|
| 英伟达大涨 | 中际旭创/新易盛 | 高开概率高 |
| 特斯拉大涨 | 宁德时代/比亚迪 | 新能源情绪好 |
| 中概互联涨 | 阿里/腾讯/美团 | 港股科技高开 |
| 半导体涨 | 北方华创/中微公司 | 半导体高开 |

**预判模板**：
```
对A股影响：
- 大盘情绪：正面/负面/中性
- 科技板块：高开X%概率
- 新能源板块：高开X%概率
- 中概映射：港股科技高开X%概率
- 风险点：XXX可能低开
```

## 三、关键财报跟踪

### 财报季日历（重点关注）

| 公司 | 财报日期 | 预期营收 | 关键看点 | 对A股影响 |
|:-----|:---------|:---------|:---------|:----------|
| 英伟达 | 每月中旬 | $38-40B | 数据中心增长 | AI算力板块 |
| 微软 | 每月末 | $60-65B | Azure增长 | AI应用板块 |
| 苹果 | 每月末 | $90-95B | iPhone销量 | 消费电子 |
| 特斯拉 | 每月初 | $25-28B | 交付量/毛利率 | 新能源板块 |

### 财报后应对

**超预期**：
- 盘后大涨 → 次日A股相关板块高开 → 不追高

**符合预期**：
- 盘后平盘 → 次日A股正常开盘

**不及预期**：
- 盘后大跌 → 次日A股相关板块低开 → 可能是买入机会
