## 二、买入执行流程

### 2.1 买入前检查清单

**基本面检查**：
- [ ] 标的是否在自选股池
- [ ] 是否已完成研究分析
- [ ] 是否了解公司基本面
- [ ] 是否了解行业趋势

**技术面检查**：
- [ ] 当前价格是否在计划区间内
- [ ] 是否跌破关键支撑位（避免抄底过早）
- [ ] 成交量是否正常
- [ ] 是否有重大技术信号

**仓位检查**：
- [ ] 买入后单股仓位是否超标（<8%）
- [ ] 买入后总仓位是否超标（<90%）
- [ ] 买入后板块集中度是否超标（<25%）

**风控检查**：
- [ ] 是否已设定止损价
- [ ] 止损幅度是否合理（-8%~-15%）
- [ ] 风险收益比是否>1:3

### 2.2 买入时机选择

**最佳买入时机**：
- ✅ 开盘30分钟后（9:30-10:00情绪稳定）
- ✅ 下午开盘前（13:00-13:30）
- ✅ 尾盘30分钟（14:30-15:00）

**避免买入时机**：
- ❌ 开盘集合竞价（价格波动大）
- ❌ 盘中突发利好追高
- ❌ 临近收盘最后5分钟

### 2.3 买入方式

**市价买入 vs 限价买入**：

| 方式 | 适用场景 | 优点 | 缺点 |
|:-----|:---------|:-----|:-----|
| **市价** | 急需成交/流动性好 | 成交快 | 价格不确定 |
| **限价** | 有明确目标价/不急 | 价格可控 | 可能不成交 |

**建议**：
- 流动性好的大盘股：限价买入
- 小盘股/急涨：市价买入
- 分批建仓：限价挂单

### 2.4 分批建仓策略

**首次建仓**：
- 仓位：目标仓位的50%
- 价格：目标买入区间中位

**二次加仓**：
- 触发：股价跌5%且基本面未恶化
- 仓位：目标仓位的30%

**三次加仓**：
- 触发：股价跌10%且基本面未恶化
- 仓位：目标仓位的20%

**停止加仓**：
- 股价跌15%：停止加仓，检查基本面
- 基本面恶化：停止加仓，准备止损

## 三、卖出执行流程

### 3.1 卖出类型

| 类型 | 触发条件 | 执行方式 | 优先级 |
|:-----|:---------|:---------|:------:|
| **止损卖出** | 触及止损价 | 市价立即卖出 | ⭐⭐⭐⭐⭐ |
| **止盈卖出** | 达到目标价 | 限价分批卖出 | ⭐⭐⭐⭐ |
| **调仓卖出** | 板块轮动/风格切换 | 择时卖出 | ⭐⭐⭐ |
| **减仓卖出** | 仓位过重/回撤控制 | 择时卖出 | ⭐⭐⭐ |

### 3.2 止损卖出

**硬性规则**：
- 触及止损价，立即市价卖出
- 不犹豫，不幻想反弹
- 不补仓摊低成本
- 止损后24小时内不重新买入

**止损执行检查**：
- [ ] 是否触及预设止损价？
- [ ] 是否为正常波动？（大盘同步下跌可暂缓）
- [ ] 基本面是否恶化？（恶化立即卖出）
- [ ] 是否为流动性问题？（小盘股开盘卖）

### 3.3 止盈卖出

**分批止盈策略**：
```
盈利+20% → 卖出1/3，止损线上移至保本
盈利+30% → 再卖出1/3，止损线上移至+20%
盈利+50% → 清仓或保留1/3观察
```

**移动止盈**：
- 最高价回撤10%：卖出1/2
- 最高价回撤15%：清仓

### 3.4 卖出时机选择

**最佳卖出时机**：
- ✅ 开盘冲高（9:30-10:00）
- ✅ 板块轮动时（强势板块切换）
- ✅ 尾盘最后30分钟（14:30-15:00）

**避免卖出时机**：
- ❌ 开盘恐慌性下跌（等反弹）
- ❌ 盘中突发利空（等情绪稳定）
- ❌ 涨停板（等开板）

## 四、盘中监控

### 4.1 关键时点监控

**9:30-10:00**：
- 观察开盘情绪
- 确认持仓表现
- 执行买入/卖出计划

**10:00-11:30**：
- 监控板块轮动
- 监控资金流向
- 调整交易计划

**13:00-14:30**：
- 下午开盘观察
- 确认是否执行下午交易

**14:30-15:00**：
- 尾盘决策时间
- 执行尾盘交易
- 确认当日交易完成

### 4.2 实时监控清单

**持仓监控**：
- [ ] 是否触及止损价
- [ ] 是否达到止盈价
- [ ] 是否有异常波动（>±7%）
- [ ] 是否有重大新闻

**计划监控**：
- [ ] 观察标的是否触及买入区间
- [ ] 是否有新机会出现
- [ ] 是否有风险信号

**大盘监控**：
- [ ] 大盘是否大跌（>-1%）
- [ ] 板块是否切换
- [ ] 北向资金流向

## 五、尾盘操作

### 5.1 尾盘决策（14:30-14:50）

**必须决策事项**：
- [ ] 当日未完成的买入/卖出计划是否执行
- [ ] 是否有新的买入/卖出机会
- [ ] 持仓是否需要调整
- [ ] 明日计划是否需要更新

**尾盘买入场景**：
- 标的全天在计划区间内，未买入
- 尾盘出现跳水，价格更有吸引力
- 大盘尾盘企稳，确认趋势

**尾盘卖出场景**：
- 触及止损价，必须卖出
- 板块尾盘走弱，提前减仓
- 锁定当日利润

### 5.2 收盘前检查（14:50-15:00）

- [ ] 所有计划交易是否完成
- [ ] 持仓是否符合风控要求
- [ ] 资金/仓位是否正确
- [ ] 是否有隔夜风险（如停牌）

## 七、交易纪律

### 7.1 铁律（必须遵守）

1. **计划你的交易，交易你的计划**
   - 没有计划，不交易
   - 计划外交易，必须记录原因

2. **不追高，不抄底**
   - 错过比做错好
   - 等待回调再买入

3. **严格止损**
   - 触及止损，立即执行
   - 不犹豫，不幻想

4. **分批建仓，分批止盈**
   - 不一次性满仓
   - 不一次性清仓

5. **控制仓位**
   - 单股<8%
   - 单板块<25%
   - 总仓位<90%

### 7.2 禁止行为

- ❌ 盘中临时决策买入未研究标的
- ❌ 听消息/跟风买入
- ❌ 借钱/融资满仓
- ❌ 情绪化交易（报复性交易）
- ❌ 频繁交易（日内T+0）
- ❌ 死扛不止损
