# 盘中监控SOP（标准操作程序）

> 版本：v1.0  
> 适用：每日盘中监控与策略建议  
> 核心理念：只监控、建议，不执行交易

---

## 零、交易日检查（前置步骤）

**在执行每日监控前，必须先检查是否为交易日**：

| 检查项 | 方法 | 结果处理 |
|:-------|:-----|:---------|
| **是否周末** | 检查星期（周六/周日） | 非交易日，跳过监控 |
| **是否法定节假日** | 查询交易所休市日历 | 非交易日，跳过监控 |
| **是否特殊休市** | 台风/疫情等特殊情况 | 非交易日，跳过监控 |

**2025-2026年A股休市日历（关键节点）**：

| 节日 | 2025年休市 | 2026年休市 |
|:-----|:-----------|:-----------|
| 元旦 | 1月1日 | 1月1-2日 |
| 春节 | 1月28日-2月4日 | **2月16-23日** |
| 清明 | 4月4-6日 | 4月4-6日 |
| 劳动节 | 5月1-5日 | 5月1-5日 |
| 端午 | 5月31日-6月2日 | 6月19-21日 |
| 中秋/国庆 | 10月1-8日 | 9月26日-10月7日 |

**春节期间特别注意**：
- 春节前最后一个交易日：通常是除夕前1-2天
- 春节后首个交易日：通常是正月初七左右
- 休市期间：关注美股/港股/大宗商品走势

**非交易日任务**：
- [ ] 整理本周监控日志
- [ ] 复盘本周策略建议
- [ ] 关注外围市场（美股/港股）
- [ ] 准备下周监控计划

## 一、监控时段与重点

### 1.1 早盘监控（9:30-10:30）

**开盘情绪判断**：
- [ ] 上证指数高开/低开幅度（<-1%警惕，>+1%观察）
- [ ] 涨跌停家数（判断情绪极端程度）
- [ ] 北向资金开盘流向（前15分钟）
- [ ] 美股期货表现（影响开盘情绪）

**板块开盘表现**：
| 板块 | 龙头股 | 开盘涨跌 | 资金流入 | 策略建议 |
|:-----|:-------|:--------:|:--------:|:---------|
| AI算力 | 中际旭创 | +/-X% | +/-XX亿 | 持有/观望/减仓 |
| 算力租赁 | 润泽科技 | +/-X% | +/-XX亿 | 持有/观望/减仓 |
| 半导体 | 北方华创 | +/-X% | +/-XX亿 | 持有/观望/减仓 |
| 高股息 | 长江电力 | +/-X% | +/-XX亿 | 持有/观望/加仓 |

**策略建议模板**：
```
【早盘监控】HH:MM
大盘情绪：恐慌/中性/贪婪
强势板块：XXX、XXX
弱势板块：XXX、XXX

策略建议：
- 若持仓上涨>3%：建议减仓1/3锁定利润
- 若持仓下跌>3%：观察基本面，暂不建议加仓
- 若大盘跌破XXX点：建议减仓至70%
```

### 1.2 午盘监控（10:30-11:30）

**板块轮动观察**：
- [ ] 早盘强势板块是否持续
- [ ] 是否有新板块崛起
- [ ] 资金是否从成长切向价值（或反向）

**资金流向监控**：
- [ ] 北向资金累计流向
- [ ] 主力资金净流入板块TOP5
- [ ] 主力资金净流出板块TOP5

**个股异动监控**：
| 标的 | 异动类型 | 涨跌幅 | 成交量 | 策略建议 |
|:-----|:---------|:------:|:------:|:---------|
| XXX | 快速拉升 | +X% | 放大 | 建议观望，不追高 |
| XXX | 放量下跌 | -X% | 放大 | 建议检查基本面 |
| XXX | 突破前高 | +X% | 放大 | 建议持有，设移动止损 |

### 1.3 下午监控（13:00-14:30）

**午后开盘观察**：
- [ ] 港股午盘表现（影响A股）
- [ ] 是否有突发新闻/公告
- [ ] 早盘强势股是否回调

**持仓健康度检查**：
- [ ] 各持仓标的涨跌情况
- [ ] 是否触及预设止损价
- [ ] 是否达到预设止盈价
- [ ] 仓位是否超标

### 1.4 尾盘监控（14:30-15:00）

**尾盘策略建议（重点）**：

| 场景 | 条件 | 策略建议 |
|:-----|:-----|:---------|
| **强势尾盘** | 大盘连续上涨，放量 | 建议减仓至80%，锁定利润 |
| **弱势尾盘** | 大盘连续下跌，缩量 | 建议观望，等待次日开盘 |
| **跳水尾盘** | 尾盘快速下跌>1% | 建议减仓至70%，避险 |
| **抢筹尾盘** | 尾盘快速拉升>1% | 建议观望，不追高 |

**次日预案建议**：
```
【尾盘建议】14:45
当前仓位：XX%

明日预案：
- 若高开X%：建议减仓XX%
- 若低开X%：建议加仓XX%
- 关键观察：XXX财报/XXX政策
```

## 三、策略建议类型

### 3.1 买入建议（只建议，不执行）

**触发条件**：
- 标的跌至预设买入区间
- 大盘恐慌性下跌（>-2%）
- 板块轮动至该板块

**建议模板**：
```
【买入建议】HH:MM
标的：XXX（代码）
当前价：XX元（已进入买入区间XX-XX元）
建议仓位：X%（单股上限X%）
买入理由：XXX
风险提示：XXX

⚠️ 注意：此建议仅供参考，请结合自身情况决策
```

### 3.2 卖出建议（只建议，不执行）

**触发条件**：
- 触及预设止损价
- 达到预设止盈价
- 板块轮动离开
- 基本面恶化信号

**建议模板**：
```
【卖出建议】HH:MM
标的：XXX（代码）
当前价：XX元
建议操作：减仓XX%/清仓
理由：触及止损价/达到目标价/板块轮动

⚠️ 注意：止损纪律需自行执行
```

### 3.3 调仓建议

**触发条件**：
- 板块轮动信号明确
- 某板块估值泡沫化
- 某板块进入黄金坑

**建议模板**：
```
【调仓建议】HH:MM
建议操作：从XXX板块调出X%，调入XXX板块
调出理由：XXX
调入理由：XXX
风险提示：XXX
```

### 3.4 仓位调整建议

**触发条件**：
- 大盘连续上涨/下跌
- 组合回撤超预警线
- 出现系统性风险信号

**建议模板**：
```
【仓位建议】HH:MM
当前仓位：XX%
建议仓位：XX%
调整理由：大盘XXX，建议XXX
操作方向：加仓/减仓
```

## 四、关键时点策略建议

### 4.1 9:30-10:00（开盘期）

**高开场景**（上证>+0.5%）：
- 持仓上涨>3%：建议减仓1/3锁定利润
- 观察是否有板块涨停潮
- 不追高买入

**低开场景**（上证<-0.5%）：
- 持仓触及止损：建议严格止损
- 观察北向资金流向
- 等待情绪稳定再决策

### 4.2 10:00-11:30（博弈期）

**板块轮动观察**：
- 若AI算力持续强势：建议持有，不追高
- 若高股息走强：建议关注风格切换
- 若小盘股普涨：建议关注中证1000

### 4.3 13:00-14:30（午后期）

**午后开盘**：
- 观察港股午盘影响
- 观察是否有突发消息
- 调整午后策略

**持仓健康度**：
- 检查各标的盈亏状况
- 检查是否触及关键价位
- 准备尾盘策略

### 4.4 14:30-15:00（尾盘期）【重点】

**尾盘30分钟策略建议**：

| 时间 | 监控重点 | 策略建议 |
|:-----|:---------|:---------|
| 14:30 | 大盘方向选择 | 确认当日趋势 |
| 14:40 | 持仓标的尾盘表现 | 是否触及止损/止盈 |
| 14:50 | 次日预案制定 | 给出明日策略建议 |

**尾盘建议输出模板**：

## 六、禁止事项

**本SOP只提供监控和建议，不涉及**：
- ❌ 直接执行买入/卖出操作
- ❌ 代替用户做决策
- ❌ 保证收益或承担损失
- ❌ 实时交易信号推送

**用户责任**：
- 自行决策是否采纳建议
- 自行执行交易操作
- 自行承担投资风险
- 结合自身情况调整策略
