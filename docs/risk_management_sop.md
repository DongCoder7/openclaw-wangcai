# 风险管理SOP（标准操作程序）

> 版本：v1.0  
> 适用：投资组合全流程风险控制  
> 核心理念：先求生存，再求发展

---

## 一、风险管理框架

### 1.1 风险层级

```
Level 1: 系统性风险（无法分散）
    ↓ 应对：仓位控制+对冲工具
    
Level 2: 行业风险（可部分分散）
    ↓ 应对：行业分散+轮动
    
Level 3: 个股风险（可分散）
    ↓ 应对：个股止损+仓位控制
    
Level 4: 操作风险（人为失误）
    ↓ 应对：纪律执行+复核机制
```

### 1.2 风险类型清单

| 风险类型 | 描述 | 发生概率 | 影响程度 | 应对措施 |
|:---------|:-----|:--------:|:--------:|:---------|
| **系统性风险** | 大盘暴跌/金融危机 | 低 | 极高 | 减仓+对冲 |
| **行业风险** | 政策打击/行业衰退 | 中 | 高 | 行业分散 |
| **个股风险** | 业绩暴雷/黑天鹅 | 中 | 中 | 止损+分散 |
| **流动性风险** | 无法及时成交 | 低 | 中 | 避免小盘股 |
| **汇率风险** | 外币资产贬值 | 中 | 中 | 对冲或回避 |
| **操作风险** | 下单错误/系统故障 | 低 | 中 | 复核机制 |
| **心理风险** | 恐慌/贪婪导致失误 | 高 | 中 | 纪律执行 |

## 二、仓位风险管理

### 2.1 总仓位控制

| 市场环境 | 总仓位上限 | 现金比例 | 说明 |
|:---------|:----------:|:--------:|:-----|
| **牛市** | 90% | 10% | 积极进攻 |
| **震荡市** | 70% | 30% | 均衡配置 |
| **熊市** | 50% | 50% | 防御为主 |
| **恐慌市** | 30% | 70% | 等待机会 |

**调整触发条件**：
- 大盘跌破60日均线 → 减仓至70%
- 大盘跌破120日均线 → 减仓至50%
- 大盘跌破250日均线 → 减仓至30%

### 2.2 个股仓位控制

| 个股类型 | 单股上限 | 说明 |
|:---------|:--------:|:-----|
| **核心持仓** | 8% | 高确定性龙头 |
| **进攻持仓** | 5% | 高成长弹性 |
| **卫星持仓** | 3% | 主题博弈 |
| **试错持仓** | 2% | 新标的试水 |

**同板块仓位上限**：
- 单一板块不超过25%
- 相关板块（如AI算力+算力租赁）合计不超过35%

### 2.3 仓位再平衡

**定期再平衡**：
- 频率：每季度一次
- 触发：单一标的偏离目标±5%
- 操作：卖出超配，买入低配

**动态再平衡**：
- 触发：个股涨幅>50% → 减仓1/3
- 触发：个股跌幅>20% → 检查基本面
- 触发：板块轮动信号 → 调仓至强势板块

## 三、止损管理

### 3.1 止损类型

| 类型 | 触发条件 | 止损幅度 | 适用场景 |
|:-----|:---------|:--------:|:---------|
| **技术止损** | 跌破关键支撑位 | -8%~-15% | 趋势交易 |
| **时间止损** | 买入后X天未涨 | -5%~-10% | 事件驱动 |
| **基本面止损** | 基本面恶化 | 立即 | 所有持仓 |
| **情绪止损** | 板块恐慌性抛售 | -10%~-20% | 系统性风险 |

### 3.2 止损执行纪律

**硬性规则**：
- 止损单必须在买入时设定
- 触及止损无条件执行
- 不补仓摊低成本（除非是计划中的分批建仓）
- 止损后24小时内不重新买入

**止损检查清单**：
- [ ] 是否触及预设止损价？
- [ ] 基本面是否恶化？
- [ ] 行业是否发生系统性风险？
- [ ] 是否为正常波动（大盘同步下跌）？

### 3.3 移动止损

**盈利后上调止损**：
```
盈利10% → 止损线上移至成本价（保本）
盈利20% → 止损线上移至+10%
盈利30% → 止损线上移至+20%
盈利50% → 减仓1/3，剩余设移动止损
```

## 四、回撤管理

### 4.1 回撤分级应对

| 回撤幅度 | 风险等级 | 应对措施 |
|:---------|:--------:|:---------|
| 5%~10% | 🟡 正常波动 | 维持仓位，观察 |
| 10%~15% | 🟠 警惕 | 减仓至70%，检查持仓 |
| 15%~20% | 🔴 风险 | 减仓至50%，清仓弱势股 |
| >20% | ⚫ 危机 | 减仓至30%，保留现金 |

### 4.2 最大回撤控制

**组合层面**：
- 目标最大回撤：15%
- 预警线：-10%（减仓信号）
- 清仓线：-20%（无条件减仓）

**单日回撤控制**：
- 单日跌幅>3%：次日减仓10%
- 单日跌幅>5%：次日减仓20%
- 连续3日下跌：减仓至50%

## 五、个股风险管理

### 5.1 买入前检查清单

**基本面检查**：
- [ ] ROE > 10%（过去3年平均）
- [ ] 负债率 < 60%
- [ ] 经营现金流为正
- [ ] 无重大诉讼/担保
- [ ] 审计意见为标准无保留

**估值检查**：
- [ ] PE < 行业平均或历史50%分位
- [ ] PB < 行业平均或历史50%分位
- [ ] 市值 > 100亿（流动性考虑）

**技术面检查**：
- [ ] 不在历史高位（PE < 90%分位）
- [ ] 未跌破关键支撑位
- [ ] 成交量正常（非缩量阴跌）

### 5.2 持仓监控清单

**每日检查**：
- [ ] 是否触及止损线
- [ ] 是否有重大利空新闻
- [ ] 板块是否发生系统性风险
- [ ] 是否触发止盈条件

**每周检查**：
- [ ] 基本面是否有变化
- [ ] 估值是否进入危险区域
- [ ] 资金是否持续流出
- [ ] 技术面是否恶化

**每月检查**：
- [ ] 财报预告是否miss
- [ ] 行业景气度是否变化
- [ ] 竞争格局是否恶化
- [ ] 政策风险是否增加

## 六、系统性风险应对

### 6.1 预警信号

| 信号 | 说明 | 应对 |
|:-----|:-----|:-----|
| 大盘跌破年线 | 长期趋势转弱 | 减仓至50% |
| VIX > 30 | 市场恐慌 | 减仓+买入对冲 |
| 信用利差扩大 | 经济衰退风险 | 减仓债券 |
| 美债倒挂 | 衰退预期 | 减仓成长股 |
| 地缘政治升级 | 黑天鹅风险 | 减仓+买黄金 |

### 6.2 对冲工具

| 工具 | 适用场景 | 对冲比例 |
|:-----|:---------|:--------:|
| **国债** | 股市下跌 | 股债配比6:4 |
| **黄金** | 地缘风险 | 5-10%仓位 |
| **期权PUT** | 个股保护 | 覆盖核心持仓 |
| **股指期货** | 大盘对冲 | 50%仓位对冲 |
| **反向ETF** | 短期对冲 | 10-20%仓位 |

### 6.3 危机应对预案

**情景1：大盘单日暴跌>5%**
- 立即减仓至50%
- 清仓所有T2/T3板块
- 保留T0板块核心持仓

**情景2：持仓股连续跌停**
- 能卖出则立即卖出
- 无法卖出则开盘挂跌停价
- 反思买入逻辑错误

**情景3：黑天鹅事件（战争/金融危机）**
- 清仓所有权益资产
- 100%现金或黄金
- 等待危机结束信号

## 七、心理风险管理

### 7.1 常见心理陷阱

| 陷阱 | 表现 | 应对 |
|:-----|:-----|:-----|
| **追涨杀跌** | 高点买入，低点卖出 | 逆向操作，设定纪律 |
| **锚定效应** | 以买入价锚定决策 | 关注价值而非成本 |
| **过度自信** | 重仓单一标的 | 强制分散 |
| **损失厌恶** | 该止损时不止损 | 机械执行止损 |
| **羊群效应** | 跟随市场热点 | 独立思考 |
| **确认偏误** | 只看支持自己的信息 | 寻找反面证据 |

### 7.2 交易纪律

**买入纪律**：
- 只买研究过的标的
- 严格执行买入区间
- 分批建仓，不一次性满仓
- 单股不超过8%

**卖出纪律**：
- 触及止损无条件卖出
- 达到目标价分批止盈
- 基本面恶化立即卖出
- 不受情绪影响

**空仓纪律**：
- 看不懂时保持空仓
- 市场恐慌时敢于空仓
- 等待明确机会再入场

## 十、风险管理的铁律

1. **永远不要亏大钱** - 单笔亏损<5%，组合回撤<20%
2. **永远保留现金** - 至少10%现金应对突发
3. **永远不要满仓** - 满仓=赌博
4. **永远不要死扛** - 止损是保命
5. **永远不要追高** - 追高=接飞刀
6. **永远不要抄底** - 左侧布局需耐心
7. **永远不要All in** - 分散是免费午餐
8. **永远不要情绪化** - 纪律重于一切
