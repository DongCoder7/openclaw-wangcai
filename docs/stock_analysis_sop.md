## 二、各步骤详细操作指南

### Step 1: 基础数据收集

**数据清单**：

| 数据类型 | 具体内容 | 来源 |
|:---------|:---------|:-----|
| **基本信息** | 股票代码、名称、行业、市值 | 行情软件 |
| **实时行情** | 股价、PE、PB、换手率 | 腾讯/新浪API |
| **财务三表** | 利润表、资产负债表、现金流量表 | AKShare |
| **财务指标** | ROE、毛利率、净利率、负债率 | 财务摘要 |
| **历史数据** | 近3-5年关键财务数据 | 财报对比 |
| **行业数据** | 行业平均PE、毛利率、增长率 | 行业报告 |

**检查项**：
- [ ] 数据是否为最新财报
- [ ] 数据单位是否统一（亿元/万元）
- [ ] 是否获取至少3年历史数据

---

### Step 2: 财报初步分析（杜邦分析）

**核心公式**：
```
ROE = 净利润率 × 总资产周转率 × 权益乘数
```

**分析模板**：

| 指标 | 公式 | 当前值 | 行业平均 | 判断 |
|:-----|:-----|:------:|:--------:|:-----|
| ROE | 净利润/净资产 | __% | 10-15% | 优秀/一般/差 |
| 净利润率 | 净利润/营收 | __% | 5-15% | 高/中/低 |
| 总资产周转率 | 营收/总资产 | __次 | 0.5-1.0 | 快/慢 |
| 权益乘数 | 总资产/净资产 | __ | 1.5-2.5 | 高杠杆/低杠杆 |

**商业模式判断**：
- **茅台模式**：高利润率 + 低周转 + 低杠杆
- **沃尔玛模式**：低利润率 + 高周转 + 低杠杆
- **银行模式**：低利润率 + 低周转 + 高杠杆
- **恶化模式**：低利润率 + 低周转 + 高杠杆 ⚠️

| 应收账款周转天数 | 持续增加 | __ 🟡 |
| 资产减值/营收 | >5% | 6% 🔴 |

**历史趋势对比**：
```

重点关注：连续恶化趋势
```

### Step 4: 深度追问"为什么"（核心步骤）

**追问方法论**：

```
现象 → 为什么？ → 原因A → 为什么？ → 原因B → 为什么？ → 根本原因
```

**追问模板**（以江钨装备为例）：

#### 第1层追问：为什么毛利率低？
- 营业成本 > 营业收入
- 成本倒挂

#### 第2层追问：为什么成本会倒挂？
- 原材料（钨精矿）涨价
- 产品（硬质合金）跌价
- 售价无法覆盖成本

#### 第3层追问：为什么售价无法覆盖成本？
- 下游制造业需求疲软
- 行业内卷，价格战
- 产品同质化，缺乏定价权

#### 根本原因：
- **滞胀困境**：上游通胀 + 下游通缩
- **产业链位置劣势**：纯加工企业，两头受气
- **产品结构问题**：低端产品占比高

**追问检查清单**：
- [ ] 至少追问3层
- [ ] 每个异常指标都追问
- [ ] 区分直接原因 vs 根本原因
- [ ] 区分内因 vs 外因

---

### Step 5: 行业与竞争分析

**五力模型分析**：

| 力量 | 强度 | 对公司影响 |
|:-----|:----:|:-----------|
| 供应商议价能力 | 高/中/低 | 成本压力？ |
| 购买者议价能力 | 高/中/低 | 定价权？ |
| 新进入者威胁 | 高/中/低 | 竞争加剧？ |
| 替代品威胁 | 高/中/低 | 被颠覆风险？ |
| 行业内竞争 | 高/中/低 | 价格战？ |

**行业周期判断**：
- 导入期 / 成长期 / 成熟期 / 衰退期
- 当前处于周期的什么位置？

**产业链位置**：
- 上游（资源） / 中游（制造） / 下游（品牌）
- 微笑曲线位置（高价值端 vs 低价值端）

---

### Step 6: 重大事件/收购分析

**事件分析框架**：

| 维度 | 分析内容 |
|:-----|:---------|
| **事件概况** | 时间、金额、交易对手、标的 |
| **战略意图** | 大股东动机、战略目的 |
| **财务影响** | 收入、利润、资产负债改善预测 |
| **协同效应** | 成本降低、收入提升、产业链整合 |
| **情景分析** | 乐观/中性/悲观三种情景 |
| **关键风险** | 执行风险、市场风险、政策风险 |

**收购财务影响测算模板**：

```
收购前：
- 营收：X亿元
- 净利润：Y亿元
- 毛利率：Z%

收购后（假设注入资产盈利A亿元）：
- 营收：X + ΔX亿元
- 净利润：Y + A亿元
- 毛利率：(原业务毛利 + 新业务毛利) / 总营收

改善幅度：
- 营收增长：ΔX/X
- 利润增长：A/|Y|（扭亏为盈倍数）
- 毛利率提升：Z_new - Z_old
```

## 四、常见分析陷阱与避免方法

| 陷阱 | 表现 | 避免方法 |
|:-----|:-----|:---------|
| **就数论数** | 只罗列数据，不分析原因 | 强制追问"为什么" |
| ** confirmation bias** | 只找支持观点的数据 | 主动寻找反面证据 |
| **忽视行业** | 脱离行业背景分析 | 先做行业分析，再分析公司 |
| **过度乐观** | 高估重组/转型成功率 | 做情景分析，考虑悲观情况 |
| **静态分析** | 只看当前，不看趋势 | 强制做3-5年历史对比 |
| **忽视现金流** | 只看利润，不看现金 | 强制分析现金流质量 |

### 8.2 技术分析深度化

**不只是单根K线，要识别形态组合**：

| 形态组合 | 识别要点 | 市场含义 | 案例 |
|:---------|:---------|:---------|:-----|
| **早晨之星** | 长阴→十字星→长阳 | 底部反转 | 江钨2025年12月-2026年1月 |
| **上升三法** | 长阳→3小阴→长阳 | 上涨中继 | 江钨2026年1月-2月 |
| **W底** | 双底+颈线突破 | 底部确立 | 江钨2025年12月形态 |
| **红三兵** | 连续3根阳线 | 强势上涨 | 资金连续流入时 |
| **一字板突破** | 跳空涨停无影线 | 极强/极弱 | 重组股启动时 |

**量价配合分析**：
```
不仅仅是"放量上涨/缩量下跌"
要分析：
- 涨停是否放量？（无量涨停=筹码锁定好）
- 回调是否缩量？（缩量回调=洗盘而非出货）
- 历史天量/地量的位置意义
- 换手率突然放大/缩小的含义
```

**关键价位标记**：
- 前期高点/低点
- 整数关口（如10元、20元）
- 均线支撑/压力（MA20/MA60/MA120）
- 颈线位（形态突破关键）

---

### 8.3 资金流向分析

**龙虎榜解读框架**：

| 维度 | 分析内容 | 信号判断 |
|:-----|:---------|:---------|
| **买卖席位** | 机构 vs 游资 vs 散户 | 机构主导=中长期 / 游资=短线 |
| **买卖金额** | 买方力量 vs 卖方力量 | 买>卖=积极 / 卖>买=出货 |
| **换手率** | 高换手(>10%) vs 低换手(<3%) | 高换手=分歧大 / 低换手=一致 |
| **振幅** | 日内波动幅度 | >15%=剧烈震荡，可能洗盘 |

**主力资金追踪**：
```
连续3日主力净流入：
- 第1天：+4556万 → 大单抢筹，启动信号
- 第2天：+3201万 → 持续流入，洗盘结束  
- 第3天：+5000万+ → 涨停封单，资金疯抢
→ 形成"红三兵"资金形态
```

**筹码分布分析**：
- 筹码集中度（集中=主力控盘 / 分散=散户主导）
- 筹码峰位置（当前价上方=套牢盘压力大）
- 筹码移动方向（向上移动=主力吸筹）

### 8.4 叙事化表达技巧

**"看图说话"方法论**：

1. **设定故事线**
   ```
   不是："该公司2023年营收增长20%"
   而是："这是一个'凤凰涅槃'的故事——从煤炭到钨业的转型"
   ```

2. **关键转折点标注**
   ```
   时间轴叙事：
   - 2025年9月：¥6.87 阶段低点，跌入谷底
   - 2025年12月：¥6.40 二次探底，绝望中孕育希望
   - 2026年1月9日：更名"江钨装备"，涨停启动，黎明破晓
   - 2026年2月12日：¥14.31 一字涨停，凤凰涅槃
   ```

3. **情感化用词**
   - 积极："涅槃重生"、"王者归来"、"困境反转"
   - 警示："高位杀跌"、"断崖式下跌"、"血本无归"
   - 中性："震荡整理"、"蓄势待发"、"静观其变"

4. **类比与比喻**
   - "空中加油"：横盘整理后继续上涨
   - "深蹲起跳"：先跌后涨
   - "筹码惜售"：涨停无人卖出
   - "资金疯抢"：放量涨停

### 8.5 策略分层精细化

**按投资者类型差异化建议**：

| 投资者类型 | 风险偏好 | 建议策略 | 仓位 | 止损 | 目标 |
|:-----------|:---------|:---------|:----:|:----:|:----:|
| **短线交易者** | 极高 | 追涨停/做T | 10-20% | -5% | +10-15% |
| **中线投资者** | 高 | 波段操作 | 30-50% | -10% | +20-30% |
| **长线价值** | 中 | 逢低布局 | 50-70% | -20% | +50%+ |
| **保守型** | 低 | 观望/回避 | 0-10% | - | - |

**分阶段策略**：

**阶段1：底部启动期**
- 特征：成交量温和放大，股价突破短期均线
- 策略：激进者试仓，保守者观望
- 仓位：20-30%

**阶段2：主升浪期**
- 特征：放量涨停，突破重要阻力位
- 策略：持有或加仓
- 仓位：50-70%

**阶段3：高位震荡期**
- 特征：放量滞涨，换手率高企
- 策略：减仓或止盈
- 仓位：降至30%以下

**阶段4：回调期**
- 特征：跌破关键支撑位，放量下跌
- 策略：止损或空仓
- 仓位：0-10%
