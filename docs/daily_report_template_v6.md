# A股收盘深度报告 - 标准模板说明

## 模板版本
**v6.0** - 2026-03-04 固化

## 报告结构（8大模块）

### 【一、外围市场环境】
- 🇺🇸 美股隔夜行情（道琼斯、纳斯达克、标普500）
- 🇭🇰 港股收盘（恒生指数、恒生科技）
- 🌍 地缘政治/外部风险（Exa搜索实时新闻）

**数据源**: 腾讯财经API + Exa全网搜索

---

### 【二、A股市场全景】
- 涨跌分布（上涨/下跌/平盘数量）
- 平均涨跌幅
- 涨跌停家数
- 两市成交额

**数据源**: 长桥API (5190+只股票)

---

### 【三、板块表现】
- 科创板、创业板、主板、北交所涨跌幅
- 行业板块强弱排序

**分析维度**: 按股票代码前缀分类统计

---

### 【四、热点新闻】
- Exa实时搜索的5类关键词：
  1. A股今日行情
  2. 美股最新走势
  3. 中东局势/地缘政治
  4. AI算力/芯片最新消息
  5. 龙虎榜/机构席位

**数据源**: Exa MCP全网语义搜索

---

### 【五、龙虎榜数据】
- 机构净买入TOP5
- 机构净卖出TOP5
- 上榜原因统计

**数据源**: Akshare (stock_lhb_detail_em)

---

### 【六、涨幅榜 TOP10】
- 当日涨幅最大的10只股票
- 代码、价格、涨跌幅

---

### 【七、跌幅榜 TOP10】
- 当日跌幅最大的10只股票
- 代码、价格、涨跌幅

---

### 【八、明日展望与操作建议】
- 📊 技术面诊断
- 💡 操作策略（仓位建议、关注方向、回避方向）
- ⚠️ 风险提示

---

## 使用方式

### 手动生成
```bash
cd /root/.openclaw/workspace
./venv_runner.sh tools/daily_market_report_template_v6.py
```

### 定时任务（Cron）
```bash
# 每日15:05自动生成
5 15 * * * cd /root/.openclaw/workspace && ./venv_runner.sh tools/daily_market_report_template_v6.py
```

### 输出文件
- 路径: `data/daily_report_YYYYMMDD.md`
- 格式: Markdown
- 编码: UTF-8

## API优先级

| 优先级 | API | 用途 |
|:---:|:---|:---|
| 1 | **长桥API** | A股全市场实时行情 |
| 2 | **腾讯财经API** | 美股/港股指数 |
| 3 | **Exa搜索** | 实时新闻/地缘政治 |
| 4 | **Akshare** | 龙虎榜数据 |
