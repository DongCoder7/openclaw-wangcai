# 第三代半导体SiC（碳化硅）产业链深度分析报告

## 一、为什么叫"第三代半导体"？

### 1.1 三代半导体的代际划分逻辑

"第三代半导体"这个称谓**并非指代技术迭代或优劣排序**，而是基于**半导体材料大规模应用带来的三次产业革命**而划分的时间阶段：

```
┌──────────────────────────────────────────────────────────────────┐
│  第一代半导体（1950s-至今）                                        │
│  代表材料：硅(Si)、锗(Ge)                                         │
│  核心应用：集成电路、CPU、存储器、光伏                            │
│  产业革命：微型计算机与信息产业的诞生                              │
│  市场占比：全球半导体产品的95%以上                                  │
├──────────────────────────────────────────────────────────────────┤
│  第二代半导体（1990s-至今）                                        │
│  代表材料：砷化镓(GaAs)、磷化铟(InP)                              │
│  核心应用：光通信激光器、射频器件、LED红光、卫星通信                │
│  产业革命：光纤通信与移动通信产业的崛起                            │
├──────────────────────────────────────────────────────────────────┤
│  第三代半导体（2000s-至今，加速期）                                 │
│  代表材料：碳化硅(SiC)、氮化镓(GaN)、氧化锌(ZnO)、金刚石            │
│  核心应用：新能源汽车、光伏储能、5G基站、数据中心电源                │
│  产业革命：能源互联网与碳中和转型                                  │
└──────────────────────────────────────────────────────────────────┘
```

### 1.2 关键认知纠偏

| 常见误解 | 正确理解 |
|:---|:---|
| "第三代比前两代先进，会全面取代" | ❌ 三代是**互补共存**关系，不是替代关系 |
| "一代比一代好" | ❌ 只是**按时间阶段划分**，各代材料特性不同、应用场景不同 |
| "硅即将被淘汰" | ❌ 硅仍占95%以上市场份额，在逻辑芯片、存储领域不可替代 |

**国外称谓对比**：国际上通常称SiC/GaN为**"宽禁带半导体"(Wide Bandgap)** 而非"第三代"。中国的"第三代"说法更强调其在产业革命中的历史定位。

### 1.3 三代材料核心参数对比

| 参数 | 第一代 Si | 第二代 GaAs | 第三代 SiC | 第三代 GaN |
|:---|:---:|:---:|:---:|:---:|
| **禁带宽度(eV)** | 1.12 | 1.43 | **3.26** | **3.44** |
| **击穿电场(MV/cm)** | 0.3 | 0.4 | **2.2** | **3.3** |
| **热导率(W/cm·K)** | 1.5 | 0.5 | **4.9** | 1.3 |
| **电子饱和速度(×10⁷cm/s)** | 1.0 | 1.0 | 2.0 | **2.7** |
| **适用功率等级** | 低-中 | 中 | **高功率** | 中-高功率 |
| **适用频率** | 低频 | 高频 | 中频 | **高频** |

> **核心结论**：SiC的核心优势是**高耐压 + 高热导率**，使其成为高功率场景（如新能源汽车主驱）的最佳选择。

## 二、SiC产业链全景图

### 2.1 产业链结构

```
┌──────────────────────────────────────────────────────────────────┐
│ 上游：衬底 + 外延片（价值占比约70%，技术壁垒最高）                  │
│ ├─ 衬底：天岳先进、天科合达、晶盛机电、三安光电                    │
│ ├─ 外延片：瀚天天成、天域半导体                                    │
│ ├─ 成本结构：衬底占器件成本47%，外延占23%                          │
│ └─ 技术瓶颈：微管缺陷、基面位错(BPD)、晶体生长速率慢                │
│                      ↓                                           │
├──────────────────────────────────────────────────────────────────┤
│ 中游：器件/模块制造（价值占比约20%）                                │
│ ├─ IDM模式：比亚迪半导体、芯联集成、斯达半导、士兰微                 │
│ ├─ 器件类型：SiC二极管 → SiC MOSFET（主流）→ SiC IGBT（研发中）      │
│ ├─ 封装：散热优化成为核心竞争力                                     │
│ └─ 技术路径：平面型 → 沟槽型 → 超结型（提升电流密度）                │
│                      ↓                                           │
├──────────────────────────────────────────────────────────────────┤
│ 下游：应用端（价值占比约10%，但决定市场空间）                        │
│ ├─ 新能源汽车（最大市场，占比~70%）                                 │
│ │   └─ 主驱逆变器、OBC车载充电机、DC-DC转换器                      │
│ ├─ 光伏储能（增速最快领域之一）                                    │
│ │   └─ 光伏逆变器、储能变流器                                       │
│ ├─ 数据中心电源（AI驱动新增量）                                     │
│ ├─ 工业电源、轨道交通、智能电网                                     │
│ └─ 消费级AR（SiC光波导镜片等前沿应用）                              │
└──────────────────────────────────────────────────────────────────┘
```

### 2.2 SiC vs GaN 分工格局

| 应用场景 | 主导材料 | 原因 |
|:---|:---|:---|
| **>650V高压场景**（主驱逆变器、电网） | **SiC** | 热导率高，散热好，可靠性高 |
| **<650V中低压高频**（手机快充、数据中心） | **GaN** | 开关速度更快，成本更低 |
| 射频/5G基站 | GaN | 高频性能更优 |

## 三、SiC技术前景分析

### 3.1 市场规模与增速

| 指标 | 2024年 | 2025年(预测) | 2030年(预测) | CAGR |
|:---|:---:|:---:|:---:|:---:|
| 全球SiC功率器件市场 | 26亿美元 | 29亿美元 | **229亿美元** | **32%** |
| 全球SiC渗透率（功率半导体） | 4.9% | 5.2% | ~25%+ | - |
| 全球SiC衬底市场 | 92亿元(RMB) | ~123亿元 | - | ~25% |
| 全球SiC外延市场 | 87亿元(RMB) | 95亿元 | - | ~10% |

> **核心判断**：SiC正处于从"高端选配件"向"主流标配"过渡的**关键转折年**（2025-2026）。

### 3.2 技术演进趋势

#### （1）尺寸升级：6英寸 → 8英寸 → 12英寸

| 尺寸 | 当前状态 | 进展 | 降本效果 |
|:---|:---|:---|:---|
| **6英寸** | 市场主流，但价格战激烈 | 国内产能过剩，价格逼近成本线 | 基准 |
| **8英寸** | 头部企业量产爬坡 | 天岳先进、晶盛机电等已量产，月产能1-1.5万片 | 理论降本**60%** |
| **12英寸** | 研发阶段 | 少数企业开始探索 | 未来方向 |

**关键数据**：
- 6英寸SiC MOS衬底价格已从年初的~5000元降至**2500-2800元**，全年降幅超**50%**
- 8英寸衬底可将器件制造成本降低**60%**（理论上），但当前良率仍需优化

#### （2）器件结构演进

```
平面型 MOSFET → 沟槽型 MOSFET → 超结型/双沟槽型
     ↑              ↑                ↑
  技术简单       导通电阻更低       电流密度更高
  可靠性高       开关损耗更小       适合高压场景
```

#### （3）成本下降曲线

| 成本驱动因素 | 当前进展 | 未来预期 |
|:---|:---|:---|
| 衬底价格下降 | 6英寸降价50%+ | 8英寸量产后再降30-40% |
| 良率提升 | 微管密度从10/cm²降至<1/cm² | 目标接近硅基水平 |
| 设备国产化 | 晶盛机电、北方华创打破Aixtron/LPE垄断 | 设备成本再降20% |
| 垂直整合 | 比亚迪等建立全链条布局 | 减少供应链成本15% |

### 3.3 下游需求驱动力

| 应用领域 | 2025年状态 | 2026-2030年展望 | 驱动强度 |
|:---|:---|:---|:---:|
| **新能源汽车800V平台** | 20万+纯电基本普及，增程式覆盖 | 向中低端车型渗透，SiC成为"核心基础设施" | ⭐⭐⭐⭐⭐ |
| **光伏储能** | 快速崛起期 | 双碳政策+装机量激增，逆变器/变流器需求爆发 | ⭐⭐⭐⭐⭐ |
| **AI数据中心电源** | 新兴应用场景 | 服务器功率攀升，SiC解决"能耗墙"问题 | ⭐⭐⭐⭐ |
| **超充桩** | 布局加速 | 480kW+超充需要SiC支撑 | ⭐⭐⭐⭐ |
| **消费级AR** | 前沿探索 | SiC光波导镜片突破玻璃限制 | ⭐⭐⭐ |

## 五、风险提示

### 🔴 高风险
| 风险 | 表现 | 应对 |
|:---|:---|:---|
| **产能过剩+价格战** | 6英寸衬底价格跌破成本线，多家企业亏损 | 关注8英寸进展，规避纯6英寸企业 |

| 风险 | 表现 | 应对 |

## 六、投资逻辑总结

> **一句话判断**：SiC正处于"从新兴材料升级为功率器件核心平台"的关键转折期，短期受价格战困扰，中长期在新能源汽车+光伏储能+AI数据中心三重驱动下，市场空间巨大。**国产替代是国内企业的核心alpha来源**。

### 投资关注点
1. **短期**（2025-2026）：关注行业洗牌进展，8英寸量产进度，价格何时企稳
2. **中期**（2026-2028）：800V平台渗透率，光伏储能放量
3. **长期**（2028-2030）：12英寸技术，新应用场景（AR、航天等）

### A股核心标的方向
| 方向 | 代表企业 | 逻辑 |
|:---|:---|:---|
| **衬底龙头** | 天岳先进、天科合达 | 技术壁垒最高，国产替代最核心环节 |
| **IDM/器件** | 比亚迪半导体(未上市)、芯联集成、斯达半导、士兰微 | 垂直整合能力，车规级突破 |
| **设备** | 晶盛机电、北方华创 | 8英寸产线建设拉动设备需求 |
| **外延** | 瀚天天成(未上市)、天域半导体(未上市) | 连接衬底与器件的关键环节 |
