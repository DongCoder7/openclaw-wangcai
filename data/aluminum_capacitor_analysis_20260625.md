## 一、产业链结构

### 1.1 产业链全景图

```
┌─────────────────────────────────────────────────────────────┐
│ 上游: 铝箔/电解液/电解纸/外壳材料                             │
│  ├─ 电极铝箔（高纯铝箔，占成本30-40%）                        │
│  ├─ 电解液（乙二醇+硼酸铵体系）                              │
│  ├─ 电解纸（隔离纸，决定ESR）                                │
│  ├─ 铝壳/橡胶塞（封装材料）                                  │
│  └─ 关键材料：高比容电极箔                                   │
│                    ↓                                         │
├─────────────────────────────────────────────────────────────┤
│ 中游: 铝电解电容器制造                                       │
│  ├─ 铝箔腐蚀 → 化成 → 卷绕/叠层                             │
│  ├─ 电解液含浸                                              │
│  ├─ 封装老化测试                                            │
│  └─ 技术壁垒：高比容电极箔、低ESR技术、长寿命技术             │
│                    ↓                                         │
├─────────────────────────────────────────────────────────────┤
│ 下游: 应用领域                                               │
│  ├─ 新能源（光伏逆变器、储能、充电桩）← 核心增长驱动          │
│  ├─ 消费电子（电源适配器、家电）                             │
│  ├─ 汽车电子（新能源车电控、车载充电）                       │
│  ├─ 工业控制（变频器、伺服系统）                             │
│  └─ 通信设备（5G基站电源、服务器）                           │
└─────────────────────────────────────────────────────────────┘
```
