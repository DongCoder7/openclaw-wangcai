### 2.3 成本结构分解（从2025年财报推导）
**推导过程**：
```
2025年：
  总营收 = 47.41亿
  总毛利 = 29.11×0.335 + 12.90×0.324 + 5.06×0.511 + 0.45×0.98
         = 9.75 + 4.18 + 2.59 + 0.44 = 16.96亿
  毛利率 = 16.96 / 47.41 = 35.8%
  净利润 = 3.76亿
  总费用 = 16.96 - 3.76 = 13.20亿
  费用率 = 13.20 / 47.41 = 27.8%

费用拆分（基于2025年财报结构）：
  研发费用 = 7.35亿（15.5%）→ 50%固定 + 50%变动 = 3.68亿固定 + 3.68亿变动
  销售费用 = 3.41亿（7.2%）→ 30%固定 + 70%变动 = 1.02亿固定 + 2.39亿变动
  管理费用 = 2.65亿（5.6%）→ 80%固定 + 20%变动 = 2.12亿固定 + 0.53亿变动
  
  固定费用（年化） = 3.68 + 1.02 + 2.12 = 6.82亿
  变动费用率 = (3.68 + 2.39 + 0.53) / 47.41 = 6.60 / 47.41 = 13.9%
```

| 成本类型 | 金额 | 占比 | 变化规律 | 说明 |
|----------|------|------|----------|------|
| 固定费用（年化） | **6.82亿** | 14.4% | 不随营收变化 | 研发+管理+销售的固定部分 |
| 变动费用率 | **13.9%** | — | 随营收线性变化 | 销售提成+部分研发+部分管理 |
| **净利润** | **3.76亿** | **7.9%** | — | 2025年实际 |

### 2.4 模型验证：2026Q1
```
2026Q1实际数据：
  营收 = 15.60亿
  毛利率 = 43.1%（已知）
  毛利 = 15.60 × 0.431 = 6.72亿

  所得税（15%） = 0.43亿
  少数股东 = 0.01亿
  计算净利润 = 2.84 - 0.43 - 0.01 = 2.40亿
  
  实际净利润 = 3.19亿
  差异 = 3.19 - 2.40 = 0.79亿
  
  差异解释：
  - 财务费用（利息收入）2026Q1为-1414万（vs 2025Q1为-1940万），差异约+526万
  - 其他收益：2197万（vs 2025Q1的1786万），差异+411万
  - 投资收益：428万（vs 2025Q1的745万），差异-317万

  - 合计差异约+806万，与模型差异0.79亿部分吻合
  - 剩余差异可能来自：费用率实际低于13.9%（Q1研发费用可能低于预期）
  - 或者固定费用Q1低于1.71亿（Q1通常是项目启动期，费用前置较少）
```

| 验证项 | 模型预测 | 实际数据 | 差异 | 原因 |
|--------|----------|----------|------|------|
| 2026Q1营收 | 15.60亿 | 15.60亿 | 0 | ✅ 完全一致 |

## 5️⃣ 2026年Q2利润预测（核心：产品拆分季度预测）

### 5.1 Q2预测逻辑
**已知2026Q1实际数据**：
- 存储芯片：10.18亿，毛利率38.9%
- 计算芯片：4.03亿，毛利率51.9%
- 模拟芯片：1.32亿，毛利率50.9%
- 总营收：15.60亿，总毛利率43.1%，净利润3.19亿

**Q2预测假设**：
1. **存储芯片**：Q2继续涨价（+5%），出货量略增（+5%）→ 营收增长约10%
   - 理由：公司调研纪要"二季度国内、海外客户陆续跟涨"，"预计二季度收入和毛利率将继续提升"
   - Q2存储营收 = 10.18 × 1.10 = 11.20亿
   - Q2存储毛利率 = 41.0%（涨价传导，从38.9%提升2.1pct）

2. **计算芯片**：KGD缺货持续，涨价约3%，出货量受限（+2%）→ 营收增长约5%
   - 理由："一季度计算芯片增长将近50%...目前看二季度会保持继续增长"
   - Q2计算营收 = 4.03 × 1.05 = 4.23亿
   - Q2计算毛利率 = 52.0%（维持高位）

3. **模拟芯片**：稳定增长（+5%）
   - Q2模拟营收 = 1.32 × 1.05 = 1.39亿
   - Q2模拟毛利率 = 52.0%（稳定）

### 5.2 Q2利润计算（中性情景）
```
Q2营收：
  存储芯片 = 11.20亿
```
