## 一、SST技术原理与"三代半导体"关联

### 1.1 什么是SST？

**固态变压器（SST, Solid State Transformer）**，又称**电力电子变压器（PET）**，是通过**电力电子技术**实现传统变压器功能的新一代智能电力设备。

**核心技术路径**：
```
传统变压器: 工频电磁感应 → 硅钢片+铜线 → 笨重被动 → 功能单一
SST:        电力电子变换 → SiC/GaN器件+高频磁材 → 智能可控 → 多端口能量路由
```

**三大颠覆性优势**：
1. **高频小型化**：基于SiC器件，频率提升至20kHz+，体积/重量减少80%+
2. **全网智能化**：毫秒级响应、双向功率控制、电能质量治理、故障限流
3. **高度模块化**：功率模块串并联灵活扩容，易于自动化量产

### 1.2 为什么叫"三代半导体"？

这里的"三代半导体"指的是**第三代半导体材料**（SiC碳化硅、GaN氮化镓），是SST的核心器件基础：

| 代际 | 材料 | 代表器件 | 适用场景 |
|:---:|:---|:---|:---|
| 第一代 | 硅(Si) | 传统IGBT、MOSFET | 低频、低功率 |
| 第二代 | 砷化镓(GaAs) | 射频器件、光电子 | 通信、光电 |
| **第三代** | **SiC/GaN** | **高压SiC MOSFET** | **SST、新能源汽车、光伏逆变器** |

**SST必须用第三代半导体的原因**：
- SiC MOSFET耐压可达10kV+，满足中压电网接入需求
- 开关频率20kHz-100kHz，是传统硅器件的1000倍
- 导通损耗降低70%，系统效率提升至98%+

## 二、产业链结构拆解

### 2.1 产业链全景图

```
┌─────────────────────────────────────────────────────────────────┐
│ 上游: 第三代半导体器件 + 高频磁材                                 │
│  ├─ SiC器件: 英飞凌、Wolfspeed、Rohm、国产（三安/斯达）           │
│  ├─ GaN器件: 纳微半导体、英诺赛克                                │
│  ├─ 高频磁材: 非晶/纳米晶磁芯（国产占全球35%份额）                │
│  └─ 价值占比: 60%+（SiC器件成本占比超60%）                        │
│                    ↓                                             │
├─────────────────────────────────────────────────────────────────┤
│ 中游: SST整机制造                                                │
│  ├─ 龙头: 金盘科技、四方股份、中国西电、为光能源                   │
│  ├─ 电力电子: 特变电工、阳光电源、科华数据                        │
│  ├─ 高频变压器: 伊戈尔、新特电气、京泉华                          │
│  └─ 价值占比: 30%                                                │
│                    ↓                                             │
├─────────────────────────────────────────────────────────────────┤
│ 下游: 应用场景                                                   │
│  ├─ AI智算中心(AIDC): 英伟达GB300强制标配 → 最大增量市场          │
│  ├─ 兆瓦级超充站: 特斯拉/蔚来/小鹏 → 电网接入核心设备             │
│  ├─ 新型电力系统: 新能源并网、微电网、虚拟电厂                     │
│  └─ 绿电直连: 高耗能企业直购绿电                                  │
└─────────────────────────────────────────────────────────────────┘
```
