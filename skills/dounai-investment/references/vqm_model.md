# VQM选股模型

## 概述

VQM模型是一个综合选股模型，结合三个维度：
- **V** - Valuation (估值)
- **Q** - Quality (质量)
- **M** - Momentum (动量)

## 评分体系

### 1. Valuation (估值) - 权重 40%

使用市盈率(PE)作为主要指标：

| PE范围 | 评分 |
|--------|------|
| 0-10 | 100 |
| 10-20 | 80 |
| 20-30 | 60 |
| 30-50 | 40 |
| 50+ | 20 |

附加考虑：
- PB < 2 加分
- 股息率 > 3% 加分

### 2. Quality (质量) - 权重 40%

使用ROE作为主要指标：

| ROE范围 | 评分 |
|---------|------|
| >25% | 100 |
| 20-25% | 80 |
| 15-20% | 60 |
| 10-15% | 40 |
| <10% | 20 |

附加考虑：
- 资产负债率 < 60% 加分
- 营收增长 > 10% 加分

### 3. Momentum (动量) - 权重 20%

使用波动率和近期表现：

| 条件 | 评分 |
|------|------|
| 波动率 < 20% | 100 |
| 近1月涨幅 > 5% | +20 |
| 近3月涨幅 > 10% | +20 |

## 选股流程

1. **初选**：过滤ST股票、停牌股票
2. **评分**：计算VQM综合评分
3. **排序**：按评分降序排列
4. **筛选**：选择Top 30作为候选
5. **风控**：过滤高风险行业

## 组合构建

- 持仓数量：10-15只
- 权重分配：等权重
- 调仓频率：每月最后一个交易日
- 再平衡：触发阈值±10%

## 风控规则

- 止损线：-8%
- 止盈线：+15%
- 单股仓位：≤15%
- 行业集中度：≤30%

## 示例

```
股票A:
- PE = 8 (评分: 100)
- ROE = 28% (评分: 100)
- 波动率 = 15% (评分: 100)

综合评分 = 100*0.4 + 100*0.4 + 100*0.2 = 100分
```
