# 知识星球抓取与汇总 Skill v2.0

## 功能

1. **自动抓取**: 从知识星球API抓取当日帖子
2. **防限流**: 超保守策略（8-15秒间隔，60秒限流等待）
3. **去重**: 全局seen_ids去重，支持断点续传
4. **结构化汇总**: 自动生成行业分布/情绪分析/观点归纳报告

## 抓取策略

### 频率控制（降低限流风险）

| 参数 | 旧值 | 新值 | 说明 |
|:---|:---:|:---:|:---|
| 最小间隔 | 5s | **8s** | 降低请求频率 |
| 最大间隔 | 10s | **15s** | 随机延迟范围 |
| 限流等待 | 30s | **60s** | 触发限流后等待 |
| 轮次冷却 | 30s | **60s** | 每轮抓取后休息 |
| 退避基数 | 5s | **10s** | 网络错误退避 |

### 限流处理

```
触发限流(code=1059):
  第1次重试: 等待60秒
  第2次重试: 等待120秒  
  第3次重试: 等待180秒
  仍然失败: 停止抓取，保存已获取数据
```

## 汇总报告结构

### 【一、数据概览】
- 抓取帖子数
- 总字数
- 平均单帖长度

### 【二、热点行业分布】
- AI/人工智能
- 半导体/芯片
- 新能源
- 军工
- 医药
- 金融
- 消费
- 通信
- 计算机
- 机器人

### 【三、市场情绪统计】
- 🟢 看多情绪（买入/增持/看好等）
- 🔴 看空情绪（卖出/减持/看空等）
- 情绪指数（正值看多，负值看空）

### 【四、政策/事件关注度】
- 政府工作报告提及次数
- 两会/政策关键词统计

### 【五、核心观点归纳】
基于高频行业自动生成:
1. AI产业观点
2. 军工板块观点
3. 新能源观点
4. 政策面解读
5. 市场情绪总结

### 【六、明日关注方向】
- 前三热点行业

### 【七、数据质量】
- 抓取状态（完整/部分）
- 文件路径
- 生成时间

## 注意事项

1. **Cookie有效期**: 知识星球Cookie约7-15天过期，需定期更新
2. **限流风险**: 即使保守策略仍可能触发限流，脚本会自动等待重试
3. **数据完整性**: 限流严重时可能无法抓取全部数据，汇总报告会标注"部分"
4. **cron时区**: 使用UTC时间，北京时间23:30对应UTC 15:30
