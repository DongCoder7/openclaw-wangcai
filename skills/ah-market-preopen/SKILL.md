# A+H股开盘前瞻

## 功能

每日开盘前生成A+H股市场分析报告：

1. **A股板块分析** - 科技/半导体、AI算力、金融、消费医药、新能源
2. **港股板块分析** - 科技、金融地产、能源资源、消费医药
3. **重点个股** - 涨幅榜/跌幅榜
4. **开盘策略建议** - 根据情景给出操作建议
5. **重点关注** - 北向资金、成交量、板块轮动、美股映射

## 使用方法

### 定时任务 (长桥API版 - 推荐)
```bash
# 每日9:15自动运行
15 9 * * * cd ~/.openclaw/workspace && source .longbridge.env && python3 skills/ah-market-preopen/scripts/generate_report_longbridge.py
```

### 手动触发 (长桥API版)
```bash
cd ~/.openclaw/workspace
source .longbridge.env
python3 skills/ah-market-preopen/scripts/generate_report_longbridge.py
```

### 环境变量配置
长桥API密钥已配置在 `~/.openclaw/workspace/.longbridge.env`
```
LONGPORT_APP_KEY=xxx
LONGPORT_APP_SECRET=xxx
LONGPORT_ACCESS_TOKEN=xxx
```

### 用户指令
- "生成A+H开盘报告"
- "盘前分析"
- "A+H市场前瞻"
- "ah-market-preopen"

## 数据源

- 长桥API（港股实时行情）
- 腾讯API（A股实时行情）
