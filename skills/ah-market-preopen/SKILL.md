# A+H股开盘前瞻分析 v2.0

## 功能

生成完整的A+H股开盘前瞻报告，包括：

1. **隔夜美股回顾** - 引用美股报告核心结论
2. **A股板块分析** - 7大板块强弱排序
3. **港股板块分析** - 6大板块强弱排序
4. **新闻驱动因子** - 技术面+新闻面双重分析
5. **开盘策略建议** - A股+港股分别策略
6. **重点个股监控** - AH涨幅/跌幅各前5

## 数据源说明（重要）

### 行情数据
| 数据类型 | 数据源 | 获取方式 |
|---------|--------|---------|
| A股行情 | 长桥API | `QuoteContext.quote()` |
| 港股行情 | 长桥API | `QuoteContext.quote()` |
| 涨跌幅计算 | 本地计算 | `(现价-昨收)/昨收 × 100%` |

### 美股回顾
| 数据类型 | 数据源 | 说明 |
|---------|--------|------|
| 美股报告 | 本地文件 | `data/us_market_daily_YYYYMMDD.md` |
| 核心结论 | 引用 | 美股报告「市场展望与总结」部分 |

### 新闻数据（多源聚合 - 已集成Exa全网搜索）
| 优先级 | 数据源 | 类型 | 说明 |
|:------:|--------|------|------|
| **P1** | **Exa全网搜索** | AI语义搜索 | `mcporter call exa.web_search_exa()` |
| P2 | 新浪财经 | API | `https://feed.mix.sina.com.cn/api/roll/get` |
| P3 | 华尔街见闻 | API | `https://api-one.wallstcn.com/apiv1/content/information-flow` |
| P4 | 第一财经 | API | `https://www.yicai.com/api/ajax/getlatest` |
| P5 | 东方财富 | API | `https://np-anotice-stock.eastmoney.com/api/security/ann` |
| P6 | Agent Reach | 工具集 | YouTube + RSS |
| - | 腾讯财经 | API | 备用（常返回空数据） |
| - | 网易财经 | 网页 | 备用（页面改版中） |

**新闻搜索配置**:
```python
# Exa搜索调用示例（已在脚本中集成）
mcporter call 'exa.web_search_exa({"query": "A股港股开盘", "numResults": 5})'
```
