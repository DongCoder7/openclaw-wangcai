# A股每日收盘深度报告 Skill v7.0

## 功能

生成完整的A股收盘分析报告，包含：

1. **外围市场环境** - 美股三大指数、港股指数
2. **A股官方指数** - 市值加权指数（上证指数/深证成指/创业板指/科创50）
3. **技术面深度分析** - MA/MACD/RSI/布林带/支撑阻力位
4. **A股市场全景** - 涨跌分布、涨跌停、成交额、等权平均涨跌幅
5. **板块深度分析** - 科创板/创业板/主板等权平均 + 领涨股
6. **热点新闻** - Exa实时搜索 + 新闻-板块关联解读
7. **龙虎榜数据** - 机构净买入/卖出TOP5
8. **个股深度分析** - 涨幅/跌幅榜 + 板块标注
9. **明日展望与操作建议** - 技术面总结、策略、风险提示

## 数据源

| 数据类型 | 数据源 | 获取方式 | 说明 |
|---------|--------|---------|------|
| A股行情 | 长桥API | `get_longbridge_api()` | 全市场5000+只股票 |
| 官方指数 | 腾讯财经API | `qt.gtimg.cn` | 市值加权（市场基准）|
| K线数据 | Tushare Pro | `pro.index_daily()` | 60日历史数据 |
| 技术指标 | 本地计算 | `pandas` | MA/MACD/RSI/布林带 |
| 龙虎榜 | Akshare | `stock_lhb_detail_em()` | 机构席位数据 |
| 新闻 | Exa MCP | `mcporter call exa` | AI语义搜索 |

## 技术指标说明

### 均线系统
- **多头排列**: MA5 > MA10 > MA20（上涨趋势）
- **空头排列**: MA5 < MA10 < MA20（下跌趋势）
- **金叉**: 短期均线上穿长期均线（买入信号）
- **死叉**: 短期均线下穿长期均线（卖出信号）

### MACD
- **金叉**: DIF上穿DEA（买入信号）
- **死叉**: DIF下穿DEA（卖出信号）
- **红柱放大**: 上涨动能增强
- **绿柱缩小**: 下跌动能减弱

### RSI
- **>80**: 严重超买（可能回调）
- **70-80**: 超买（警惕）
- **30-70**: 正常区间
- **20-30**: 超卖（关注）
- **<20**: 严重超卖（可能反弹）

### 布林带
- **突破上轨**: 相对高位（可能回落）
- **中轨附近**: 正常区间
- **跌破下轨**: 相对低位（可能反弹）
- **收口**: 波动率下降，变盘在即
- **开口**: 波动率上升，趋势加速
