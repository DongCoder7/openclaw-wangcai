# 实盘跟踪 Skill

## 功能

监控持仓组合的实时表现，基于技术分析生成次日操作计划。

## 持仓配置

持仓组合在 `tools/portfolio_tracker.py` 中配置：

```python
PORTFOLIO = {
    "300750.SZ": {"name": "宁德时代", "shares": 1000},
    "300274.SZ": {"name": "阳光电源", "shares": 1500},
    # ... 更多持仓
}
```

## 使用方法

```bash
cd /root/.openclaw/workspace
./venv_runner.sh tools/portfolio_tracker.py
```

## 输出内容

1. **持仓实时数据**: 每只股票的现价、涨跌、市值
2. **短期技术分析**: 基于short-term-analysis skill的评分
3. **板块分布**: 组合的行业配置
4. **明日计划**: 根据技术评分生成的操作建议

## 明日计划生成逻辑

| 今日涨跌 | 技术评分 | 明日计划 |
|:---|:---:|:---|
| > +5% | ≥2.0 | 持有观望，观察突破 |
| +2~5% | 1~2 | 持有，趋势良好 |
| -2~+2% | -1~1 | 观望，等待方向 |
| -5~-2% | -1~0 | 观察，关注支撑 |
| < -5% | <0 | 关注止损，考虑减仓 |

## 风险提示

- 技术分析仅供参考，不构成投资建议
- 请结合自身风险承受能力决策
- 建议设置止损位，控制单只持仓风险
