### v4.1 核心新增理念

**1. 结构优先于MACD（v4.1核心升级）**

MACD是**滞后确认指标**，结构（分型→笔→线段）可以**领先**MACD。

> 实例：2026-06-17，30F价格>55线(4044) + 结构已走出第五段上涨 + 回踩30F中轨(4093)确认，但MACD=-2.57<0。
> 
> v4.0判定：55线上方X段 ❌（机械）
> 
> v4.1判定：主涨段雏形 ✅（结构已确认，MACD负值是回踩中的暂时状态）
> 
> 后续：若MACD转正 → 正式确认主涨段；若持续为负+跌破中轨 → 降级为X段

**判定原则：**
```
结构条件（必备）:
├── 价格 > MA55 ✅
├── 结构完整（第五段上涨或第三段上涨）✅
├── 回踩N-1级别中轨/55线确认 ✅
└── MACD 暂时 < 0（回踩中）⚠️
    → 判定：主涨段雏形
    → MACD转正 → 正式确认
    → MACD持续为负+跌破中轨 → 降级为X段
```

**2. 套娃传导链（v4.1核心新增）**

套娃 = N+2级别主涨段套住N级别主涨段，N级别主涨段套住N-2级别主涨段，形成**级别嵌套**。

```
套娃传导链（典型）：

30F第五段上涨（结构已确认）
    ↓ 套娃
120F主涨段特征（价格>55线 + MACD>0）
    ↓ 套娃
双日第三段上涨（结构推演）
    ↓
双日MACD金叉（最终目标）
```

**套娃成立条件：**
1. 维持120F55线以上
2. 30F MACD转正（确认主涨段）
3. 30F第五段结构持续（不跌破30F中轨）

**套娃断裂条件：**
1. 跌破5F中轨 → 30F主涨段结束
2. 30F顶背离 + 跌破5F55线 → 120F极强解除传导链
3. 回踩30F55线 → 套娃失败，转为震荡

**3. 三路径情景推演（v4.1核心新增）**

任何分析必须给出**三条路径**，不能只给突破/跌破：

| 路径 | 名称 | 条件 | 走势 | 结果 |
|:---:|:---|:---|:---|:---|
| **A** | **理想路径** | 高开高走，结构持续，MACD转正 | 快速突破，30F第五段持续 | 双日金叉+明显斜率/幅度 |
| **B** | **震荡路径** | 不突破也不深跌，关键位附近横盘 | 4090-4115区间震荡 | 用时间换取空间，等待双日MACD金叉后再突破 |
| **C** | **风险路径** | 跌破5F中轨/5F55线，传导链启动 | 回踩30F55线(4044) | 30F主涨段结束，套娃断裂 |

> **关键认知**：路径B（震荡）不是失败，而是"正常选项"！很多情况下市场选择路径B来等待大周期指标修复。

**4. 预警/确认双体系（v4.1核心新增）**

区分**操作层面的预警线**和**战略层面的确认线**：

```
操作预警线（减仓/加小心）:
├── 5F中轨跌破 → 30F主涨段可能结束
├── 5F55线跌破(4090) → 120F极强传导链预警
├── 15F55线跌破 → 30F X段可能变异
└── 30F中轨跌破 → 30F结构破坏

战略确认线（清仓/确认）：
├── 120F55线跌破(4098) → 120F极强正式解除
├── 30F55线跌破(4044) → 30F主涨段/X段确认失败
├── 日线55线跌破 → 中期趋势转空
└── 联合支撑区跌破(4065-4074) → 无条件清仓
```

**传导链逻辑（不是直接解除）：**
```
5F55线跌破(4090) [操作预警]
    ↓ 传导链启动
15F形态恶化 → MACD转负
    ↓
30F主涨段结束/顶背离确认
    ↓
60F主涨段承压
    ↓
120F极强传导失效 [传导链中间]
    ↓
最终跌破120F55线(4098) [战略确认]
```

### Step 3: MACD稳定性分析（v4.1结构优先认知）

#### MACD六种状态定义

MACD的"强弱"不是行情强弱，而是**"稳定性"**——稳定性强弱是相对于走势而言的。

| 状态 | DIF | DEA | MACD | 含义 | 操作原则 |
|:---|:---|:---|:---|:---|:---|
| **极强** | DIF≥0 | DEA≤0 | >0 | 上涨持续性极高 | **不做空** |
| 强 | DIF>DEA>0 | | >0 | 上涨持续性高 | 做多胜率高 |
| 中性偏强 | 0>DIF>DEA | | >0 | 上涨但动能减弱 | 观察 |
| **极弱** | DIF≤0 | DEA≥0 | <0 | 下跌持续性极高 | **不做多** |
| 弱 | DIF<DEA<0 | | <0 | 下跌持续性高 | 做空胜率高 |
| 中性偏弱 | 0<DIF<DEA | | <0 | 下跌但动能减弱 | 观察 |

**核心规则：**
- 极强形态 → 会导致上涨更有持续性，这时做多胜率高
- 极弱形态 → 会导致下跌更有持续性，这时做空胜率高
- **极强形态不做空，极弱形态不做多**

#### ⚠️ v4.1 重要认知：MACD是滞后确认指标

> **"结构优先于指标，MACD是滞后确认。"**

- 结构（分型→笔→线段）可以**领先**MACD 1-3根K线
- 价格>55线 + 结构已确认 → 即使MACD暂时<0，也应判定**主涨段雏形**
- MACD转正 = **确认**，MACD持续为负 = **预警降级**
- 大周期（双日）MACD极弱，但结构可能已走第三段上涨 → 需要**前瞻性推演**

### Step 4: 主涨段判定体系（v4.1结构领先升级）

#### 主涨段判定三档

| 档位 | 条件 | 含义 | 操作 |
|:---|:---|:---|:---|
| **主涨段（正式）** | 价格>55线 + MACD>0 + 结构完整 | 确认状态 | 持仓/加仓 |
| **主涨段雏形** | 价格>55线 + 结构完整 + MACD暂时<0 | 结构已领先，等待确认 | 持仓，等待MACD转正 |
| **X段** | 价格>55线 + MACD<0 + 结构不完整/被破坏 | 主涨段失败 | 减仓/警惕 |

#### 主涨段雏形判定条件（v4.1新增）

```
结构三条件（缺一不可）：
├── 价格 > MA55 ✅
├── 结构完整：
│   ├── 第三段上涨（盘整结构）或
│   ├── 第五段上涨（趋势结构）或
│   └── 其他标准趋势结构 ✅
├── 回踩N-1级别中轨/55线确认：
│   ├── 有回踩动作
│   └── 回踩后反弹（未有效跌破）✅
└── MACD < 0（暂时，回踩中）⚠️

→ 判定：主涨段雏形
→ 后续跟踪：
    ├── MACD转正 → 正式确认主涨段 ✅
    ├── MACD持续为负 + 跌破中轨 → 降级为X段 ⚠️
    └── 创新低 → 主涨段失败 ❌
```

**实例（2026-06-17）：**
> 30F 价格=4108>55线(4044) + 结构已走第五段上涨 + 回踩30F中轨(4093)后反弹，但MACD=-2.57<0
> 
> v4.0：55线上方X段 ❌
> v4.1：主涨段雏形 ✅（结构已确认，MACD负值是回踩中的暂时状态）

#### 主涨段启动公式

```
N+2级别 MACD极强状态 或 金叉状态
    ↓
N级别 在低位出现结构后，出现主涨段
    ↓
低位结构 = 第三段上涨 或 第五段上涨
    ↓
第三段上涨前：回踩N-1级别55线
    （N+2级别金叉 → 传导至N+1级别极强 → 有回踩N-1级别55线确认的可能性）
    ↓
极度强势情况下：第一段就出主涨段（概率偏小）
```

#### 主涨段结束判定

**主涨段特征解除条件：有效跌破N-1级别中轨**
- **有效跌破 = 反抽不上**（实战中需灵活运用，记录为动态优化点）
- 主涨特征解除后，**不等于要下跌**，一般出现两种情况：

**情况①（弱势路径）：**
- 跌破N-1级别中轨后 → 反抽N-1级别中轨
- 但被压制，甚至弱势情况下被N-2级别中轨压制
- → 转入下跌

**情况②（强势路径）：**
- 跌破N-1级别中轨后 → 反抽突破中轨 → 甚至出现N-1级别MACD金叉
- 后续不管是否新高，**二次跌破N-1级别中轨**（最后一次有效跌破）
- → 至少回踩N-1级别55线
- → 甚至多数情况下应当**刺穿N-1级别55线**
- → 从而实现对N级别中轨的回踩

#### 主涨段重新焕发（X段桥梁）

```
N级别中轨支撑 或 N-1级别55线支撑
    ↓
也可以视作N级别中轨支撑
    ↓
上述回踩视作N-1级别的X段
    ↓
后续上涨结构将重新焕发N级别主涨段
    ↓
重复上述流程
    ↓
直到N级别中轨跌破 → 确认整个上涨结束
```

### Step 5: 3段/5段结构基础

#### 分型定义（严格）

**顶分型：** 相邻3根K线中，第2根K线的高点 **是3根K线高点中最高的**，且第2根K线的低点 **也是3根K线低点中最高的**。

**底分型：** 相邻3根K线中，第2根K线的低点 **是3根K线低点中最低的**，且第2根K线的高点 **也是3根K线高点中最低的**。

**K线合并（包含关系处理）：**
- 向上时：相邻K线有包含关系，取高高
- 向下时：相邻K线有包含关系，取低低

#### 笔的定义（严格）

**构成条件：**
1. 两个相邻的顶分型和底分型（或底分型和顶分型）
2. **顶和底之间至少有1根独立K线**
3. 必须经过K线合并处理后的K线图

#### 线段的定义（严格）

**构成条件：**
1. **至少由3笔构成**
2. **前三笔必须有重叠部分**
3. 线段有两种：从向上笔开始的（向上线段），从向下笔开始的（向下线段）

**线段破坏定理：**
> "线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。"
> "线段被破坏的充要条件，就是被另一个线段破坏。"

#### 走势类型与段数的关系

**3段/5段的本质：**

| 段数 | 结构 | 走势类型 | 中枢数量 | 示例 |
|:---:|:---|:---:|:---:|:---|
| **3段** | 上-下-上 | 盘整 | 1个 | 第1段上涨 → 第2段回调 → 第3段上涨 |
| **5段** | 上-下-上-下-上 | 趋势 | 2个 | 标准趋势 |
| **7段+** | 多段交替 | 趋势延续/扩展 | 2+个 | 中枢扩展 |

**v4.1新增：历史段数结构分析**

> 不只统计最近段数，必须做大周期历史段数分析：
> - "30F级别走出3927以来第五段上涨"
> - 需要足够历史数据（至少200根以上30FK线）
> - 历史段数分析帮助判断当前处于趋势的哪个阶段（末端/中段）

### Step 6: X段判定体系（v4.1新增：与主涨段雏形区分）

#### X段 vs 主涨段雏形（v4.1关键区分）

| 特征 | X段 | 主涨段雏形 |
|:---|:---|:---|
| **结构** | 不完整或已被破坏 | 完整（第五段/第三段），回踩确认 |
| **价格** | >55线 | >55线 |
| **MACD** | <0 | <0（暂时） |
| **回踩** | 无明确回踩或回踩失败 | 回踩N-1中轨/55线后反弹 |
| **后续** | 可能创新低 | 等待MACD转正确认 |
| **判定** | 警惕，可能转跌 | 乐观，等待确认 |

**区分关键：** 不是看MACD，而是看**结构是否完整** + **回踩是否确认**。

#### X段定义（v4.1精确化）

**X段区别于带结构的段：**
- 带结构的段：必须有顶分型、底分型、合并后至少有一根K线
- X段：在N+1级别没有结构，向下分解后的N级别走势

**判定公式：**
```
N+2级别出现主涨段
    ↓
期间会有对N+1级别中轨的回踩
    ↓
N+1级别回踩中轨的过程，在N+1级别不足以画出一段结构
    ↓
称N级别作为X段
    ↓
【N级别能不能画出结构不重要，重要的是N+1级别没有结构】
```

**X段出现的三个条件（v4.1核心）：**

1. **复合结构**：X段必然出现在**复合结构**中，**不能单独存在**
2. **起点=最高点**：X段必然出现在某个上涨结构的**最高点过后**
3. **后验确认**：结束后必然**突破N级别55均线**，且新高前不创新低

### Step 7: 套娃传导链分析（v4.1核心新增）

```
双日第三段上涨（结构推演，MACD暂时极弱）
```

**套娃成立条件：**
1. 维持120F55线以上（当前4110>4098）
2. 30F MACD转正（从雏形→正式确认）
3. 30F第五段结构持续（不跌破30F中轨4093）

```
├── 强 → 做多胜率高
├── 中性偏强 → 观察
├── 极弱 → 不做多，下跌持续性强
├── 弱 → 做空胜率高
└── 中性偏弱 → 观察
```

### 决策3: 主涨段判定（v4.1升级：三档）

```
价格 > MA55?
├── YES
│   ├── 结构完整（第五段/第三段）+ 回踩确认
│   │   ├── MACD > 0 → 主涨段（正式）✅
│   │   └── MACD < 0 → 主涨段雏形 ⚠️（等待转正）
│   └── 结构不完整/被破坏
│       └── MACD < 0 → X段 ❌
└── NO
    ├── MACD < 0 → 主跌段 ❌
    └── MACD > 0 → 55线下方X段 ⚠️

主涨段雏形跟踪:
├── MACD转正 → 正式确认 ✅
├── MACD持续为负 + 跌破中轨 → 降级为X段 ⚠️
└── 创新低 → 主涨段失败 ❌
```

```
├── 操作线：5F中轨跌破 → 减仓1/3
```

```
│   ├── MACD<0 + 结构完整 → 主涨段雏形，持仓

└── 跌破55线
    ├── MACD<0 + 柱放大 → 真跌破 ❌
    └── MACD>0 或 柱收敛 → 假跌破 ⚠️
```

## 动态优化点（v4.1记录）

### 1. 有效跌破的判定
**当前定义：** "有效跌破 = 反抽不上"
**存在问题：** 界定存在一定困难，实战中需灵活运用
**优化方向：** 后续需通过大量案例反复验证

### 2. 联合区的数值界定
**当前定义：** "数值接近时可以认为是联合支撑或者压力区"
**优化方向：** 不同指数/个股的点差标准可能不同

### 3. X段与主涨段雏形的边界（v4.1新增）
**当前定义：** 结构完整+回踩确认 = 雏形；结构破坏 = X段
**优化方向：** "结构完整"的自动化判定（分型→笔→线段→计数）

### 4. 套娃传导链的量化（v4.1新增）
**当前定义：** 经验性传导链描述
**优化方向：** 需要更多案例验证传导链的触发条件和确认条件

### 5. 震荡路径的概率（v4.1新增）
**当前定义：** 三路径并列
**优化方向：** 统计哪条路径概率最高，帮助用户预期管理

### 6. 主涨段雏形的确认时间（v4.1新增）
**当前定义：** MACD转正 = 确认
**优化方向：** 统计从雏形到确认的平均时间（几根K线）

### 7. 顶背离的自动识别（v4.1新增）
**当前定义：** 价格新高+MACD不新高
**优化方向：** 自动化顶背离识别，集成到脚本中

### 8. 盘中结构的自动化（v4.1新增）
**当前定义：** 手动描述
**优化方向：** 基于1F/5F数据的自动日内结构分析
