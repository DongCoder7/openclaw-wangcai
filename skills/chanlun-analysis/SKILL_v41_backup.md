### Step 3: MACD稳定性分析（v4.0核心新增）

#### MACD六种状态定义

MACD的"强弱"不是行情强弱，而是**"稳定性"**——稳定性强弱是相对于走势而言的。

| 状态 | DIF | DEA | MACD | 含义 | 操作原则 |
|:---|:---|:---|:---|:---|:---|
| **极强** | DIF≥0 | DEA≤0 | >0 | 上涨持续性极高 | **不做空** |
| 强 | DIF>DEA>0 | | >0 | 上涨持续性高 | 做多胜率高 |
| 中性偏强 | 0>DIF>DEA | | >0 | 上涨但动能减弱 | 观察 |
| **极弱** | DIF≤0 | DEA≥0 | <0 | 下跌持续性极高 | **不做多** |
| 弱 | DIF<DEA<0 | | <0 | 下跌持续性高 | 做空胜率高 |
| 中性偏弱 | 0<DIF<DEA | | <0 | 下跌但动能减弱 | 观察 |

**核心规则：**
- 极强形态 → 会导致上涨更有持续性，这时做多胜率高
- 极弱形态 → 会导致下跌更有持续性，这时做空胜率高
- **极强形态不做空，极弱形态不做多**

#### MACD六种金叉死叉形态

| 形态 | 定义 | 稳定性 | 对应操作 |
|:---|:---|:---|:---|
| **零轴金叉** | DIF/DEA在0轴附近金叉 | **极强** | 高胜率做多 |
| **零轴死叉** | DIF/DEA在0轴附近死叉 | **极弱** | 高胜率做空 |
| 水上金叉 | DIF>DEA>0，在0轴上方金叉 | 强 | 做多 |
| 水上死叉 | 0轴上方死叉 | 弱 | 做空 |
| 水下金叉 | 0轴下方金叉 | 中性偏强 | 谨慎做多 |
| 水下死叉 | DIF<DEA<0，在0轴下方死叉 | 中性偏弱 | 谨慎做空 |

**体系中：零轴金叉视为极强形态，零轴死叉视为极弱形态。**

#### 极强形态三特征（v4.0核心）

**1. 背离失效**
- N+2级别极强 → 引发N级别主涨段
- N级别以下的顶背离，可能都是**阶段性无效**
- 当极强度过、小级别主涨特征失效 → 小级别顶背离才会生效
- 生效标志：**跌破该级别55线**

**2. 55线大概率支撑位**
- N+2级别极强形态下，首次回踩N级别55线 → **极大概率得到支撑**
- → 技术体系里胜率极高的**做多开仓点**
- 如果N级别跌破了55线 → 认为N+2级别极强形态解除 → 后续不再那么"稳定"

**3. 传递性**
- 极强形态具有传递性（从低位上涨时出现）
- 极强形态 = 低位突破本级别55均线的**过程**
- 这个传递性可以**衡量上涨的持续性**
- 上涨有惯性：指数有效突破本级别55线后 → 有向**更高级别55线**运动的惯性 → 直到被更高级别55线压制

### Step 4: 主涨段判定体系（v4.0核心新增）

#### 主涨段启动公式

```
N+2级别 MACD极强状态 或 金叉状态
    ↓
N级别 在低位出现结构后，出现主涨段
    ↓
低位结构 = 第三段上涨 或 第五段上涨
    ↓
第三段上涨前：回踩N-1级别55线
    （N+2级别金叉 → 传导至N+1级别极强 → 有回踩N-1级别55线确认的可能性）
    ↓
极度强势情况下：第一段就出主涨段（概率偏小）
```

**实例映射：**
> 季度级别MACD极强/金叉 → 周线级别低位结构后出主涨段
> 周线级别低位结构 = 第三段或第五段上涨
> 这里周线级别，是从2635开始的**第五段上涨**

#### 主涨段结束判定

**主涨段特征解除条件：有效跌破N-1级别中轨**
- **有效跌破 = 反抽不上**（实战中需灵活运用，记录为动态优化点）
- 主涨特征解除后，**不等于要下跌**，一般出现两种情况：

**情况①（弱势路径）：**
- 跌破N-1级别中轨后 → 反抽N-1级别中轨
- 但被压制，甚至弱势情况下被N-2级别中轨压制
- → 转入下跌

**情况②（强势路径）：**
- 跌破N-1级别中轨后 → 反抽突破中轨 → 甚至出现N-1级别MACD金叉
- 后续不管是否新高，**二次跌破N-1级别中轨**（最后一次有效跌破）
- → 至少回踩N-1级别55线
- → 甚至多数情况下应当**刺穿N-1级别55线**
- → 从而实现对N级别中轨的回踩

#### 主涨段重新焕发（X段桥梁）

```
N级别中轨支撑 或 N-1级别55线支撑
    ↓
也可以视作N级别中轨支撑
    ↓
上述回踩视作N-1级别的X段
    ↓
后续上涨结构将重新焕发N级别主涨段
    ↓
重复上述流程
    ↓
直到N级别中轨跌破 → 确认整个上涨结束
```

**也就是说：**
- 这次N级别第五段上涨，作为主张涨段结束后
- 会有回踩N-1级别55线到N级别中轨之间的动作
- 如果在中间任何位置止跌回升 → 这段回踩为N-1级别X段
- 后续仍将有N级别主涨段出现

### Step 5: 3段/5段结构基础（v4.0补充 — 基于Exa搜索的缠论原文定义）

#### 分型定义（严格）

**顶分型：** 相邻3根K线中，第2根K线的高点 **是3根K线高点中最高的**，且第2根K线的低点 **也是3根K线低点中最高的**。
- 顶分型的最高点 = 该分型的顶

**底分型：** 相邻3根K线中，第2根K线的低点 **是3根K线低点中最低的**，且第2根K线的高点 **也是3根K线高点中最低的**。
- 底分型的最低点 = 该分型的底

**K线合并（包含关系处理）：**
- 向上时：相邻K线有包含关系，取高高（最高中的最高，最低中的最高）
- 向下时：相邻K线有包含关系，取低低（最低中的最低，最高中的最低）
- 经过合并处理后，才能正确识别分型

#### 笔的定义（严格）

**构成条件：**
1. 两个相邻的顶分型和底分型（或底分型和顶分型）
2. **顶和底之间至少有1根独立K线**（不被包含的K线）
3. 必须经过K线合并处理后的K线图

**分类：**
- 向上笔 = 底分型 + 上升K线（至少1根） + 顶分型
- 向下笔 = 顶分型 + 下降K线（至少1根） + 底分型

**特殊情况处理：**
- 两个顶或底之间没有其他顶底：如果转折级别太小，忽略第一个顶/底
- 先顶后底 = 向下笔；先底后顶 = 向上笔

#### 线段的定义（严格）

**构成条件：**
1. **至少由3笔构成**
2. **前三笔必须有重叠部分**（即第1笔的顶 > 第2笔的底，且第2笔的顶 > 第3笔的底，有价格区间重叠）
3. 线段有两种：从向上笔开始的（向上线段），从向下笔开始的（向下线段）

**线段破坏定理：**
> "线段被破坏，当且仅当至少被有重叠部分的连续三笔的其中一笔破坏。"
> "构成有重叠部分的连续三笔，那么必然会形成一线段。"
> "线段被破坏的充要条件，就是被另一个线段破坏。"

**特征序列法（精确判定线段终结）：**
- 向上线段：向下笔为特征序列（X序列）；只考察顶分型
- 向下线段：向上笔为特征序列（S序列）；只考察底分型
- 特征序列可以进行K线合并处理（非包含处理），形成标准特征序列
- 标准特征序列出现分型 = 线段结束的前提条件

**特征序列缺口处理：**
- 无缺口：特征序列顶分型/底分型出现 → 线段在该分型高点/低点结束
- 有缺口：从分型最高/最低点开始，反向序列的特征序列出现分型 → 线段结束

#### 走势类型与段数的关系

**缠论核心定理：**
> "任何级别的任何走势类型，都至少由三段以上次级别走势类型构成。"
> "任何级别的任何走势，都可以分解成同级别'盘整'、'下跌'与'上涨'三种走势类型的连接。"

**走势类型定义：**
- **盘整**：某完成的走势类型只包含一个走势中枢（由至少3段次级别走势重叠构成）
- **趋势**：某完成的走势类型至少包含两个以上依次同向的走势中枢
  - 方向向上 = 上涨；方向向下 = 下跌

**3段/5段的本质：**

| 段数 | 结构 | 走势类型 | 中枢数量 | 示例 |
|:---:|:---|:---:|:---:|:---|
| **3段** | 上-下-上（或下-上-下） | 盘整 | 1个 | 第1段上涨 → 第2段回调 → 第3段上涨 |
| **5段** | 上-下-上-下-上（或下-上-下-上-下） | 趋势 | 2个 | 第1段涨 → 第2段回调 → 第3段涨 → 第4段回调 → 第5段涨 |
| **7段+** | 多段交替 | 趋势延续/扩展 | 2+个 | 中枢扩展或更大级别走势 |

**注意：** 用户原文中的"第三段上涨"或"第五段上涨"指的是**一个主涨段内部**的次级别走势段数：
- 3段上涨 = 简单盘整后上涨完成（1个中枢）
- 5段上涨 = 趋势性上涨（2个中枢，标准趋势）
- 更多段数 = 中枢扩展或更高级别走势构建中

**带结构的段 vs X段：**
- **带结构的段**：在N级别可以画出分型→笔→段的标准结构（有顶底分型、K线合并、特征序列确认）
- **X段（不带结构的段）**：在N+1级别没有结构（不足以画出标准段），向下分解到N级别称为X段
- **判定关键**：N+1级别是否有标准段结构，而不是N级别是否有

#### 段结构判定要点

| 结构类型 | 段数 | 特征 | 后续预期 |
|:---|:---:|:---|:---|
| 简单结构 | 1段 | 少数情况 | 快速完成 |
| 标准盘整 | 3段 | 最常见（1个中枢） | 反转或延续 |
| 标准趋势 | 5段 | 趋势行情（2个中枢） | 末端注意反转 |
| 复合结构 | 5段+X段+1段 | 主涨段中 | X段作为桥梁 |
| 中枢扩展 | 9段+ | 更大级别中枢 | 升级 |
| 双平台式 | 多段 | 中枢震荡 | 等待方向选择 |

### Step 6: X段判定体系（v4.0核心重写）

#### 前提条件
X段是**只有在主涨段或主跌段的情况下才会出现**。
- 本文以主涨段举例，主跌段情况可自行历史复盘

#### X段定义（v4.0精确化）

**X段区别于带结构的段：**
- 带结构的段：从缠论上讲，必须有顶分型、底分型、合并后至少有一根K线
- X段：在N+1级别没有结构，向下分解后的N级别走势

**判定公式：**
```
N+2级别出现主涨段
    ↓
期间会有对N+1级别中轨的回踩
    ↓
N+1级别回踩中轨的过程，在N+1级别不足以画出一段结构
    ↓
称N级别作为X段
    ↓
【N级别能不能画出结构不重要，重要的是N+1级别没有结构】
    ↓
只要N+1级别没有结构，就称N+1级别向下分解后的N级别走势，为N级别的X段
```

**实例：**
> 周线主涨段中的日线走势：
> - AB段：没有顶底分型，或强行分型后中间合并不出至少一根K线 → **不带结构回踩日线中轨**
> - CD段：明显带有顶底分型，中间合并K线数量够 → **带结构回踩日线中轨**
> - 在60F级别：AB段 = **X段**，CD段 = **三段下跌结构**

#### X段出现的三个条件（v4.0核心）

**条件1：复合结构**
- X段必然出现在**复合结构**中，**不能单独存在**
- 复合结构形式："五段上涨 + X段 + 一段上涨"
- 正常的上涨结构有三段、五段、双平台式，少数情况下存在一段
- **重点不在于除X段之外的结构走成什么样，而在于X段成为了连接前后两个上涨结构的桥梁**

**条件2：起点=最高点**
- X段必然出现在某个上涨结构的**最高点过后**
- **起点一定是最高点而非次高点**，否则就是别的结构
- X段的终点 = 回踩N+1级别中轨的**结束点**
- 实例：60F的X段，起点是五段上涨的最高点，终点是回踩日线中轨

**条件3：后验确认**
- X段一个**后验条件**：结束后必然**突破N级别55均线**
- 且新高前一个上涨结构高点（即X段起点）前，**不再发生新低**
- 否则变异为**带结构的下跌段**
- 实例：X段结束后突破60F55线，且后续新上涨结构新高了X段起点3636

#### X段操作判定流程

```
出现N+2级别主涨段 + N+1级别不带结构的回踩
    ↓
优先考虑N级别X段
    ↓
后端确认：突破N级别55均线 或 不跌破N级别55均线
    ↓
确认成功 → 后续基本可以确定会新高
    ↓
如果被N级别55线压制 且 新低 → 下跌发生升级，需另做考虑
```

**X段不能孤立来看：**
- 由于级差体系有两套（组A和组B）
- 有时可能相互穿插影响
- 需要具体问题具体分析，不能太形而上学

- 日线第三段下跌 → 等待段完成时间窗口

**时间窗口条件：** 底背离 + 突破中轨
- **不是每天都买，而是等待时间窗口!**

---

### Step 15: 补偿性买点识别

经典缠论买点结构：跌破55线 → 回踩确认 → 突破中轨 → 突破上级55线

条件：
- 价格跌破MA55
- 在MA55下方企稳
- MACD底背离 或 缩量
- 买点区域：MA55下方2%以内
- 确认信号：突破中轨 + 放量
- 目标：上级55线

---

### Step 16: 复合风控（核心）

单一指标风控 → 复合信号风控

经典复合信号：
- 30F55线压制 + 5F顶背离 = 兑现一部分多头
- **双日死叉 + 30F主跌段 = 清仓**
- 5F55线突破 + 30F底背离 = 加仓
- **跌破联合支撑区 = 无条件清仓**
- 假突破 + 联合压制区 = 减仓1/3

### 决策1: 数据完整性

```
数据条数 >= 55?
├── YES → 继续分析
└── NO → 标记"数据不足"，使用更大级别替代
```

### 决策2: MACD稳定性判断

```
MACD状态?
├── 极强 (DIF≥0, DEA≤0, MACD>0) → 不做空，上涨持续性强
├── 强 (DIF>DEA>0, MACD>0) → 做多胜率高
├── 中性偏强 (0>DIF>DEA, MACD>0) → 观察
├── 极弱 (DIF≤0, DEA≥0, MACD<0) → 不做多，下跌持续性强
├── 弱 (DIF<DEA<0, MACD<0) → 做空胜率高
└── 中性偏弱 (0<DIF<DEA, MACD<0) → 观察

金叉形态?
├── 零轴金叉 → 极强形态 → 高胜率做多
├── 零轴死叉 → 极弱形态 → 高胜率做空
├── 水上金叉 → 强 → 做多
├── 水上死叉 → 弱 → 做空
├── 水下金叉 → 中性偏强 → 谨慎做多
└── 水下死叉 → 中性偏弱 → 谨慎做空
```

### 决策3: 主涨段判定（v4.0核心新增）

```
N+2级别MACD状态?
├── 极强或金叉
│   ├── N级别低位结构出现 (3段或5段)
│   │   ├── 第三段前回踩N-1级别55线 → 主涨段确认 ✅
│   │   └── 第一段直接出主涨段 → 概率偏小
│   └── 无低位结构 → 等待
└── 非极强/非金叉 → 无主涨段信号

N级别是否处于主涨段?
├── 价格>55线 + MACD>0 → 主涨段 ✅
├── 价格>55线 + MACD<0 → 55线上方X段 ⚠️
└── 价格<55线 → 非主涨段

主涨段结束?
├── 有效跌破N-1级别中轨 → 主涨特征解除
│   ├── 反抽被N-1中轨压制 → 转入下跌
│   └── 反抽突破+二次跌破 → 至少回踩N-1级别55线
└── 未跌破N-1中轨 → 主涨段继续

主涨段重新焕发?
├── N-1级别55线/N级别中轨支撑 → X段确认
│   └── 突破N-1级别55线+不创新低 → 重新焕发 ✅
└── 被55线压制+新低 → 下跌升级
```

### 决策4: X段判定（v4.0核心新增）

```
N+2级别主涨段?
├── YES
│   └── N+1级别回踩N+1级别中轨
│       ├── N+1级别有结构 (顶底分型+合并K线) → 正常回调段
│       └── N+1级别无结构 → N级别X段 ⚠️
└── NO → 无X段

X段确认条件?
├── 条件1: 出现在复合结构 (前段+X段+后段) ✅
├── 条件2: 起点=前段最高点 ✅
└── 条件3: 终点=回踩N+1级别中轨 ✅

X段后端确认?
├── 突破N级别55线 + 新高前不创新低 → X段确认成功 → 后续会新高 ✅
└── 被N级别55线压制 + 新低 → 变异为下跌段 ❌
```

### 决策5: 假突破识别

```
价格 vs MA55?
├── 站上55线
│   ├── MACD>0 + 柱放大 → 真突破 ✅ → 加仓
│   └── MACD<0 或 柱收敛 → 假突破 ⚠️ → 不追涨
├── 55线上方运行
│   ├── MACD>0 → 主涨段，持仓
│   └── MACD<0 → X段，警惕回落
├── 55线下方运行
│   ├── MACD<0 → 主跌段，清仓/观望
│   └── MACD>0 → X段，关注补偿性买点
└── 跌破55线
    ├── MACD<0 + 柱放大 → 真跌破 ❌ → 止损
    └── MACD>0 或 柱收敛 → 假跌破 ⚠️ → 不恐慌
```

### 决策6: 联合区判断

```
跨级别关键价位?
├── 数值接近 → 联合支撑/压力区
│   ├── 联合支撑 → 强支撑，跌破清仓
│   └── 联合压制 → 强压力，突破满仓
└── 差值较大 → 各自独立判断
```

### 决策7: 二买/二卖判断

```
当前段结构?
├── 下跌段完成 + 第1笔up + 第2笔down
│   ├── 第2笔down不创新低 → 二买成立 ✅ → 加仓
│   └── 第2笔down创新低 → 二买失败 ❌ → 减仓
├── 上涨段完成 + 第1笔down + 第2笔up
│   ├── 第2笔up不创新高 → 二卖成立 ❌ → 减仓
│   └── 第2笔up创新高 → 二卖失败 ✅ → 持仓
└── 段未完成
    └── 等待段完成后再判断
```

### 决策8: 级别传导链

```
传导链方向?
├── 3F顶背离 → 下跌传导链启动
│   ├── 跌破3F中轨 → 减仓1/3
│   ├── 跌破15F55线 → 再减仓1/3
│   ├── 30F二买失败 → 清仓
│   └── 日线第三段 → 等待联合支撑区
├── 3F底背离 → 上涨传导链启动
│   ├── 突破3F中轨 → 加仓1/3
│   ├── 突破15F55线 → 再加仓1/3
│   ├── 30F二买成立 → 满仓
│   └── 突破联合压制 → 追涨
└── 无明显背离
    └── 震荡，高抛低吸
```

### 决策9: 双日级别判断

```
双日MACD状态?
├── MACD>0 + Hist>0 → 中长期多头，持仓
├── MACD>0 + Hist接近0 → 警惕，可能死叉
├── MACD<0 + Hist<0 → 中长期空头，清仓
```

## 动态优化点（v4.0记录 + 新增4个）

### 1. 有效跌破的判定

**当前定义：** "有效跌破 = 反抽不上"

**存在问题：** 界定存在一定困难，实战中需要灵活运用

**优化方向：**
- 后续需通过大量案例反复验证
- 结合具体级别、MACD状态、量能综合判断
- 记录为动态优化点，持续迭代

### 2. 联合区的数值界定

**当前定义：** "数值接近时可以认为是联合支撑或者压力区"

**优化方向：**
- 需要结合具体数字灵活运用
- 不同指数/个股的点差标准可能不同
- 实践中积累各品种的"接近"标准

### 3. X段与带结构段的边界

**当前定义：** "N+1级别没有结构"是关键

**优化方向：**
- "没有结构"的判定标准（顶底分型、合并K线）需更精确
- 不同级别、不同品种的"结构"标准可能不同

### 4. 双日极弱解除条件（v4.0新增 - 经验规则）

**经验规则：** N级别突破MA55线 → 可传导至N+2级别，解除极弱形态

**具体应用：**
- 30F级别突破MA55线（有效突破，非假突破）→ 双日极弱形态可能解除
- 逻辑：小级别走势的持续性积累，可改变大级别MACD状态
- 注意：此规则为经验总结，需后续案例验证

**优化方向：**
- 记录为经验规则，非严格定理
- 后续需区分：真突破 vs 假突破对解除条件的影响
- 结合MACD柱状体放大/收敛验证

### 5. 段数结构分解（v4.0新增）

**需求：** 主涨段分析必须包含段数结构分解

**格式：**
```
N级别结构分解:
├── 起点: [价格]（一买/一卖低点）
├── 段数: [3段/5段/X段+1段]
├── 高点: [价格]（顶分型确认）
├── 当前: [回踩/反抽/新段进行中]
├── 目标: [突破前高/测试MA55/测试中轨]
└── 重置: [X段回踩重置上涨结构]
```

**优化方向：**
- 自动化段数识别（分型→笔→线段→计数）
- 起点/高点自动识别
- 与X段"重置"功能联动

### 6. X段"重置"功能（v4.0新增）

**定义：** X段不仅是桥梁/骗炮，还可以"重置上涨结构"

**机制：**
- 上涨段过高后，MACD背离/乖离过大
- X段回踩清除乖离，修复MACD
- 回踩后重新突破 → 新的"干净"上涨段

**标志：**
- X段结束后突破前高
- MACD重新放大（柱状体增长）
- 新段与旧段相比，MACD状态更健康

**与"骗炮"的区别：**
- 重置：X段后突破前高，MACD转正且放大
- 骗炮：X段后无法突破，MACD继续收敛

### 7. 双周K线实体分析（v4.0新增）

**原则：** K线实体 > 指标MACD（价格优先于指标）

**分析框架：**
```
双周状态:
├── K线实体 vs 中轨：实体>中轨 → 多头力量有效
├── MACD状态：极弱/极强/金叉/死叉
├── 矛盾处理：实体+MACD矛盾时，以实体为先
│   ├── 实体>中轨 + MACD极弱 → 极弱可能过度，观察修复
│   └── 实体<中轨 + MACD极强 → 极强可能过度，观察回调
└── 目标推导：双日第三段上涨目标价
```

**优化方向：**
- 实体计算：Close/Open vs 中轨
- 多周期K线实体对比（双周/周线/日线）
- 实体+影线综合分析
