## 十四、结构分析融合（v4.4新增）

### 14.1 笔段识别

**顶分型**: 连续三根K线，中间高点最高，低点也最高
**底分型**: 连续三根K线，中间低点最低，高点也最低

**笔的形成条件**:
- 顶底分型之间至少有1根独立K线
- 顶分型必须高于底分型
- 笔的结束需要反向分型确认

### 14.2 中枢定义

**中枢区间**: 连续三段重叠区域
- 取三段的最高低点中的最低高点和最高低点
- 中枢形成后，价格围绕中枢震荡

**中枢类型**:
- 上涨中枢: 下上下结构
- 下跌中枢: 上下上结构
- 扩展中枢: 中枢区间扩大

### 14.3 买点识别

#### 一买（第一类买点）
**定义**: 下跌背驰点
**条件**:
1. 下跌趋势明确（至少两段下跌）
2. 第二段下跌力度弱于第一段（MACD面积/高度减小）
3. 出现底分型确认
4. 成交量萎缩后放量

**实战意义**: 最佳买点，风险收益比最优
**止损**: 底分型低点下方

#### 二买（第二类买点）
**定义**: 一买后的回调不破新低
**条件**:
1. 一买后上涨一段
2. 回调不创新低
3. 出现底分型
4. MACD在零轴上方或附近金叉

**实战意义**: 更安全的买点，适合大资金
**止损**: 一买低点下方

#### 三买（第三类买点）
**定义**: 中枢上方回调不回到中枢
**条件**:
1. 中枢形成后向上突破
2. 回调不回到中枢区间
3. 出现底分型

**实战意义**: 主升浪启动点
**止损**: 中枢上轨下方

### 14.4 级别状态标注

#### 日线极强/极弱
**极强**:
- 日线MA55上方
- 日线MACD红柱放大
- 双日MACD金叉运行
- **操作**: 持仓为主，回调加仓

**极弱**:
- 日线MA55下方
- 日线MACD绿柱放大
- 双日MACD死叉运行
- **操作**: 空仓或轻仓，反弹减仓

#### 60F/120F段数分析
**第四段拐点**（高手经验）:
- 60F或120F运行至第四段时，容易出现拐点
- 第四段背驰 → 可能反转
- 第四段不背驰 → 可能延续

**判定方法**:
1. 标识60F/120F的笔段
2. 统计段数
3. 第四段对比第二段力度
4. 背驰 → 拐点预期

#### 双日周期状态
**极弱**:
- 双日MACD零轴下方运行
- 双日MA55压制
- **影响**: 大周期压制，上涨空间有限

**拐点**:
- 双日MACD金叉雏形
- 价格接近双日MA55
- **操作**: 关注能否突破

### 14.5 高手经验固化

#### 4005支撑逻辑
**来源**: 5F级别结构分析
**逻辑**:
1. 5F级别形成二买后冲高回落
2. 5F中枢下轨在4005附近
3. 若再次测试4005不跌破，形成5F二买确认
4. **量能配合**: 缩量测试+放量反弹

**操作**:
- 4005附近出现5F底分型+缩量 → 轻仓试错
- 跌破4005且放量 → 止损
- 反弹突破30F55线 → 加仓

#### 30F55线突破意义
**高手观点**: 突破30F55线 = 解除日线极弱
**条件**:
1. 站稳30F55线3根K线以上
2. 30F MACD金叉或红柱放大
3. 成交量大于前5日均量1.2倍
4. 5F出现三买确认

**路径**:
- 快速突破: 30F极强直接突破
- 慢速突破: 120F金叉下5F主涨段 → 30F回踩二买 → 第三段突破

#### 恐慌买点（类似6.8）
**特征**:
1. 快速下跌，跌幅>2%
2. 成交量急剧放大（恐慌盘涌出）
3. 5F/15F出现底背离
4. 情绪极度悲观

**操作**:
- 不抄底，等确认
- 底分型+缩量 → 试仓
- 突破5F55线 → 加仓
- 跌破底分型低点 → 止损

## 十五、左侧交易框架（v4.4新增）

### 15.1 左侧 vs 右侧

| 维度 | 左侧交易 | 右侧交易 |
|------|---------|---------|
| 核心 | 预测拐点 | 跟随确认 |
| 买点 | 一买/二买 | 三买/突破 |
| 风险 | 较高（可能抄底失败） | 较低（确认后跟进） |
| 收益 | 更高（买在最低点） | 稳健（牺牲部分收益） |
| 适用 | 结构性拐点 | 趋势延续 |

### 15.2 左侧买点识别流程

```
步骤1: 大级别定位
  ├── 双日/日线处于极弱末期
  ├── 大级别MACD底背离
  └── 时间窗口临近

步骤2: 中级别结构
  ├── 60F/120F运行至第四段
  ├── 第四段力度减弱
  └── 中枢背驰或盘整背驰

步骤3: 小级别确认
  ├── 5F/15F出现底分型
  ├── 5F/15F MACD金叉
  ├── 成交量萎缩后放量
  └── 关键位（如4005）支撑有效

步骤4: 买点确认
  ├── 一买: 底分型+背驰
  ├── 二买: 回调不创新低+缩量
  └── 量能配合: 放量突破5F55线
```

### 15.3 与右侧退出结合

**融合策略**:
```
左侧建仓 + 右侧保护

建仓条件（左侧）:
  ├── 5F二买确认（如4005支撑）
  ├── 仓位: 3-5成
  └── 止损: 底分型低点下方

保护机制（右侧）:
  ├── 若跌破锚定线（30F55）→ 减仓1/2
  ├── 若跌破右侧退出线（60F MA55）→ 清仓
  └── 若触发双轨跌破 → 无条件清仓

加仓条件（右侧确认）:
  ├── 突破30F55线+站稳3根K线
  ├── 30F MACD金叉
  └── 仓位: 加至7-8成
```

### 15.4 量能分析增强

#### 成交量结构
**缩量**:
- 下跌缩量: 抛压减轻，可能企稳
- 上涨缩量: 买盘不足，可能回调

**放量**:
- 下跌放量: 恐慌盘/主力出货
- 上涨放量: 资金入场/突破确认

#### 量价配合标准
| 走势 | 量能 | 判定 |
|------|------|------|
| 上涨 | 放量 | ✅ 健康 |
| 上涨 | 缩量 | ⚠️ 背离，警惕 |
| 下跌 | 缩量 | ✅ 洗盘，可能企稳 |
| 下跌 | 放量 | ❌ 恐慌，继续跌 |
| 突破 | 放量>1.5倍 | ✅ 真突破 |
| 突破 | 缩量 | ❌ 假突破 |

#### 恐慌盘识别
**特征**:
1. 成交量突增（>前5日均量2倍）
2. 长下影线或实体大阴线
3. 5F/15F底背离
4. 情绪指标极端（如VIX飙升）

**操作**:
- 恐慌不抄底，等确认
- 底分型+缩量 → 试仓
- 连续恐慌 → 等待，不急于入场

## 十九、心理建设（v4.4更新）

### 左侧交易者心理

- **抄底时**: 我不是抄底，我是在结构买点试错
- **止损时**: 止损是试错成本，不是错误
- **等待时**: 没有买点就等待，不乱动
- **确认时**: 结构+量能+确认，三位一体

### 右侧交易者心理（原有）

- **开盘前**: 我不预测，我只应对
- **持仓时**: 锚定线之上，我是木头人
- **空仓时**: 不抄底，等收复
- **止损时**: 止损是成本，不是错误

### 融合后的心理框架

```
分析时: 结构为主，指标为辅，量能确认
操作时: 左侧试仓，右侧确认，双轨保护
心态上: 不预测，只应对；有买点就试，没买点就等
纪律上: 止损坚决，不加死仓，保护本金第一
```
