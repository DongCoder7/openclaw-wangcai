## 一、分级容错机制

### 1.1 为什么不同级别需要不同容错？

- **小级别（1F/5F）**：波动大，噪音多，容错放宽 → ±3点
- **核心级别（15F/30F）**：结构精确，容错最严格 → ±2点
- **大级别（60F/120F）**：趋势稳定，容错适中 → ±3点
- **超大级别（日线/双日）**：周期长，容错放宽 → ±5点

### 1.2 容错标准表

| 级别 | 容错点数 | 容错比例 | 说明 |
|------|----------|----------|------|
| 1F/5F | ±3点 | ~0.07% | 小级别波动大，放宽 |
| **15F/30F** | **±2点** | **~0.05%** | **核心级别，最严格** |
| 60F/120F | ±3点 | ~0.07% | 大级别，适中 |
| 日线/双日 | ±5点 | ~0.12% | 超大级别，最宽 |

### 1.3 判定规则

```
价格 vs 关键位:
├── 价格 > 关键位 + 容错 → 上方 ✅
├── 价格 < 关键位 - 容错 → 下方 ❌
└── |价格 - 关键位| ≤ 容错 → 模糊地带 ⚠️

刺破判定:
├── 最低 < 关键位 - 容错 → 刺破
│   └── 刺破深度 = (关键位 - 最低) / 关键位 * 100%
│   └── 深度<0.3%: 轻微刺破，支撑有效
│   └── 深度0.3-0.6%: 中度刺破，支撑减弱
│   └── 深度>0.6%: 深度刺破，支撑可能失效
└── 最低 ≥ 关键位 - 容错 → 未刺破
```

## 二、联合压制区

### 2.1 定义

两个及以上级别的关键位在同一价格区域重叠，形成多级别共振压制。

### 2.2 典型组合

- **30F中轨 + 15F55线**：30F中轨(4123)与15F55线(4115)重叠区域4115-4123
- **30F上轨 + 5F上轨**：短期与中期压制共振
- **60F中轨 + 30F MA55**：大周期与核心周期压制重叠

### 2.3 判定标准

```
联合压制区形成条件：
├── 两个及以上级别关键位距离 < 10点（约0.25%）
├── 价格处于这些关键位下方或附近
└── 多个级别MACD同时转负或柱缩短

联合压制区强度分级：
├── 强压制：3个及以上级别重叠
├── 中压制：2个级别重叠
└── 弱压制：1个级别+辅助指标
```

### 2.4 实战意义

价格处于联合压制区下方时：
- 上方压制：多级别共振区域
- 突破难度：需要放量+MACD配合
- 被压制回落概率：高（>60%）
- 操作：不追涨，等待突破确认或回踩低吸

## 三、X段证伪

### 3.1 X段本质

主涨段/主跌段结束后，N+1级别无结构时的过渡段。

### 3.2 X段证伪条件

```
X段证伪判定：
├── 条件1：X段运行时间超过正常周期（>3根N级别K线）
├── 条件2：X段价格被联合压制区压制回落
├── 条件3：X段MACD负值扩大
├── 条件4：更大级别（N+2）MACD同步转负
└── 满足2个及以上条件 -> X段证伪

X段证伪后的升级：
├── 30F X段证伪 -> 升级为60F级别下跌
├── 60F X段证伪 -> 升级为120F级别下跌
```

### 3.3 操作

X段运行中（未证伪）：持仓或轻仓，等待X段完成。  
X段证伪后：立即减仓1/2或清仓。

## 四、右侧退出双轨制

### 4.1 双轨标准

```
右侧退出线A：60F MA55
├── 优点：反应灵敏
├── 缺点：可能被短期波动触发
└── 触发：收盘<60F MA55且后续K线继续下行

右侧退出线B：120F中轨
├── 优点：过滤短期波动
├── 缺点：反应滞后
└── 触发：收盘<120F中轨且后续K线继续下行

双轨制触发规则：
├── 线A或线B任一有效跌破 -> 减仓1/2（预警）
├── 线A和线B同时有效跌破 -> 无条件清仓
```

### 4.2 与分级容错结合

```
60F MA55（线A）：60F容错±3点，有效跌破：收盘<MA55-3且后续下行
120F中轨（线B）：120F容错±3点，有效跌破：收盘<中轨-3且后续下行
```

## 五、弱势震荡默认路径

### 5.1 判定依据

- 15F55线连续失守（短期偏弱）
- 30F中轨未收复（中期承压）
- 联合压制区有效
- 日线极强解除后无新推动力
- MACD修复需要时间（1-2天）

### 5.2 特征

价格：前低-联合压制区之间，日内振幅<1.5%，小阴小阳。  
指标：MACD在零轴附近震荡，MA55上下波动，布林带收口。  
时间：1-3天，目的修复指标，等待方向选择。

### 5.3 操作

持仓者：锚定线之上无视波动，不频繁交易。  
空仓者：高抛低吸，3-5成仓位，不重仓。

## 六、快速上涨反直觉

### 6.1 判定依据

- 日线极强已解除
- 双日金叉运行周期中，动能减弱
- 15F55线失守，短期套娃链断裂
- 联合压制区有效，突破需大量资金
- 无重大利好消息驱动

### 6.2 概率评估

快速上涨条件（需同时满足）：
1. 放量突破联合压制区（>前日1.5倍）
2. 15F55线快速收复（开盘30分钟内）
3. 30F MACD柱由负转正
4. 60F MACD止跌回升
5. 有消息或资金驱动

**同时满足概率：<20%**

## 九、刺破 vs 有效跌破

### 9.1 刺破（Pierce）

- 定义：盘中跌破关键位，但收盘拉回至关键位上方
- 深度：通常<1%
- MACD：仍>0或接近0
- 判定：支撑仍有效，可能是洗盘或测试

### 9.2 有效跌破（Break）

- 定义：跌破关键位后，收盘未能站回，且后续K线继续下行
- 深度：通常>1%或连续多根K线收于关键位下方
- MACD：正式转负，且负值扩大
- 判定：支撑失效，进入调整或下跌

### 9.3 灰色地带

刺破深度0.5%-1% + MACD接近0 + 收盘勉强站回 → 支撑减弱，次日需确认。

## 十、15F55线分水岭

### 10.1 传导链逆向推导

```
15F55线失守
    ↓
30F55线支撑失效
    ↓
60F55线被考验
    ↓
120F极强解除
    ↓
日线极强失败
    ↓
双日零轴金叉延迟
```

### 10.2 为了维持双日金叉，需要什么？

```
为了维持双日金叉
    └── 需要日线极强维持
        └── 需要120F极强维持
            └── 需要60F55线守住
                └── 需要30F55线支撑有效
                    └── 需要15F55线快速收复
```

## 十一、三路径动态修正

### 11.1 开盘前给出初始概率

```
路径A（理想）：条件 → 走势 → 目标 → 概率X%
路径B（震荡）：条件 → 走势 → 时间换空间 → 概率Y%
路径C（风险）：条件 → 走势 → 止损位 → 概率Z%
```

### 11.2 盘中关键事件动态调整

```
今日突破关键位 + 放量 → 明日路径A概率上升
今日跌破关键位 + 顶背离 → 明日路径C概率上升
今日震荡无方向 → 维持原概率
```

### 11.3 次日分析必须包含

- 昨日实际走势描述
- 哪些预期被证实/证伪
- 哪些新结构改变了格局
- 新概率分配

## 十二、时间窗口同步解除

### 12.1 判定

当两个及以上级别的时间窗口在同一时间点附近到期时，波动率会异常放大。

### 12.2 操作

提前1-2天降低仓位，或收紧止损。

## 十三、量能辅助确认

### 13.1 突破关键位

- 放量>前日1.5倍 → 真突破概率高
- 缩量 → 假突破概率高

### 13.2 回踩关键位

- 缩量 → 洗盘概率高
- 放量下跌 → 真跌破概率高

### 13.3 注意

量能不是核心，结构优先于量能。

## 十七、心理建设

### 开盘前

- 我不预测，我只应对
- 任何走势都是可能的
- 我的任务是执行触发条件

### 持仓时

- 锚定线之上，我是木头人
- 模糊地带，我不操作
- 跌破关键位，我立即执行

### 空仓时

- 不抄底，等收复
- 不追涨，等回踩
- 右侧退出后，我耐心等待

### 亏损时

- 止损是成本，不是错误
- 右侧退出保护本金
- 留得本金在，不怕没机会

### 盈利时

- 不贪心，按触发条件执行
- 让利润奔跑，但设好跟踪止损
- 套娃链完整时，持仓不动
