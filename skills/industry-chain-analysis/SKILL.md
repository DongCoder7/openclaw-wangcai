# 产业链深度分析Skill (v26全因子版)

## 核心方法论：五维分析框架

### 维度1: 产业链结构拆解

**分析模板**:
```
上游(材料/设备) → 中游(制造/加工) → 下游(应用/终端)
     ↓                    ↓                    ↓
   价格弹性最大         产能瓶颈环节         库存重估收益
```

**关键环节识别**:
- **最紧缺环节**: 供需缺口最大、涨价最猛、扩产最慢
- **产能瓶颈**: 设备受限、技术壁垒高、认证周期长
- **库存重估**: 下游囤货、价格上涨、存货增值

### 维度2: 价格周期数据

**必查数据清单**:

| 指标类别 | 具体指标 | 数据来源 |
|:---------|:---------|:---------|
| **价格涨幅** | 产品涨价幅度、频率 | 行业协会、券商研报 |
| **供需缺口** | 需求增速 vs 供给增速 | 集邦咨询、卓创资讯 |
| **库存水平** | 当前库存周转天数 | 公司财报、产业链调研 |
| **产能利用率** | 头部企业开工率 | 上市公司公告 |
| **订单能见度** | 订单排期、预付款比例 | 公司调研 |

**价格周期判断**:
- 🟢 **涨价初期**: 涨幅<30%，供需缺口<10%，最佳介入时机
- 🟡 **涨价中期**: 涨幅30%-100%，供需缺口10%-20%，持有
- 🔴 **涨价末期**: 涨幅>100%，产能扩张落地，考虑退出

### 维度3: 业绩验证体系

**业绩预增分析框架**:

```
业绩预增质量 = 涨价传导能力 × 产能利用率 × 成本控制
```

**评估维度**:
1. **涨价传导**: 产品是否直接受益涨价（存储模组 > 芯片设计）
2. **产能弹性**: 产能扩张进度、设备到位情况
3. **成本锁定**: 原材料库存水平、长单锁价情况
4. **订单质量**: 大客户占比、订单持续时间

**业绩预增分级**:
- 🟢 **高质量预增**: 涨价驱动 + 产能满产 + 订单可见度>6个月
- 🟡 **中等质量**: 部分涨价传导 + 产能爬坡中
- 🔴 **低质量预增**: 一次性收益 + 不可持续

### 维度4: v26全因子评分

**因子权重配置** (产业链投资专用):

| 因子类别 | 权重 | 适用场景 |
|:---------|:----:|:---------|
| **动量因子** (ret_20/60) | 25% | 捕捉涨价启动时机 |
| **质量因子** (sharpe_like) | 20% | 筛选稳健龙头 |
| **资金因子** (money_flow) | 20% | 确认机构介入 |
| **财务因子** (roe/revenue_growth) | 15% | 业绩验证 |
| **波动率因子** (vol_20) | 10% | 风险控制 |
| **趋势因子** (price_pos_20) | 10% | 技术确认 |

**评分标准**:
- **>+10分**: ⭐⭐⭐ 强烈推荐
- **+5~+10分**: ⭐⭐ 推荐
- **0~+5分**: ⭐ 观望
- **<0分**: ⚠️ 回避

### 维度5: Tier分级投资建议

**Tier 1: 高确定性+高弹性 (60%仓位)**
- 筛选标准: 业绩预增>+50% + 直接受益涨价 + v26评分>+10
- 代表类型: 紧缺环节龙头、库存重估标的
- 预期收益: +15%~+25%

**Tier 2: 产业趋势明确 (30%仓位)**
- 筛选标准: 产能扩张中 + 订单饱满 + v26评分>+5
- 代表类型: 设备商、材料商
- 预期收益: +10%~+15%

**Tier 3: 主题催化/底部反转 (10%仓位)**
- 筛选标准: 事件驱动 + 高风险高弹性
- 代表类型: 新进入者、困境反转
- 预期收益: +20%~+50% (高风险)

## 分析流程

### Step 1: 产业链拆解 (30分钟)
1. 画出产业链结构图
2. 识别各环节龙头公司
3. 标记紧缺环节和产能瓶颈

### Step 2: 数据收集 (60分钟)
1. 收集价格数据（涨幅、频率）
2. 收集供需数据（缺口、库存）
3. 收集业绩数据（预增、产能）

### Step 3: v26因子评分 (30分钟)
1. 获取产业链相关股票池
2. 计算26个因子得分
3. 排名并筛选Top 10

### Step 4: 综合评估 (30分钟)
1. 结合价格周期判断位置
2. 结合业绩质量验证持续性
3. Tier分级分配仓位

### Step 5: 生成报告 (30分钟)
1. 输出产业链分析结论
2. 给出具体投资组合
3. 明确止损止盈位
