# 轮动信号识别

## 五维景气度评分

### 评分维度与权重

| 维度 | 权重 | 评估内容 | 数据来源 |
|:-----|:----:|:---------|:---------|
| **政策** | 30% | 国家级政策、地方跟进、监管态度 | 国务院、发改委、工信部 |
| **订单** | 25% | 龙头订单、产能扩张、pipeline | 公司公告、产业调研 |
| **业绩** | 25% | 季度业绩、增速变化、超预期程度 | 财报、业绩预告 |
| **估值** | 10% | PE历史分位、相对估值 | 行情软件 |
| **资金** | 10% | 北向流入、主力资金、连续天数 | 同花顺、东方财富 |

### 评分标准

| 分数 | 标准 | 信号 |
|:----:|:-----|:-----|
| 5分 | 极强利好 | 国家级重大政策/订单翻倍/业绩超预期50%+/PE<30%分位/资金连续5日流入 |
| 4分 | 利好 | 部委政策/订单增长50%/业绩超预期20%/PE<40%分位/资金连续3日流入 |
| 3分 | 中性 | 无重大变化/订单平稳/业绩符合预期/PE中等/资金波动 |
| 2分 | 偏空 | 政策收紧/订单下滑/业绩miss/PE>70%分位/资金连续流出 |
| 1分 | 利空 | 重大政策打击/订单腰斩/业绩暴雷/PE>90%分位/资金大幅流出 |

## 买入信号（五维共振）

### 触发条件

满足**4项以上**向好信号，触发买入/加仓：

```
✓ 政策：国家级政策出台/地方大力跟进
✓ 订单：龙头订单超预期/产能扩张加速
✓ 业绩：季度业绩超预期/增速提升
✓ 估值：PE处于历史30%分位以下
✓ 资金：北向连续3日净流入/主力资金流入
```

### 买入操作

- **T0级板块**：加仓至目标仓位上限
- **T1级板块**：加仓至目标仓位
- **T2级板块**：小仓位试水

## 卖出信号（风险警示）

### 触发条件

满足**2项以上**恶化信号，启动减仓：

| 维度 | 信号 | 减仓幅度 |
|:-----|:-----|:---------|
| **政策** | 政策转向/监管趋严 | 减仓50% |
| **业绩** | 季度业绩miss/增速下滑 | 减仓30% |
| **估值** | PE处于历史90%分位以上 | 减仓30% |
| **资金** | 北向连续5日净流出 | 减仓20% |
| **技术** | 跌破60日均线/放量下跌 | 减仓20% |

### 卖出操作

- 逐步减仓，不一次性清仓
- 优先减仓弱势标的
- 保留核心底仓观察

## 轮动信号综合判断

### 强势板块特征

```
✓ 板块涨幅>3%
✓ 多股涨停
✓ 成交额放大
✓ 北向资金流入
✓ 领涨市场
```

### 弱势板块特征

```
✗ 板块跌幅>2%
✗ 多股跌停
✗ 成交额萎缩
✗ 资金流出
✗ 领跌市场
```

### 板块切换信号

| 信号 | 解读 | 操作 |
|:-----|:-----|:-----|
| 原强势板块涨不动 | 资金获利了结 | 减仓原板块 |
| 新板块放量上涨 | 资金开始流入 | 加仓新板块 |
| 高切低（高位板块→低位板块） | 风险偏好下降 | 减仓成长，加仓价值 |
| 低切高（低位板块→高位板块） | 风险偏好上升 | 加仓成长 |

## 特殊场景轮动

### 财报季轮动（1/4/7/10月）

- **财报前1周**：减仓业绩不确定标的
- **财报发布当天**：
  - 超预期：持有/加仓
  - 符合预期：持有
  - 不及预期：减仓/止损

### 政策驱动轮动

- **政策突发**：评估影响，减仓受损板块，加仓受益板块
- **政策预期**：提前布局受益板块
- **政策落地**：观察执行力度，决定继续持有或兑现

### 外围市场映射

- **美股科技大涨**：A股AI算力板块高开
- **美股大跌**：A股开盘承压，关注防御板块
- **港股大涨**：A股H股映射标的跟涨
