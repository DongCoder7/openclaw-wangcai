# 板块分级详解

## T0级 - 核心持仓（必配，长期景气）

### AI算力板块
- **权重建议**：15-20%
- **核心逻辑**：AI革命基础设施，算力需求持续增长
- **代表标的**：中际旭创、天孚通信、新易盛、工业富联
- **跟踪频率**：每日
- **关键指标**：英伟达股价映射、光模块出货量、AI服务器订单

### 算力租赁/IDC板块
- **权重建议**：8-10%
- **核心逻辑**：AI时代"卖水人"，轻资产高弹性
- **代表标的**：润泽科技、奥飞数据、英维克、数据港
- **跟踪频率**：每日
- **关键指标**：机柜上架率、AI算力租赁价格、订单增速

### 半导体设备板块
- **权重建议**：10-12%
- **核心逻辑**：国产替代最确定，政策强力支持
- **代表标的**：北方华创、中微公司、拓荆科技、华海清科
- **跟踪频率**：每周
- **关键指标**：招标订单、技术突破、国产替代率

### 储能板块
- **权重建议**：8-10%
- **核心逻辑**：新能源高景气，储能强制配储政策
- **代表标的**：阳光电源、宁德时代、亿纬锂能、鹏辉能源
- **跟踪频率**：每周
- **关键指标**：储能装机量、电池价格、海外订单

### 高股息红利板块
- **权重建议**：20-25%
- **核心逻辑**：利率下行+防御底仓，类债券属性
- **代表标的**：长江电力、中国神华、工商银行、中国移动
- **跟踪频率**：每周
- **关键指标**：股息率、分红率、利率走势

- **关键指标**：产品迭代、量产进度、订单落地

### 自动驾驶板块

## 板块轮动规律

### 典型轮动周期

```
经济复苏期：顺周期板块领先（有色、化工、机械）
    ↓
经济过热期：成长板块强势（科技、新能源）
    ↓
经济滞胀期：防御板块占优（高股息、消费）
    ↓
经济衰退期：债券+黄金领先，权益减仓
```

### 市场风格轮动

| 风格 | 强势板块 | 弱势板块 |
|:-----|:---------|:---------|
| **成长风格** | AI算力、半导体、创新药 | 高股息、银行 |
| **价值风格** | 高股息、银行、公用事业 | AI算力、半导体 |
| **大盘风格** | 白酒、金融、大消费 | 小盘题材 |
| **小盘风格** | 题材概念、ST摘帽 | 大盘蓝筹 |
