# AH比价与套利策略

## AH股基本概念

### 定义
- **A股**：人民币普通股，在上海/深圳证券交易所上市
- **H股**：港股上市的内地企业股票，以港元计价
- **AH股**：同一家公司在A股和H股同时上市的股票

### AH股特点
- 同股同权：同一公司，相同投票权
- 不同市场：A股面向内地投资者，H股面向全球投资者
- 不同货币：A股人民币计价，H股港元计价
- 价格差异：由于市场分割，AH股存在价差

## AH溢价计算

### 计算公式

```
AH溢价率 = (A股价格 / H股价格 / 汇率 - 1) × 100%

其中：
- A股价格：人民币
- H股价格：港元
- 汇率：人民币/港元（约0.92-0.93）
```

### 溢价率解读

| 溢价率 | 含义 | 投资建议 |
|:------:|:-----|:---------|
| > 50% | H股大幅折价 | 强烈建议选H股 |
| 30% - 50% | H股明显折价 | 建议选H股 |
| 15% - 30% | 正常溢价区间 | 根据流动性和红利税综合判断 |
| < 15% | 溢价较低 | 建议选A股（流动性更好） |
| < 0% | H股溢价（罕见） | 选A股 |

## 红利税影响

### 税率对比

| 市场 | 持股期限 | 红利税 | 备注 |
|:-----|:---------|:------:|:-----|
| **A股** | <1个月 | 20% | - |
| **A股** | 1个月-1年 | 10% | - |
| **A股** | >1年 | 0% | 长期免税 |
| **H股（港股通）** | - | 20-28% | 内地个人投资者通过港股通 |

### 红利税调整影响

**若港股通红利税减免**：
- 高股息H股吸引力大幅提升
- AH溢价收窄
- 建议加仓高股息H股

## 投资决策框架

### 选择H股的场景

1. **AH溢价 > 30%**
   - H股折价明显，安全边际高
   - 适合长期持有

2. **高股息策略**
   - 即使考虑红利税，H股股息率仍高
   - 适合追求稳定收益的投资者

3. **外资看好**
   - 外资持续流入港股
   - 南向资金持续买入

4. **港股流动性改善**
   - 港股通扩容
   - 纳入国际指数

### 选择A股的场景

1. **AH溢价 < 15%**
   - 溢价较低，A股流动性优势显现
   - 适合短期交易

2. **高成长策略**
   - A股估值容忍度更高
   - 成长股溢价合理

3. **内地资金主导**
   - 内地政策利好
   - 北向资金持续流入

4. **打新/套利需求**
   - A股打新收益
   - 可转债等衍生品

## 风险因素

### 汇率风险
- 人民币兑港元波动影响实际收益
- 港币挂钩美元，受美联储政策影响

### 流动性风险
- H股部分标的小盘股流动性差
- 建议优选港股通标的

### 政策风险
- 两地监管政策差异
- 港股受国际市场影响更大

### 市场风险
- 港股波动通常大于A股
- 外围市场冲击更直接
