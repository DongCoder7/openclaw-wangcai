# H股个股分析Skill

## 核心能力

本Skill整合以下SOP方法：
- 股票深度分析SOP → 8步分析框架
- 全球资产配置SOP → 港股配置逻辑与AH比价

- 风险管理SOP → 仓位控制与止损规则

## 港股特色分析维度

### AH股比价机制

**核心逻辑**：同股同权，价差反映市场差异

**比价规则**：
- AH溢价 > 30%：选H股（港股更便宜）
- AH溢价 < 15%：选A股（A股流动性更好）
- 15% < 溢价 < 30%：根据流动性和红利税综合判断

**红利税影响**：
- H股红利税：20-28%（港股通）
- A股红利税：0%（持股1年以上）
- 若港股通红利税减免：加仓高股息H股

### 南向资金跟踪

**南向资金定义**：内地资金通过港股通流入港股

**跟踪指标**：
- 南向资金单日净流入金额
- 连续流入天数
- 重点买入标的
- 行业偏好变化

**信号意义**：
- 南向连续3日大幅流入：积极信号
- 南向连续5日流出：警示信号
- 重点买入标的：关注抱团机会

### 港元流动性分析

**联系汇率制度**：
- 港元挂钩美元（7.75-7.85区间）
- 港元流动性受美联储政策影响

**流动性指标**：
- HIBOR（香港银行同业拆借利率）
- 港元汇率（靠近7.85=流动性紧）
- 港币兑美元走势

**对港股影响**：
- HIBOR上升：融资成本增加，压制估值
- 港元走强（靠近7.75）：流动性充裕，利好港股

## 8步分析框架（港股适配）

### Step 1: 基础数据收集

**港股特色数据**：
- AH溢价率
- 港股通标的身份
- 做空比例（融券余额/流通市值）
- 回购记录（近3个月）
- 大股东增持/减持

### Step 2: 财报分析（IFRS适配）

**港股财报特点**：
- 中报/年报为主（季报非强制）
- IFRS准则（与A股CAS略有差异）
- 中英文双语披露
- 关注ESG报告

**关键检查点**：
- 收入确认政策
- 商誉减值测试
- 关联交易披露
- 汇率敏感性分析

### Step 3: 异常指标识别

**港股特色红旗**：
- 做空比例 > 5%：高做空压力
- 大股东持续减持
- 频繁更换审计师
- 停牌时间过长

### Step 4-8: 标准流程

参考A股个股分析Skill的8步框架。

## 回购与做空监控

### 回购分析

**回购信号**：
- 连续回购：管理层认为股价低估
- 大额回购：强烈看好信号
- 回购注销 vs 回购用于激励

**回购计算公式**：
```
回购收益率 = 年度回购金额 / 市值
> 3%：积极回购
> 5%：非常积极
```

### 做空监控

**做空比例**：
- < 1%：正常水平
- 1-3%：有一定做空压力
- > 5%：高做空压力，需警惕

**做空信号**：
- 做空比例快速上升
- 股价下跌伴随做空增加
- 负面报告引发做空潮

## 中美缓和受益评估

### 受益板块

| 板块 | 受益逻辑 | 代表标的 |
|:-----|:---------|:---------|
| **互联网** | 监管放松预期 | 腾讯、阿里、美团 |
| **创新药** | 出海限制放松 | 药明生物、百济神州 |
| **半导体** | 技术封锁缓解 | 中芯国际、华虹半导体 |
| **跨境电商** | 关税降低预期 | 阿里国际、京东 |

### 风险评估

- 中美关系恶化：减仓出口链，增配高股息
- 港股作为中美缓和弹性标的：关系缓和时弹性大

## 风险控制规则

### 港股特色风险

| 风险类型 | 说明 | 应对措施 |
|:-----|:---------|:---------|
| **汇率风险** | 港元挂钩美元，人民币兑港元波动 | 考虑汇率对冲 |
| **流动性风险** | 小盘股流动性差 | 避免小盘股，选择港股通标的 |
| **政策风险** | 两地监管差异 | 关注两地政策变化 |
| **做空风险** | 港股做空机制成熟 | 监控做空比例 |

### 止损规则

- 个股止损：-20%（港股波动较大）
- 止损后观察：3日内不补仓
- 组合止损：-15%减仓至50%
