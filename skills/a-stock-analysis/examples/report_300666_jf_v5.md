## 三、产业链定位与竞争格局（✅ 修正版）

### 3.1 产业链定位

```
上游: 高纯金属原材料（铝、钛、钽、铜）
  ↓
中游: 【江丰电子】高纯溅射靶材制造 ← **国内绝对龙头**
  ↓
下游: 台积电/中芯国际/华虹/长江存储
```

### 3.2 竞争格局分析

**江丰电子的独特地位**:
- **A股唯一高纯溅射靶材龙头**，无直接同等规模竞争对手
- 主要竞争对手为**海外巨头**：霍尼韦尔（美）、东曹（日）、普莱克斯（美）
- 国内唯一实现**5nm/7nm先进制程靶材量产**的企业
