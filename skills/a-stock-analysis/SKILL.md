# A股个股分析Skill

## 核心能力

本Skill整合以下SOP方法：
- 股票深度分析SOP → 8步深度分析框架
- 财报分析检查SOP → 三表检查与杜邦分析
- 盘中监控SOP → 实时行情监控与异动识别
- 风险管理SOP → 止损止盈与仓位控制

## 使用场景

1. **个股深度研究** - 新标的全面分析
2. **持仓日常监控** - 每日盘中跟踪
3. **交易决策支持** - 买入/卖出时机判断
4. **财报季分析** - 季报/年报深度解读

## 快速开始

### 个股深度分析（完整8步法）

```
分析流程：
1. 基础数据收集 → 2. 财报初步分析 → 3. 异常指标识别
4. 深度追问"为什么" → 5. 行业与竞争分析 → 6. 重大事件分析
7. 情景分析与投资建议 → 8. 输出分析报告
```

**关键检查点**：
- 至少追问3层"为什么"
- 标记所有红旗指标
- 区分直接原因vs根本原因

### 实时监控流程

**早盘（9:30）**：
- 检查持仓竞价情况
- 确认是否触发止损/止盈

**盘中（每小时）**：
- 跟踪个股涨跌幅、成交量
- 监控主力资金流向
- 检查异常波动

**尾盘（14:30）**：
- 生成当日操作建议
- 制定次日预案

## 分析框架详解

### 8步分析法

详见 [references/analysis_framework.md](references/analysis_framework.md)

核心要点：
- 数据收集：基本信息 + 实时行情 + 财务三表 + 历史数据
- 杜邦分析：ROE = 净利润率 × 总资产周转率 × 权益乘数
- 异常识别：毛利率<5%、ROE<5%、负债率>70%等红旗指标
- 深度追问：现象→为什么→原因A→为什么→根本原因

### 财务检查清单

详见 [references/financial_checklist.md](references/financial_checklist.md)

检查维度：
- 盈利能力：ROE、ROA、毛利率、净利率
- 运营效率：应收周转天数、存货周转天数
- 偿债能力：资产负债率、流动比率
- 成长能力：收入增速、利润增速

### 风险控制规则

详见 [references/risk_management.md](references/risk_management.md)

核心规则：
- 个股止损：-15%（成长股）/-20%（价值股）
- 单股仓位上限：8%（核心）、5%（进攻）、3%（卫星）
- 组合回撤：-10%减仓至70%，-20%清仓

## 报告模板

### 深度分析报告结构

```markdown
# [股票名称]（[代码]）深度分析报告

## 一、执行摘要
- 投资评级：买入/持有/卖出
- 核心逻辑：（3句话）
- 目标价/止损位

## 二、公司概况
- 基本信息、主营业务、股权结构

## 三、财务分析
- 杜邦分析、历史趋势、异常指标

## 四、深度拆解（"为什么"部分）
- 每个异常指标的3层追问

## 五、行业分析
- 行业周期、竞争格局、产业链位置

## 六、投资建议
- 综合评分、操作策略、风险提示
```

## 关键指标速查

### 红旗指标（立即标记）

| 指标 | 危险阈值 | 信号 |
|:-----|:---------|:-----|
| 毛利率 | <5% 或持续下滑 | 🔴 |
| ROE | <5% 或负数 | 🔴 |
| 资产负债率 | >70% | 🔴 |
| 经营现金流/净利润 | <0.5 | 🟡 |
| 存货周转天数 | 持续增加 | 🔴 |

### 健康指标（积极信号）

| 指标 | 健康标准 | 信号 |
|:-----|:---------|:-----|
| ROE | >15% | 🟢 |
| 毛利率 | >30% | 🟢 |
| 净利率 | >10% | 🟢 |
| 经营现金流/净利润 | >1 | 🟢 |
