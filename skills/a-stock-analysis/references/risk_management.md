# 风险控制规则

## 风险层级

```
Level 1: 系统性风险（无法分散）
    ↓ 应对：仓位控制+对冲工具
    
Level 2: 行业风险（可部分分散）
    ↓ 应对：行业分散+轮动
    
Level 3: 个股风险（可分散）
    ↓ 应对：个股止损+仓位控制
    
Level 4: 操作风险（人为失误）
    ↓ 应对：纪律执行+复核机制
```

## 仓位控制规则

### 总仓位控制

| 市场环境 | 总仓位上限 | 现金比例 | 说明 |
|:---------|:----------:|:--------:|:-----|
| **牛市** | 90% | 10% | 积极进攻 |
| **震荡市** | 70% | 30% | 均衡配置 |
| **熊市** | 50% | 50% | 防御为主 |
| **恐慌市** | 30% | 70% | 等待机会 |

### 个股仓位控制

| 个股类型 | 单股上限 | 说明 |
|:---------|:--------:|:-----|
| **核心持仓** | 8% | 高确定性龙头 |
| **进攻持仓** | 5% | 高成长弹性 |
| **卫星持仓** | 3% | 主题博弈 |
| **试错持仓** | 2% | 新标的试水 |

### 同板块仓位上限
- 单一板块不超过25%
- 相关板块合计不超过35%

## 止损管理

### 止损类型

| 类型 | 触发条件 | 止损幅度 | 适用场景 |
|:-----|:---------|:--------:|:---------|
| **技术止损** | 跌破关键支撑位 | -8%~-15% | 趋势交易 |
| **时间止损** | 买入后X天未涨 | -5%~-10% | 事件驱动 |
| **基本面止损** | 基本面恶化 | 立即 | 所有持仓 |
| **情绪止损** | 板块恐慌性抛售 | -10%~-20% | 系统性风险 |

### 个股止损规则

| 标的类型 | 止损线 | 操作 |
|:---------|:-------|:-----|
| 成长股 | -25% | 检查基本面，恶化则卖出 |
| 价值股 | -20% | 检查基本面，恶化则卖出 |
| 高股息股 | -15% | 检查分红可持续性，恶化则卖出 |

### 止损执行纪律

**硬性规则**：
- 止损单必须在买入时设定
- 触及止损无条件执行
- 不补仓摊低成本（除非是计划中的分批建仓）
- 止损后24小时内不重新买入

### 移动止损

**盈利后上调止损**：
```
盈利10% → 止损线上移至成本价（保本）
盈利20% → 止损线上移至+10%
盈利30% → 止损线上移至+20%
盈利50% → 减仓1/3，剩余设移动止损
```

## 回撤管理

### 回撤分级应对

| 回撤幅度 | 风险等级 | 应对措施 |
|:---------|:--------:|:---------|
| 5%~10% | 🟡 正常波动 | 维持仓位，观察 |
| 10%~15% | 🟠 警惕 | 减仓至70%，检查持仓 |
| 15%~20% | 🔴 风险 | 减仓至50%，清仓弱势股 |
| >20% | ⚫ 危机 | 减仓至30%，保留现金 |

### 最大回撤控制

**组合层面**：
- 目标最大回撤：15%
- 预警线：-10%（减仓信号）
- 清仓线：-20%（无条件减仓）

**单日回撤控制**：
- 单日跌幅>3%：次日减仓10%
- 单日跌幅>5%：次日减仓20%
- 连续3日下跌：减仓至50%

## 买入前检查清单

### 基本面检查
- [ ] ROE > 10%（过去3年平均）
- [ ] 负债率 < 60%
- [ ] 经营现金流为正
- [ ] 无重大诉讼/担保
- [ ] 审计意见为标准无保留

### 估值检查
- [ ] PE < 行业平均或历史50%分位
- [ ] PB < 行业平均或历史50%分位
- [ ] 市值 > 100亿（流动性考虑）

### 技术面检查
- [ ] 不在历史高位（PE < 90%分位）
- [ ] 未跌破关键支撑位
- [ ] 成交量正常（非缩量阴跌）

## 持仓监控清单

### 每日检查
- [ ] 是否触及止损线
- [ ] 是否有重大利空新闻
- [ ] 板块是否发生系统性风险
- [ ] 是否触发止盈条件

### 每周检查
- [ ] 基本面是否有变化
- [ ] 估值是否进入危险区域
- [ ] 资金是否持续流出
- [ ] 技术面是否恶化

### 每月检查
- [ ] 财报预告是否miss
- [ ] 行业景气度是否变化
- [ ] 竞争格局是否恶化
- [ ] 政策风险是否增加

## 黑天鹅应对预案

### 场景1：AI泡沫破裂
- 触发：AI概念指数回撤>40%
- 应对：
  1. 清仓所有纯AI概念标的
  2. 保留有真实收入的AI应用层
  3. 美股仓位降至10%
  4. 增配债券至50%

### 场景2：地缘政治冲突升级
- 触发：战争/重大冲突事件
- 应对：
  1. 清仓新兴市场
  2. 减仓出口链
  3. 加仓黄金至15%
  4. 加仓中债至35%

### 场景3：美联储重启加息
- 触发：CPI>4%+美联储放鹰
- 应对：
  1. 清仓美债
  2. 减仓成长股
  3. 增配现金至15%
  4. 增配高股息防御

## 风险管理的铁律

1. **永远不要亏大钱** - 单笔亏损<5%，组合回撤<20%
2. **永远保留现金** - 至少10%现金应对突发
3. **永远不要满仓** - 满仓=赌博
4. **永远不要死扛** - 止损是保命
5. **永远不要追高** - 追高=接飞刀
6. **永远不要抄底** - 左侧布局需耐心
7. **永远不要All in** - 分散是免费午餐
8. **永远不要情绪化** - 纪律重于一切
