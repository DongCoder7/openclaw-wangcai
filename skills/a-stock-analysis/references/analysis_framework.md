# 8步分析框架详解

## 完整流程

```
Step 1: 基础数据收集
    ├── 基本信息：股票代码、名称、行业、市值
    ├── 实时行情：股价、PE、PB、换手率
    ├── 财务三表：利润表、资产负债表、现金流量表
    ├── 财务指标：ROE、毛利率、净利率、负债率
    ├── 历史数据：近3-5年关键财务数据
    └── 行业数据：行业平均PE、毛利率、增长率
    
Step 2: 财报初步分析（杜邦分析）
    └── ROE = 净利润率 × 总资产周转率 × 权益乘数
    
Step 3: 异常指标识别
    └── 标记红旗指标
    
Step 4: 深度追问"为什么"（至少3层）
    └── 现象 → 为什么 → 原因A → 为什么 → 根本原因
    
Step 5: 行业与竞争分析
    └── 五力模型 + 行业周期 + 产业链位置
    
Step 6: 重大事件/收购分析
    └── 事件概况、战略意图、财务影响、协同效应
    
Step 7: 情景分析与投资建议
    └── 乐观/中性/悲观三情景
    
Step 8: 输出分析报告
    └── 标准化报告结构
```

- [ ] 总市值、流通市值

### 实时行情
- [ ] 当前股价

### 财务数据
- [ ] 最新季度/年度财报
- [ ] 利润表（收入、成本、费用、利润）
- [ ] 资产负债表（资产、负债、权益）
- [ ] 现金流量表（经营、投资、筹资）

### 历史数据
- [ ] 近3-5年收入趋势
- [ ] 近3-5年利润趋势

- [ ] 近3-5年毛利率趋势

## Step 2 杜邦分析模板

| 指标 | 公式 | 当前值 | 行业平均 | 判断 |
|:-----|:-----|:------:|:--------:|:-----|
| ROE | 净利润/净资产 | __% | 10-15% | 优秀/一般/差 |
| 净利润率 | 净利润/营收 | __% | 5-15% | 高/中/低 |
| 总资产周转率 | 营收/总资产 | __次 | 0.5-1.0 | 快/慢 |
| 权益乘数 | 总资产/净资产 | __ | 1.5-2.5 | 高杠杆/低杠杆 |

### 商业模式判断
- **茅台模式**：高利润率 + 低周转 + 低杠杆
- **沃尔玛模式**：低利润率 + 高周转 + 低杠杆
- **银行模式**：低利润率 + 低周转 + 高杠杆
- **恶化模式**：低利润率 + 低周转 + 高杠杆 ⚠️

## Step 3 异常指标识别

### 红旗指标清单

| 指标 | 危险阈值 | 信号 |
|:-----|:---------|:-----|
| 毛利率 | <5% 或持续下滑 | 🔴 |
| ROE | <5% 或负数 | 🔴 |
| 净利率 | 为负 | 🔴 |
| 资产负债率 | >70% | 🔴 |
| 存货周转天数 | 持续增加 | 🔴 |
| 经营现金流/净利润 | <0.5 | 🟡 |
| 应收账款周转天数 | 持续增加 | 🟡 |
| 资产减值/营收 | >5% | 🔴 |

## Step 4 深度追问方法论

### 追问模板

```
现象 → 为什么？ → 原因A → 为什么？ → 原因B → 为什么？ → 根本原因
```

### 示例（以毛利率低为例）

**第1层追问：为什么毛利率低？**
- 营业成本 > 营业收入
- 成本倒挂

**第2层追问：为什么成本会倒挂？**
- 原材料涨价
- 产品跌价
- 售价无法覆盖成本

**第3层追问：为什么售价无法覆盖成本？**
- 下游需求疲软
- 行业内卷，价格战
- 产品同质化，缺乏定价权

**根本原因**：
- 滞胀困境：上游通胀 + 下游通缩
- 产业链位置劣势：纯加工企业
- 产品结构问题：低端产品占比高

## Step 5 行业分析框架

### 五力模型

| 力量 | 分析内容 |
|:-----|:---------|
| 供应商议价能力 | 成本压力？ |
| 购买者议价能力 | 定价权？ |
| 新进入者威胁 | 竞争加剧？ |
| 替代品威胁 | 被颠覆风险？ |
| 行业内竞争 | 价格战？ |

### 行业周期
- 导入期 / 成长期 / 成熟期 / 衰退期

### 产业链位置
- 上游（资源）/ 中游（制造）/ 下游（品牌）
- 微笑曲线位置

## Step 6 事件分析框架

| 维度 | 分析内容 |
|:-----|:---------|
| 事件概况 | 时间、金额、交易对手、标的 |
| 战略意图 | 大股东动机、战略目的 |
| 财务影响 | 收入、利润、资产负债改善预测 |
| 协同效应 | 成本降低、收入提升、产业链整合 |
| 情景分析 | 乐观/中性/悲观三种情景 |
| 关键风险 | 执行风险、市场风险、政策风险 |

## Step 7 情景分析模板

| 情景 | 概率 | 关键假设 | 财务结果 | 股价预期 |
|:-----|:----:|:---------|:---------|:--------:|
| **乐观** | 30% | 收购成功+行业景气 | 净利润3-5亿 | 目标价 |
| **中性** | 50% | 收购成功但整合困难 | 净利润0-1亿 | 目标价 |
| **悲观** | 20% | 收购失败/行业恶化 | 继续亏损 | 目标价 |

## Step 8 报告结构

```markdown
1. 执行摘要（TL;DR）
   - 投资评级
   - 核心逻辑（3句话）
   - 目标价/止损位

2. 公司概况
   - 基本信息
   - 主营业务
   - 股权结构

3. 财务分析
   - 杜邦分析
   - 历史趋势
   - 异常指标识别

4. 深度拆解（"为什么"部分）
   - 每个异常指标的3层追问
   - 根本原因总结

5. 行业分析
   - 行业周期
   - 竞争格局
   - 产业链位置

6. 事件/收购分析
   - 事件概况
   - 财务影响测算
   - 情景分析

7. 投资建议
   - 综合评分
   - 操作策略
   - 风险提示

8. 附录
   - 财务数据表格
   - 估值模型
   - 参考资料
```

## 分析时间分配建议

| 步骤 | 时间占比 | 说明 |
|:-----|:--------:|:-----|
| 数据收集 | 15% | 自动化工具为主 |
| 初步分析 | 20% | 杜邦分析+异常识别 |
| **深度追问** | **30%** | **核心步骤，不可压缩** |
| 行业分析 | 15% | 必要时查资料 |
| 事件分析 | 10% | 重大事项时 |
| 投资建议 | 10% | 综合判断 |
