## 缠论中枢正确定义与计算

### ⚠️ 重要：中枢定义

> **缠论中枢 = 连续三个次级别走势类型的重叠区间**
> 
> **中枢区间 = [max(三个低点), min(三个高点)]**

#### 构成要素
1. **分型**: 顶分型（5根K线高点最高）和底分型（5根K线低点最低）
2. **笔**: 顶底分型 + 至少1根K线间隔 = 1笔
3. **线段**: 至少3笔构成1个线段（走势类型）
4. **中枢**: 连续3个线段的重叠区间

#### 计算流程
```
K线数据
  ↓
识别分型（顶/底）
  ↓
构造笔（顶底交替）
  ↓
构造线段（3笔=1线段）
  ↓
计算中枢（3线段重叠区间）
  ↓
[中枢下轨, 中枢上轨]
```

#### 正确中枢代码
```python
def calculate_zhongshu(xianduan_list):
    """
    缠论中枢计算 (正确定义)
    中枢 = 连续三个次级别走势类型的重叠区间
    """
    if len(xianduan_list) < 3:
        return None
    
    # 取最近三个线段
    recent_3 = xianduan_list[-3:]
    
    # 计算重叠区间
    max_low = max([xd['low'] for xd in recent_3])   # 三个低点中的最大值
    min_high = min([xd['high'] for xd in recent_3]) # 三个高点中的最小值
    
    # 检查是否有重叠
    if max_low >= min_high:
        return None  # 趋势行情，无中枢
    
    return {
        'lower': max_low,   # 中枢下轨 (ZG)
        'upper': min_high,  # 中枢上轨 (ZD)
        'center': (max_low + min_high) / 2
    }
```

#### 实用版中枢（趋势行情中）
当严格中枢未形成时（趋势行情），使用价格密集区作为"类中枢"：
```python
def calculate_practical_zhongshu(df, window=20):
    """
    实用版中枢 - 用于趋势行情中
    找最近window根K线的高低点30%-70%分位区间
    """
    highs = df['high'].values[-window:]
    lows = df['low'].values[-window:]
    
    upper = np.percentile(highs, 70)
    lower = np.percentile(lows, 30)
    
    return {
        'lower': lower,
        'upper': upper,
        'center': (upper + lower) / 2
    }
```

#### 合理的中枢宽度
- **大盘指数**: 0.3% - 1.5% 为合理宽度
- **个股**: 1% - 3% 为合理宽度
- **小于0.2%**: 过窄，开盘波动即突破，无意义
- **大于5%**: 过宽，失去操作指导意义

# 顶分型 (压力位)

## 常见错误警示

### ❌ 错误1：中枢过窄
```python
# 错误：只用了12根K线的均值区间
zhongshu = [3880.49, 3883.41]  # 只有3个点，0.08%宽度
# 开盘波动0.1%就突破，无实际意义
```

### ✅ 正确：中枢宽度合理
```python
# 正确：基于3个线段的重叠区间
zhongshu = [3876.52, 3887.68]  # 11个点，0.29%宽度
# 开盘波动0.3%以上才突破，有操作价值
```

### ❌ 错误2：大周期中枢用于开盘
```python
# 错误：日线中枢4127点用于5分钟开盘
zhongshu_day = [3891.85, 4127.82]  # 日线中枢
# 当前3880点，距离上轨247点，无法指导开盘操作
```

### ✅ 正确：使用同级别中枢
```python
# 正确：5分钟中枢用于5分钟操作
zhongshu_5f = [3876.52, 3887.68]  # 5分钟中枢
# 当前3880点，在区间内部，可操作
```

## 开盘分析框架（专业版）

### 双级别分析体系

```
┌─────────────────────────────────────────────────────┐
│  30F级别（定当天强弱）                               │
│  ├── 中枢区间 → 多空分水岭                           │
│  ├── 三买三卖 → 趋势确认信号                         │
│  └── 分层支撑压力 → 目标位                           │
├─────────────────────────────────────────────────────┤
│  5F级别（日内分时操作）                              │
│  ├── 中枢区间 → 日内操作边界                         │
│  ├── 背驰判断 → 买卖点时机                           │
│  └── 次级别买卖点 → 精确入场                         │
└─────────────────────────────────────────────────────┘
```

### 分层支撑压力体系

| 层级 | 30F级别 | 5F级别 | 交易意义 |
|:---|:---|:---|:---|
| **第一支撑** | 中枢下沿 | 中枢下沿 | 破位=三卖确认 |
| **强支撑** | 前期低点+扩展位 | 底分型 | 一买区，等背驰 |
| **极限支撑** | 1.618扩展位 | 双级别背驰区 | 黄金坑，重仓吸 |
| **第一压力** | 中枢上沿 | 中枢上沿 | 减仓位 |
| **强压力** | 前期高点+均线 | 顶分型 | 反转确认/生死线 |

### 开盘交易计划模板

#### 一、30F级别（决定当天强弱）
```
中枢: [下沿, 上沿]
当前位置: 中枢上方/内部/下方

支撑位（下探目标）:
• 第一支撑（中枢下沿）
  开盘直接破、反抽不回 → 30F三卖确认
• 强支撑（前期低点+背驰位）
  到这里看 5F底背驰 才考虑低吸
• 极限支撑（1.618扩展位）
  30F+5F双背驰区，最安全低吸点

压力位（反弹目标）:
• 第一压力（中枢上沿/5F上沿）
  缩量到这里 → 减仓
• 强压力（30F中枢上沿+均线）
  放量站稳才叫反转，否则都是反弹
```

#### 二、5F级别（日内分时操作）
```
1️⃣ 低开/下探路径（最可能）:
   • 开盘破 第一支撑
     - 不回中枢 → 30F三卖确认
     - 第一目标: 强支撑
     - 5F底背驰+放量 → 日内小多
     - 无量反抽 → 继续下看极限支撑
   
   • 到极限支撑
     - 30F+5F双背驰 → 重仓低吸
     - 不背驰、直接破 → 放弃

2️⃣ 高开/弱反路径（小概率）:
   • 开盘在第一压力附近
     - 放量突破 → 看强压力
     - 缩量不过 → 逢高减仓
   
   • 反弹到强压力
     - 放量站稳+5F三买 → 反转成立
     - 缩量遇阻+顶背驰 → 清仓
```

#### 三、极简缠论点位表
```
┌─────────────────────────────┐
│  强支撑: XX、XX            │
│  弱支撑: XX                │
│  弱压力: XX                │
│  强压力: XX                │
└─────────────────────────────┘
```

#### 四、一句话口诀
```
破XX看XX，双背XX重仓吸；
过XX看XX，站稳反转否则撤。
```

#### 五、大反弹条件（概率<15%）
必须同时满足:
1. 开盘不跌破第一支撑
2. 放量突破第一压力→强压力
3. 5F连续三买、不背驰

更大概率: 先下探强支撑-极限支撑，再走弱反到第一压力-强压力

### 背驰判断要点

#### 底背驰（买入信号）
```python
# 价格创新低，但MACD不创新低
price_new_low = current < prev_low
macd_not_low = macd_current > macd_prev_low

if price_new_low and macd_not_low:
    signal = "底背驰"
    if volume_increasing:  # 放量确认
        action = "买入"
```

#### 顶背驰（卖出信号）
```python
# 价格创新高，但MACD不创新高
price_new_high = current > prev_high
macd_not_high = macd_current < macd_prev_high

if price_new_high and macd_not_high:
    signal = "顶背驰"
    action = "卖出"
```

### 三买三卖判定

#### 30F三买（强势做多）
```
条件:
1. 突破30F中枢上沿
2. 回踩不进入中枢
3. 5F出现底背驰或放量阳线

操作: 回踩确认后加仓
```

#### 30F三卖（清仓避险）
```
条件:
1. 跌破30F中枢下沿
2. 反抽不回到中枢内
3. 5F出现顶背驰或缩量

操作: 反抽不过清仓
```
