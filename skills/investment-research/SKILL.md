## 二、行业/宏观分析

### 2.1 通用供需分析

**核心问题**:
1. 供给端总供给量？来自哪些来源？
2. 需求端总需求量？增长驱动因素？
3. 平衡状态？
4. 趋势预测？
5. 投资机会？

### 2.2 供给估算方法
```
总供给 = 现有产能 + 在建产能 + 规划产能 - 淘汰产能
```

### 2.3 需求估算方法
```
总需求 = Σ(终端产品销量 × 单位产品消耗)
需求 = 基期需求 × (1 + 增速)^年数
```

### 2.4 供需平衡分析
```
缺口 = 需求 - 供给
比例 = 缺口 / 需求 × 100%
```

| 状态 | 条件 | 含义 |
|------|------|------|
| 🔴 供不应求 | 缺口>10% | 价格上涨 |
| 🟡 紧平衡 | ±10% | 景气度高 |
| 🟢 供过于求 | 缺口<-10% | 价格承压 |

### 2.5 行业分析模板
```
## 一、行业概览
- 定义
- 产业链结构

## 二、供给端
[来源|规模|占比|可靠性]

## 三、需求端
[场景|量|占比|增速]

## 四、供需平衡
[时期|供给|需求|缺口|状态]

## 五、产业链机会
[环节|标的|逻辑]

## 六、结论
```

## 四、个股分析 (参考 a-stock-analysis/h-stock-analysis)

### 4.1 分析维度
| 维度 | 内容 | 数据来源 |
|------|------|----------|
| 基本面 | 营收、利润、ROE | efinance |
| 估值 | PE、PB、PS | efinance |
| 资金 | 主力流入/流出 | 估算 |
| 技术 | 均线、趋势 | 价格数据 |
| 消息 | 公告、新闻 | web_fetch |

### 4.2 个股分析模板
```
## 基本面
- 营收/利润
- ROE、毛利率
- 增长率

## 估值
- PE/PB/PS
- 行业对比

## 资金面
- 主力资金
- 北向/南向资金

## 技术面
- 均线位置
- 趋势方向

## 结论
- 买入/卖出/持有
- 目标价
- 风险提示
```

### 4.3 财报分析
```
杜邦分析:
ROE = 净利润率 × 资产周转率 × 杠杆率
```

## 五、组合管理

### 5.1 组合构建
```
1. 行业配置: 分散度
2. 个股选择: 龙头优先
3. 仓位管理: 根据风险偏好
4. 再平衡: 定期调整
```

### 5.2 因子模型
| 因子 | 权重 | 说明 |
|------|------|------|
| 动量 | 20% | ret20, ret60 |

| 质量 | 25% | ROE, 毛利率 |
| 成长 | 15% | 营收增长 |
| 资金 | 15% | 主力流入 |

1. **数据优先**: 先用efinance获取实时数据
2. **来源必标**: 所有数据都要标注来源
3. **可靠性分级**: 区分高/中/低可信度
4. **逻辑闭环**: 供需→投资建议要有推导
5. **风控提醒**: 永远提示风险
6. **定期更新**: 数据要及时刷新
