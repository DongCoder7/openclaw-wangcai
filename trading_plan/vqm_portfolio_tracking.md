## 二、交易规则备忘录

### 2.1 T+1交易规则

| 操作 | 规则 | 说明 |
|:-----|:-----|:-----|
| **买入** | T日买入 | 当日成交，当日显示持仓 |
| **卖出** | T+1日可卖 | 买入后下一个交易日才能卖出 |
| **特殊情况** | 节假日顺延 | 周五买入，下周一才能卖 |

### 2.2 止损规则（严格风控版）

**目标：组合最大回撤 ≤ 8%**

**三层止损机制**：

#### Layer 1: 单股预警（-5%）
```
触发条件：当前价 ≤ 买入价 × 0.95

计算：
预警价 = 买入价 × 0.95

示例：
平安银行：买入价10.96 × 0.95 = 10.41元
若股价跌至10.41元或以下，触发预警，密切监控
```

#### Layer 2: 单股止损（-8%）
```
触发条件：当前价 ≤ 买入价 × 0.92

计算：
止损价 = 买入价 × 0.92

示例：
平安银行：买入价10.96 × 0.92 = 10.08元
若股价跌至10.08元或以下，立即止损（如可卖）
```

#### Layer 3: 组合止损（-6%减仓，-8%清仓）
```
组合回撤 = (当前总市值 - 初始资金) / 初始资金

- 若组合回撤 ≥ -6%：减仓至70%
- 若组合回撤 ≥ -8%：减仓至50%或清仓
```

#### 市场系统性风险止损
```
若上证指数单日跌 ≥ -3%：
- 次日开盘减仓至50%
- 暂停新增买入

若上证指数连续3日跌 ≥ -2%：
- 清仓观望
- 等待市场企稳
```

### 2.3 调仓规则

```
触发条件：每月最后一个交易日14:30后

执行步骤：
1. 重新计算所有股票综合得分（PE 60% + ROE 40%）
2. 生成新的前10名列表
3. 对比当前持仓，确定：
   - 保留股（在新前10名中）
   - 调出股（不在新前10名中）
   - 调入股（新进入前10名）
4. 生成交易清单：
   - 先卖出调出股（注意T+1限制）
   - 再买入调入股
5. 等权重再平衡
```
