## 1. 板块基本画像

### 1.1 产业链定位
- **板块名称**: AI硬件产业链（AI算力基础设施）
- **产业链定位**: 从芯片设计→制造→封装→服务器组装→数据中心部署的全链条
- **核心产品**: GPU/HBM/光模块/液冷系统/高端PCB/先进封装
- **技术节点**: 3nm→2nm→1.6nm工艺演进，HBM3E→HBM4，800G→1.6T光模块

### 1.2 核心驱动因素
| 驱动因素 | 重要性 | 当前状态 | 持续性 |
|:---------|:------:|:---------|:-------|
| **AI大模型算力需求** | ⭐⭐⭐⭐⭐ | 🟢爆发期 | 5年+ |
| **模型参数增长** | ⭐⭐⭐⭐⭐ | GPT-5万亿级参数 | 3-5年 |
| **推理算力需求** | ⭐⭐⭐⭐⭐ | 🟢快速起量 | 长期 |
| **国产替代** | ⭐⭐⭐⭐ | 🟢加速期 | 5年+ |
| **技术迭代** | ⭐⭐⭐⭐ | 🟢1年一代 | 持续 |

### 1.3 AI硬件链全景
```
AI大模型训练/推理需求
    ↓
├─ 云端训练（万卡集群）
│   ├─ GPU/ASIC芯片 ← 英伟达、AMD、华为昇腾
│   ├─ HBM高带宽存储 ← 海力士、三星、美光
│   ├─ 先进封装（CoWoS）← 台积电、长电科技
│   ├─ 光模块（800G/1.6T）← 中际旭创、新易盛
│   ├─ 液冷散热 ← 英维克、曙光数创
│   ├─ 高端PCB ← 沪电股份、深南电路
│   └─ 电源/连接 ← 欧陆通、鼎通科技
│
└─ 边缘推理（终端设备）
    ├─ 端侧AI芯片 ← 高通、联发科、苹果
    ├─ LPDDR5X/DDR5 ← 三星、海力士
    └─ 模组/散热 ← 国内配套
```
