# 新能源行业深度研究

> 研究日期：2026-02-12  
> 研究范围：光伏/风电/储能/锂电全产业链  
> 核心问题：产能过剩是否结束？何时迎来反转？哪些环节值得投资？

---

## 一、产业链总览图谱

```
新能源发电端
    │
    ├─ 光伏：硅料→硅片→电池→组件→电站
    │
    ├─ 风电：零部件→整机→风电场
    │
    └─ 储能：电池→PCS→系统集成→电站配储
    │
    ▼
新能源用电端
    │
    ├─ 锂电：材料→电池→新能源车/储能
    │
    ├─ 电网：特高压/智能电网
    │
    └─ 充电桩/换电
```

## 二、光伏产业链详解

### 2.1 产业链分工与价值分布

```
硅料（通威/大全） → 硅片（隆基/中环） → 电池（通威/爱旭） → 组件（隆基/晶科/天合/晶澳） → 电站（央企/民企）
     15%               20%                 15%                  25%                    25%
```

**各环节龙头对比**：

| 环节 | 龙头公司 | 市占率 | 毛利率 | 核心壁垒 | 当前状态 |
|:-----|:---------|:------:|:------:|:---------|:---------|
| **硅料** | 通威股份 | ~25% | 5-10% | 成本控制 | 过剩，价格战 |
| **硅片** | TCL中环、隆基绿能 | 各~20% | 0-5% | 规模+成本 | 产能过剩 |
| **电池** | 通威股份、爱旭股份 | ~15% | 0-3% | 技术迭代 | N型替代P型 |
| **组件** | 晶科能源、天合光能、隆基绿能、晶澳科技 | 各~10-15% | 5-8% | 品牌+渠道 | 一体化竞争 |

### 2.2 产能过剩分析

**2023-2024年：产能过剩最严重阶段**

| 环节 | 2024年产能 | 2024年需求 | 产能利用率 | 过剩程度 |
|:-----|:----------:|:----------:|:----------:|:--------:|
| 硅料 | 200万吨 | 120万吨 | 60% | 🔴 严重过剩 |
| 硅片 | 800GW | 500GW | 62% | 🔴 严重过剩 |
| 电池 | 600GW | 500GW | 83% | 🟡 结构性过剩 |
| 组件 | 800GW | 500GW | 62% | 🔴 严重过剩 |

**价格走势**：
- 硅料：从30万/吨跌至4万/吨（-87%）
- 硅片：从7元/片跌至1.5元/片（-79%）
- 组件：从2元/W跌至0.8元/W（-60%）

### 2.3 何时迎来反转？

**反转信号跟踪**：

| 信号 | 现状 | 反转阈值 |
|:-----|:-----|:---------|
| 价格企稳 | 仍在寻底 | 价格不再创新低 |
| 产能出清 | 刚刚开始 | 中小企业退出 |
| 库存下降 | 仍处高位 | 库存去化至1个月以下 |
| 盈利修复 | 大面积亏损 | 龙头恢复盈利 |
| 需求加速 | 平稳增长 | 增速>20% |

**预判**：
- **2025年**：继续产能出清，价格筑底
- **2026年下半年**：可能看到反转信号
- **2027年**：行业回归合理盈利

### 2.4 投资逻辑

**当前阶段：回避**

| 维度 | 分析 |
|:-----|:-----|
| 供需 | 严重过剩，价格战激烈 |
| 盈利 | 全产业链亏损 |
| 估值 | 虽低但可能是价值陷阱 |
| 催化剂 | 暂无，等待出清 |

**反转后值得关注的公司**：
- 通威股份：硅料+电池一体化，成本最低
- TCL中环：硅片技术领先，N型硅片占比高
- 晶科能源：组件全球龙头，品牌渠道强

## 三、锂电产业链详解

### 3.1 产业链分工

```
上游材料 → 中游电池 → 下游应用
    │         │          │
    ├─ 锂矿   ├─ 动力电池  ├─ 新能源车
    ├─ 正极   ├─ 储能电池  ├─ 储能系统
    ├─ 负极   └─ 消费电子  └─ 电动工具
    ├─ 电解液
    ├─ 隔膜
    └─ 铜箔/铝箔
```

### 3.2 各环节竞争格局

**正极材料**：

| 公司 | 市占率 | 主要路线 | 特点 |
|:-----|:------:|:---------|:-----|
| **容百科技** | ~15% | 三元高镍 | 技术领先 |
| **当升科技** | ~12% | 三元/铁锂 | 海外客户多 |
| **湖南裕能** | ~25% | 磷酸铁锂 | 成本领先 |
| 德方纳米 | ~15% | 磷酸铁锂 | 液相法工艺 |

**负极材料**：

| 公司 | 市占率 | 特点 |
|:-----|:------:|:-----|
| **贝特瑞** | ~20% | 天然+人造，龙头 |
| **杉杉股份** | ~15% | 人造石墨 |
| **璞泰来** | ~15% | 高端人造石墨 |

**电解液**：

| 公司 | 市占率 | 特点 |
|:-----|:------:|:-----|
| **天赐材料** | ~30% | 一体化成本优势 |
| **新宙邦** | ~15% | 海外客户多 |

**隔膜**：

| 公司 | 市占率 | 特点 |
|:-----|:------:|:-----|
| **恩捷股份** | ~35% | 全球龙头 |
| **星源材质** | ~15% | 干法+湿法 |

### 3.3 锂电池：宁德时代vs比亚迪

| 维度 | 宁德时代 | 比亚迪 |
|:-----|:---------|:-------|
| **市场地位** | 全球动力电池第一（37%） | 全球第二（16%），自用为主 |
| **技术路线** | 三元+铁锂，麒麟电池 | 铁锂为主，刀片电池 |
| **客户结构** | 特斯拉/宝马/大众/蔚来等 | 主要自用，外供有限 |
| **盈利水平** | 毛利率22%，净利率11% | 毛利率18%，净利率5% |
| **商业模式** | To B供应商 | 整车+电池一体化 |
| **出海进展** | 海外建厂（德国/匈牙利） | 海外销售增长快 |

### 3.4 产能过剩分析

**2024年：产能利用率约60-70%**

| 环节 | 产能利用率 | 过剩程度 |
|:-----|:----------:|:--------:|
| 正极 | 65% | 🟡 过剩 |
| 负极 | 70% | 🟡 过剩 |
| 电解液 | 60% | 🔴 过剩 |
| 隔膜 | 75% | 🟡 结构性过剩 |
| 电池 | 65% | 🔴 过剩 |

**价格走势**：
- 碳酸锂：从60万/吨跌至8万/吨（-87%）
- 磷酸铁锂电芯：从1元/Wh跌至0.4元/Wh（-60%）

### 3.5 投资逻辑

**当前阶段：回避或左侧布局**

**可关注的机会**：
1. **宁德时代**：全球龙头地位稳固，估值合理（PE 15-20倍）
2. **比亚迪**：整车+电池一体化，新能源车高增长
3. **天赐材料**：电解液成本优势，一体化布局
4. **恩捷股份**：隔膜龙头，盈利相对稳定

**需要回避**：
- 二线电池厂（亿纬锂能、国轩高科等）：竞争激烈，盈利困难
- 正极材料：同质化严重，价格战激烈
- 锂矿股：碳酸锂价格低迷，业绩承压

## 四、储能产业链详解

### 4.1 市场分类

| 类型 | 应用场景 | 占比 | 增速 |
|:-----|:---------|:----:|:----:|
| **表前储能** | 电网侧/电源侧配储 | 60% | 50%+ |
| **工商业储能** | 工厂/商业园区 | 25% | 100%+ |
| **户用储能** | 家庭光伏配套 | 15% | 30%（国内） |

### 4.2 储能系统成本结构

```
储能系统成本构成：
- 电池（60%）：宁德时代/亿纬/比亚迪
- PCS（15%）：阳光电源/上能电气
- BMS（10%）：派能科技/科华数据
- EMS（5%）：软件系统
- 其他（10%）：集装箱/温控等
```

### 4.3 竞争格局

**储能电池**：
- 宁德时代：全球第一，市占率40%+
- 比亚迪：第二，市占率15%+
- 亿纬锂能：第三，市占率10%+

**储能PCS/系统集成**：

| 公司 | 全球排名 | 特点 |
|:-----|:--------:|:-----|
| **阳光电源** | 第一 | 光伏逆变器+储能双龙头 |
| **Fluence** | 第二 | 美国公司 |
| **特斯拉** | 第三 | 美国公司 |
| **比亚迪** | 第四 | 电池+系统集成 |
| **海博思创** | 第五 | 国内集成商龙头 |
| **科华数据** | 国内前三 | PCS+系统集成 |

### 4.4 投资逻辑

**储能是新能源产业链中景气度最高的环节**

| 驱动因素 | 说明 |
|:---------|:-----|
| 政策驱动 | 新能源强制配储政策 |
| 经济性改善 | 电池成本下降，峰谷套利可行 |
| 电力市场化 | 辅助服务市场打开盈利空间 |

**重点关注公司**：
- **阳光电源**：光储双龙头，海外布局领先
- **宁德时代**：储能电池龙头
- **上能电气**：PCS黑马，增速快
- **科华数据**：PCS+系统集成

## 五、风电产业链详解

### 5.1 产业链结构

```
零部件 → 整机 → 风电场
    │       │       │
    ├─ 叶片  ├─ 金风  ├─ 三峡能源
    ├─ 齿轮箱 ├─ 远景  ├─ 龙源电力
    ├─ 发电机 ├─ 明阳  ├─ 华润电力
    ├─ 轴承   └─ 运达  └─ 中广核新能源
    └─ 塔筒
```

### 5.2 海风 vs 陆风

| 维度 | 陆上风电 | 海上风电 |
|:-----|:---------|:---------|
| 成本 | 0.2-0.3元/kWh | 0.3-0.4元/kWh |
| 增速 | 10-15% | 30-40% |
| 竞争格局 | 价格战激烈 | 壁垒较高 |
| 国内企业优势 | 一般 | 全球领先 |

### 5.3 投资逻辑

**风电整体处于成熟期，增长放缓**

**可关注的机会**：
1. **海上风电**：增速快，壁垒高
   - 东方电缆：海缆龙头
   - 中天科技：海缆+新能源
   - 明阳智能：海上风机龙头

2. **出海**：国内过剩，海外有空间
   - 金风科技：海外订单增长
   - 远景能源：未上市，海外布局好

3. **高股息运营商**：
   - 三峡能源：新能源运营商龙头
   - 龙源电力：风电运营龙头

## 六、当前周期位置判断

### 6.1 各环节周期位置

| 环节 | 周期位置 | 预判 |
|:-----|:---------|:-----|
| 光伏 | 🔴 下行期末期 | 2026年下半年可能触底 |
| 锂电 | 🟡 下行期中期 | 2025-2026年磨底 |
| 储能 | 🟢 成长期 | 高景气延续 |
| 风电 | 🟡 成熟期 | 平稳增长，海风是亮点 |

### 6.2 2026年投资策略

**进攻方向**：
- 储能（阳光电源、宁德时代储能业务）
- 海上风电（东方电缆、明阳智能）

**防守方向**：
- 风电运营商（三峡能源、龙源电力）
- 锂电龙头（宁德时代、比亚迪）

**逆向布局（左侧）**：
- 光伏硅料龙头（通威股份）：等待反转
- 光伏一体化组件（晶科能源）：等待反转

**坚决回避**：
- 二三线光伏/锂电企业（可能被淘汰）
- 锂矿股（碳酸锂价格低迷）

## 八、投资逻辑总结

### 8.1 新能源各细分赛道投资优先级

| 优先级 | 细分赛道 | 核心逻辑 | 代表公司 |
|:------:|:---------|:---------|:---------|
| ⭐⭐⭐⭐⭐ | **储能** | 高增长+确定性+出海 | 阳光电源、宁德时代 |
| ⭐⭐⭐⭐ | **海风** | 高增速+壁垒高 | 东方电缆、明阳智能 |
| ⭐⭐⭐ | **风电运营** | 高股息+稳定增长 | 三峡能源、龙源电力 |
| ⭐⭐⭐ | **锂电龙头** | 格局清晰+估值合理 | 宁德时代、比亚迪 |
| ⭐⭐ | **光伏（左侧）** | 等待反转 | 通威、晶科（小仓位） |
| ⭐ | **其他材料** | 产能过剩 | 回避 |

### 8.2 风险提示

1. **产能过剩风险**：光伏/锂电产能过剩可能持续更久
2. **技术迭代风险**：新技术（钙钛矿/固态电池）颠覆
3. **政策风险**：新能源补贴退坡、电价政策变化
4. **地缘政治风险**：海外贸易壁垒
5. **原材料价格波动**：碳酸锂/硅料价格大幅波动
