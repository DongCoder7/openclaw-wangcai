## 1. 量价关系基础 (10分钟)

### 1.1 核心原则

```
价格变动 + 成交量 = 趋势可信度
```

**高成交量** = 市场关注度高，信号可靠 ✅
**低成交量** = 市场参与度低，信号可疑 ⚠️

### 1.2 四种基本量价关系

| 价格 | 成交量 | 名称 | 含义 | 可靠性 |
|:-----|:-------|:-----|:-----|:-------|
| 上涨 | 增加 | 量价齐升 | 健康上涨 | ⭐⭐⭐⭐⭐ |
| 上涨 | 减少 | 量价背离 | 上涨乏力 | ⭐⭐☆☆☆ |
| 下跌 | 增加 | 放量下跌 | 恐慌抛售 | ⭐⭐⭐⭐☆ |
| 下跌 | 减少 | 缩量下跌 | 抛压减轻 | ⭐⭐⭐☆☆ |

**口诀**: "量价齐升是真涨，量价背离要警惕"

### 1.3 成交量均线

```
成交量5日均线 = 近5日成交量平均
量比 = 当日成交量 / 5日均量
```

| 量比 | 状态 | 含义 |
|:-----|:-----|:-----|
| > 2.0 | 极端放量 | 重大信号 |
| 1.5-2.0 | 明显放量 | 趋势确认 |
| 0.8-1.5 | 正常 | 观望 |
| 0.5-0.8 | 缩量 | 动能减弱 |
| < 0.5 | 极端缩量 | 变盘前兆 |

## 2. 关键量能形态 (15分钟)

### 2.1 放量突破 (Volume Breakout)

```
形态特征:
价格:         阻力位
              │
         ┌────┴────┐  ← 突破K线(大阳线)
         │         │
         └────┬────┘
              │
              
成交量:       │
              │    ┌───────┐
              │    │       │  ← 放量(是均量2倍以上)
              └────┘       └────

信号: 🟢🟢 大量资金推动，突破有效
操作: 突破确认后买入
```

**判断标准**:
- ✅ 成交量 > 5日均量 × 2
- ✅ 突破幅度 > 3%
- ✅ 突破后3天内站稳

**假突破警告**:
- ❌ 放量但未能站稳阻力位
- ❌ 成交量放大但价格涨幅很小

### 2.2 缩量整理 (Consolidation)

```
形态特征:
价格:    ┌───┐   ┌───┐   ┌───┐
        │   │   │   │   │   │  ← 横盘震荡
        └───┘   └───┘   └───┘
        
成交量:  │
        │    ┌─┐      ┌─┐
        └────┘ └──┘  └──┘      ← 成交量萎缩
        
信号: ⏸️ 洗盘或蓄势，等待方向选择
操作: 观望，等待放量突破
```

**两种缩量**:

| 类型 | 位置 | 后续走势 |
|:-----|:-----|:---------|
| 上涨中继缩量 | 上升趋势中 | 继续上涨概率大 |
| 下跌末端缩量 | 长期下跌后 | 可能即将见底 |

### 2.3 天量 (Extreme Volume)

```
特征: 成交量创近期新高(是均量3倍以上)
形态: 
成交量:  │
        │          ┌─────────┐
        │          │         │  ← 天量
        └────┬─────┘         └───
             正常量
```

**天量的含义因位置而异**:

| 位置 | 可能含义 | 操作 |
|:-----|:---------|:-----|
| 上涨末端 | 主力出货 | 🔴 警惕，考虑卖出 |
| 突破位置 | 资金抢筹 | 🟢 买入 |
| 下跌中段 | 恐慌抛售 | ⏸️ 观望，等待企稳 |
| 底部区域 | 资金抄底 | 👀 关注，准备买入 |

### 2.4 地量 (Extremely Low Volume)

```
特征: 成交量创近期新低(是均量1/3以下)
形态:
成交量:  │
        │  ┌─┐  ┌─┐  ┌─┐
        └──┘  └──┘  └──┘  ← 地量
```

**俗语**: "地量见地价"

| 位置 | 可能含义 | 操作 |
|:-----|:---------|:-----|
| 下跌末端 | 抛压耗尽 | 🟢 可能见底 |
| 上涨中段 | 洗盘结束 | ➡️ 继续持有 |
| 横盘末期 | 变盘前奏 | 👀 等待方向 |

## 3. 量价背离 (15分钟)

### 3.1 顶背离 (看跌信号)

```
价格走势:     高点1        高点2 (更高)
              │            │
              │            │
              
成交量走势:   │            │
              │  ┌──┐      │  ┌──┐ (更低)
              └──┘  └──────┘  └──┘

信号: 🔴 价格上涨但买盘不足，可能见顶
操作: 警惕，考虑减仓
```

**识别要点**:
- 价格创新高
- 成交量未创新高，反而减少
- 通常伴随MACD顶背离

### 3.2 底背离 (看涨信号)

```
价格走势:     低点1        低点2 (更低)
              │            │
              │            │
              
成交量走势:   │            │
              │  ┌──┐      │  ┌─┐ (持平或减少)
              └──┘  └──────┘  └──┘

信号: 🟢 价格下跌但卖盘减少，可能见底
操作: 关注，准备抄底
```

**识别要点**:
- 价格创新低
- 成交量未放大，反而减少
- 恐慌盘已经出清

### 3.3 量价背离可靠性

| 背离类型 | 单独出现 | +MACD背离 | +K线形态 |
|:---------|:---------|:----------|:---------|
| 顶背离 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| 底背离 | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |

## 4. 量能与K线、MACD结合

### 4.1 强势买入信号 (三因素共振)

```
条件:
1. K线形态: 大阳线/锤子线/看涨吞没
2. MACD: 金叉或零轴上方
3. 成交量: 放量(量比 > 1.5)

可靠性: ⭐⭐⭐⭐⭐
操作: 积极买入
```

### 4.2 弱势卖出信号 (三因素共振)

```
条件:
1. K线形态: 大阴线/流星线/看跌吞没
2. MACD: 死叉或零轴下方
3. 成交量: 放量下跌

可靠性: ⭐⭐⭐⭐⭐
操作: 果断卖出
```

### 4.3 警示信号 (需警惕)

```
条件:
1. K线形态: 新高但实体变小
2. MACD: 可能顶背离
3. 成交量: 缩量或异常放量

可靠性: ⭐⭐⭐
操作: 警惕，减仓观望
```

## 5. 实战判断流程

```
第1步: 看价格走势
  └─ 上涨/下跌/横盘

第2步: 看成交量
  ├─ 放量 → 信号可靠，关注
  ├─ 缩量 → 动能不足，警惕
  └─ 天量/地量 → 特殊信号，具体分析

第3步: 量价配合判断
  ├─ 量价齐升 → ✅ 健康，顺势操作
  ├─ 量价背离 → ⚠️ 警惕，准备反向操作
  ├─ 放量滞涨 → 🔴 出货信号
  └─ 缩量企稳 → 👀 观望，等待放量

第4步: 结合K线和MACD
  └─ 三因素确认 → 操作
```

### 练习2: 综合分析 (10分钟)

**场景1**: 股票A
- 价格: 连续上涨后出现十字星
- 成交量: 较昨日缩量60%
- MACD: 仍在零轴上方，但红柱缩小

**分析**:
1. K线: 十字星 = 多空平衡，可能反转
2. 量能: 缩量 = 上涨动能不足
3. MACD: 红柱缩小 = 多头动能减弱
4. 综合: ⚠️ 警惕信号，可能见顶

**场景2**: 股票B
- 价格: 长期下跌后放量大阳线
- 成交量: 是5日均量3倍
- MACD: 零轴下方金叉

**分析**:
1. K线: 大阳线 = 强势反弹
2. 量能: 天量 = 资金介入
3. MACD: 零轴下金叉 = 底部信号
4. 综合: 🟢 强烈买入信号，多因素共振
