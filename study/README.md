## 二、核心知识点（已收集）

### 2.1 K线基础 (Candlestick Basics)

**来源**: Investopedia - Candlestick Chart

#### K线组成部分
- **实体 (Real Body)**: 显示开盘价和收盘价之间的区域
  - 白色/绿色: 收盘价 > 开盘价 (看涨)
  - 黑色/红色: 收盘价 < 开盘价 (看跌)
  
- **影线 (Shadows)**: 显示当日最高价和最低价
  - 上影线: 最高价与实体顶部的距离
  - 下影线: 最低价与实体底部的距离

#### K线类型与含义
| K线形态 | 特征 | 市场含义 |
|:--------|:-----|:---------|
| 长阳线 | 长白色实体 | 强烈买入压力，看涨信号 |
| 长阴线 | 长黑色实体 | 强烈卖出压力，看跌信号 |
| 十字星 | 实体极小 | 多空平衡，可能反转 |
| 锤子线 | 小实体+长下影线 | 底部反转信号 |
| 流星线 | 小实体+长上影线 | 顶部反转信号 |

### 2.2 MACD指标详解

**来源**: Investopedia - MACD

#### MACD计算公式
```
MACD Line = 12周期EMA - 26周期EMA
Signal Line = MACD Line的9周期EMA
Histogram = MACD Line - Signal Line
```

#### MACD交易信号
| 信号类型 | 条件 | 操作建议 |
|:---------|:-----|:---------|
| 金叉 | MACD线上穿信号线 | 买入信号 |
| 死叉 | MACD线下穿信号线 | 卖出信号 |
| 零轴上方 | MACD > 0 | 多头趋势 |
| 零轴下方 | MACD < 0 | 空头趋势 |
| 顶背离 | 价格新高，MACD未新高 | 看跌背离 |
| 底背离 | 价格新低，MACD未新低 | 看涨背离 |

### 2.3 量能 (Volume) 分析

#### 量价关系原则
- **量价齐升**: 上涨趋势确认，健康信号
- **量价背离**: 上涨无量/下跌放量，警惕反转
- **缩量盘整**: 趋势休息，等待方向选择
- **放量突破**: 突破关键位置，趋势加速

## 三、多周期分析框架

### 3.1 周期搭配策略

| 交易风格 | 主要周期 | 确认周期 | 入场周期 |
|:---------|:---------|:---------|:---------|
| 日内交易 | 日线 | 15分钟 | 5分钟 |
| 短线交易 | 周线 | 日线 | 60分钟 |
| 波段交易 | 月线 | 周线 | 日线 |

### 3.2 周期共振原则
1. **大周期定方向**: 日线趋势决定多空基调
2. **中周期找结构**: 60分钟找支撑阻力
3. **小周期找入场**: 15/5分钟找精确入场点
