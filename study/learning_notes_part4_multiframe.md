## 1. 学习要点总结

### 为什么需要多周期分析？

**单周期的问题**:
- 只看日线: 入场点不精确
- 只看分钟线: 噪音多，假信号多
- 没有大局观: 容易逆势操作

**多周期的优势**:
- 日线定方向: 避免逆势
- 小时线找结构: 确定支撑阻力
- 分钟线找入场: 精确买卖点

### 时间周期搭配

| 分析目的 | 趋势周期 | 结构周期 | 入场周期 |
|:---------|:---------|:---------|:---------|
| 日内交易 | 60分钟 | 15分钟 | 5分钟 |
| 短线交易 | 日线 | 60分钟 | 15分钟 |
| 波段交易 | 周线 | 日线 | 60分钟 |

## 2. 三周期分析框架

### Step 1: 日线分析 (定方向)

**分析要点**:
1. 价格在MA20上方还是下方？
2. MACD在零轴上方还是下方？
3. 当前趋势是上涨/下跌/横盘？

**决策**:
- 日线多头 → 只做多，不做空
- 日线空头 → 只做空，不做多
- 日线横盘 → 观望或区间操作

### Step 2: 60分钟分析 (找结构)

**分析要点**:
1. 价格接近支撑位还是阻力位？
2. 是否出现企稳或滞涨信号？
3. MACD是否准备金叉或死叉？

**决策**:
- 接近支撑位 + 企稳 → 准备买入
- 接近阻力位 + 滞涨 → 准备卖出
- 中间位置 → 观望

### Step 3: 15分钟分析 (找入场)

**分析要点**:
1. K线形态(锤子线/吞没/十字星)？
2. MACD金叉或死叉确认？
3. 成交量是否配合？

**决策**:
- 三因素共振 → 执行交易
- 信号不一致 → 放弃或等待

## 3. 短期走势分析 checklist

### 买入前检查清单

**日线 (必须满足)**:
- [ ] 价格在MA20上方
- [ ] MACD在零轴上方或即将上穿
- [ ] 不是明显的下跌趋势

**60分钟 (必须满足)**:
- [ ] 回调到支撑位或均线
- [ ] 出现企稳K线
- [ ] MACD不处于强烈下跌中

**15分钟 (必须满足)**:
- [ ] 看涨K线形态
- [ ] MACD金叉或即将金叉
- [ ] 成交量放大
- [ ] 风险收益比 ≥ 1:2

### 卖出前检查清单

与买入相反，检查看跌信号。

## 5. 多周期MACD应用

### 理想的多头共振

```
日线: MACD在零轴上方 + 红柱
    ↓
60分钟: MACD金叉 + 由负转正
    ↓
15分钟: MACD金叉 + 放量
    ↓
结果: 强烈买入信号
```

### 理想的空头共振

```
日线: MACD在零轴下方 + 绿柱
    ↓
60分钟: MACD死叉 + 由正转负
    ↓
15分钟: MACD死叉 + 放量下跌
    ↓
结果: 强烈卖出信号
```

### 周期矛盾的处理

**场景**: 日线看涨，但小时线看跌
- 可能是上涨中的回调
- 可以等待小时线企稳后入场
- 不逆势做空
