# 多周期分析框架

> 掌握时间周期搭配，用大周期定方向，小周期找入场

---

## 一、为什么要多周期分析？

### 1.1 单周期的问题

| 问题 | 说明 |
|:-----|:-----|
| 噪音干扰 | 小周期假信号多 |
| 趋势不明 | 大周期入场点不精确 |
| 方向矛盾 | 不同周期方向不一致 |

### 1.2 多周期的优势

```
多周期分析 = 趋势方向 + 入场精度 + 风险控制
```

- **大周期定方向**: 避免逆势操作
- **中周期找结构**: 识别支撑阻力
- **小周期找入场**: 精确买卖点位

## 二、时间周期搭配

### 2.1 经典搭配方案

| 交易风格 | 趋势周期 | 结构周期 | 入场周期 | 持仓时间 |
|:---------|:---------|:---------|:---------|:---------|
| 超短线 | 15分钟 | 5分钟 | 1分钟 | 数分钟-数小时 |
| 日内交易 | 60分钟 | 15分钟 | 5分钟 | 数小时 |
| 短线交易 | 日线 | 60分钟 | 15分钟 | 1-5天 |
| 波段交易 | 周线 | 日线 | 60分钟 | 1-4周 |
| 长线投资 | 月线 | 周线 | 日线 | 数月-数年 |

### 2.2 三周期法则

**原则**: 只使用3个时间周期，过多会混乱

```
推荐比例: 1 : 4 : 16

示例1: 日线(1) : 60分钟(4) : 15分钟(16)
示例2: 60分钟 : 15分钟 : 5分钟
示例3: 周线 : 日线 : 60分钟
```

### 2.3 周期之间的关系

```
        大周期 (趋势)
            │
            │ 决定方向
            ↓
        中周期 (结构)
            │
            │ 寻找位置
            ↓
        小周期 (入场)
              
              精确执行
```

## 三、多周期分析流程

### 3.1 第一步: 大周期定方向

**操作**:
1. 打开最大周期图表(如日线)
2. 判断当前趋势:
   - 价格在MA20上方 + MACD在零轴上方 = 多头
   - 价格在MA20下方 + MACD在零轴下方 = 空头
   - 其他情况 = 震荡/不明

**规则**: 
- 多头趋势 → 只做多，不做空
- 空头趋势 → 只做空，不做多
- 震荡趋势 → 减少交易或区间操作

### 3.2 第二步: 中周期找结构

**操作**:
1. 切换到中间周期(如60分钟)
2. 识别关键位置:
   - 支撑位 (前期低点、趋势线)
   - 阻力位 (前期高点、趋势线)
   - 整理区间 (横盘区域)

**关注**:
- 价格接近支撑位 → 准备做多
- 价格接近阻力位 → 准备做空/减仓
- 价格突破结构 → 趋势延续信号

### 3.3 第三步: 小周期找入场

**操作**:
1. 切换到最小周期(如15分钟)
2. 等待确认信号:
   - K线形态 (锤子线、吞没等)
   - MACD信号 (金叉、背离)
   - 量能确认 (放量突破)

**入场条件** (以做多为例):
```
必须同时满足:
✓ 日线: 多头趋势
✓ 60分钟: 价格接近支撑位
✓ 15分钟: 出现看涨K线 + MACD金叉 + 放量
```

## 四、周期共振

### 4.1 什么是共振？

**定义**: 多个时间周期同时发出相同方向的信号

```
共振示意:

日线:    ────/\\───────  (上升趋势)
              │
60分钟:      /\\   /\\    (回调后上涨)
            /  \\ /  \\
              │
15分钟:      /\\          (金叉确认)
            /  \\
            
结果: 三周期共振，强烈做多信号
```

### 4.2 共振类型

| 共振类型 | 条件 | 可靠性 | 仓位建议 |
|:---------|:-----|:-------|:---------|
| 三周期同向 | 大中小周期同向 | ★★★★★ | 重仓(70-80%) |
| 两周期同向 | 任意两周期同向 | ★★★☆☆ | 轻仓(30-50%) |
| 周期矛盾 | 各周期方向不一 | ★★☆☆☆ | 观望 |

### 4.3 共振交易策略

**策略A: 趋势延续共振**

```
场景: 上涨趋势中，回调结束

日线: 上升趋势
    ↓
60分钟: 回调到MA20/支撑位，出现企稳K线
    ↓
15分钟: MACD金叉 + 放量阳线

操作: 做多
止损: 15分钟低点下方
止盈: 前高或1:2盈亏比
```

**策略B: 趋势反转共振**

```
场景: 长期趋势末端，出现反转信号

周线: 上涨末期，出现背离
    ↓
日线: 跌破上升趋势线，MACD死叉
    ↓
60分钟: 反弹无力，再次下跌

操作: 做空(或多头平仓)
止损: 反弹高点
止盈: 下方支撑位
```

## 五、多周期MACD应用

### 5.1 周期参数设置

| 周期 | MACD参数 | 用途 |
|:-----|:---------|:-----|
| 日线 | 12,26,9 | 判断主趋势 |
| 60分钟 | 12,26,9 | 找回调买点 |
| 15分钟 | 12,26,9 | 精确入场 |

### 5.2 MACD多周期信号

**最佳买入信号**:
```
日线MACD: 零轴上方金叉或持续向上
    ↓
60分钟MACD: 回调后金叉(零轴上方)
    ↓
15分钟MACD: 金叉确认
```

**最佳卖出信号**:
```
日线MACD: 零轴下方死叉或持续向下
    ↓
60分钟MACD: 反弹后死叉(零轴下方)
    ↓
15分钟MACD: 死叉确认
```

### 5.3 MACD背离多周期确认

**顶背离确认**:
```
日线: 价格新高，MACD未新高 (第一警告)
    ↓
60分钟: 价格新高，MACD明显走低 (确认)
    ↓
15分钟: MACD死叉 (执行卖出)
```

**底背离确认**:
```
日线: 价格新低，MACD未新低 (第一警告)
    ↓
60分钟: 价格新低，MACD明显走高 (确认)
    ↓
15分钟: MACD金叉 (执行买入)
```

## 六、实战分析步骤

### 6.1 每日分析流程

**开盘前 (9:15-9:25)**:
```
1. 查看日线大势
   - 判断当天多空基调
   - 关注关键支撑阻力位

2. 分析60分钟结构
   - 找到当天可能的高低点区域
   - 识别趋势还是震荡

3. 制定交易计划
   - 什么位置做多/做空
   - 止损止盈位置
```

**盘中 (9:30-15:00)**:
```
4. 观察15分钟入场信号
   - 等待计划中的位置
   - 确认K线形态 + MACD + 量能

5. 执行交易
   - 符合所有条件才入场
   - 严格止损
```

**收盘后 (15:00-16:00)**:
```
6. 复盘
   - 记录交易结果
   - 分析多周期信号是否正确
   - 总结经验教训
```

### 6.2 分析检查表

**大周期检查**:
- [ ] 日线趋势方向明确
- [ ] 日线MACD位置
- [ ] 关键支撑阻力位

**中周期检查**:
- [ ] 60分钟趋势与日线一致
- [ ] 价格位置(高位/低位/中位)
- [ ] 60分钟MACD状态

**小周期检查**:
- [ ] 15分钟K线形态
- [ ] 15分钟MACD信号
- [ ] 成交量配合
- [ ] 入场点风险可控

## 七、常见错误

### ❌ 错误1: 在小周期逆势操作
**问题**: 日线下跌趋势，却在5分钟找做多机会
**后果**: 小周期反弹有限，容易被大趋势吞没
**解决**: 大周期定方向，只做顺势交易

### ❌ 错误2: 周期混乱
**问题**: 同时使用太多周期(日线、60分、30分、15分、5分)
**后果**: 各周期信号矛盾，无法决策
**解决**: 固定使用3个周期

### ❌ 错误3: 忽视大周期
**问题**: 只看5分钟图频繁交易
**后果**: 被短期噪音干扰，胜率低
**解决**: 必须先看日线方向

### ❌ 错误4: 信号过于追求完美
**问题**: 要求所有周期完美共振才交易
**后果**: 错过很多机会
**解决**: 核心周期(日线+60分)确认即可
