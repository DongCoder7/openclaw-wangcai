## 三、多因子策略（核心！）

### 3.1 评分算法

```python
# 综合评分 = PE排名 × 60% + ROE排名 × 40%
# PE排名：PE越低，排名越靠前（1=最低）
# ROE排名：ROE越高，排名越靠前（1=最高）

score = pe_rank × 0.6 + roe_rank × 0.4
```

**逻辑解释**：
- **PE权重60%**: 估值便宜是首要条件
- **ROE权重40%**: 质量不能太差（ROE>10%）
- **综合得分越低越好**

## 五、最终策略定稿

### 5.1 策略名称：价值质量多因子策略（VQM）

**全称**: Value-Quality Multi-factor Strategy

### 5.2 策略规则

#### 选股条件
1. **基础筛选**: PE > 0（剔除亏损股）
2. **综合评分**: `score = PE排名×60% + ROE排名×40%`
3. **选股数量**: 得分最低的10只
4. **权重分配**: 等权重（每只10%）

#### 行业分散限制（风险控制）
```
单一行业占比 ≤ 30%
若某行业入选超过3只，仅保留得分最高的3只
```

✅ **完美分散！**

### 5.3 调仓规则

| 规则 | 设定 |

| **调仓频率** | 每月最后一个交易日 |
| **再平衡** | 每月重置等权重 |
| **止损线** | 个股下跌15%止损 |

| 行业风险 | 单一行业≤30% |

## 六、预期收益与风险

### 6.1 预期收益估算

**情景分析**（假设PE从11.28倍回归至历史中位数14倍）：

```
预期收益 = (14 - 11.28) / 11.28 = +24%
时间周期 = 6-12个月
年化收益 = 20-30%
```

**ROE贡献**：
- 组合平均ROE 19.8%
- 每年内生增长约15-20%

### 6.2 风险因素

| 风险 | 概率 | 影响 | 应对 |
|:-----|:----:|:----:|:-----|
| 价值陷阱（PE长期低迷） | 20% | 🔴 高 | 加入ROE质量因子 |
| 行业系统性风险 | 30% | 🟡 中 | 行业分散≤30% |
| 成长股碾压 | 40% | 🟡 中 | 接受短期跑输 |
