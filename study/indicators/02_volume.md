# 量能分析详解

> 掌握量价关系，用成交量验证价格走势的可靠性

---

## 一、量价关系基础

### 1.1 为什么量能重要？

**成交量 (Volume)** 代表市场参与者的活跃程度和资金流动：

- **高成交量** = 市场关注度高，趋势更可靠
- **低成交量** = 市场参与度低，趋势可能脆弱
- **成交量变化** = 揭示资金流向和趋势强度

### 1.2 量价关系基本原则

```
量价关系公式:
价格变动 + 成交量 = 趋势强度
```

| 组合 | 名称 | 含义 | 可靠性 |
|:-----|:-----|:-----|:-------|
| 价格上涨 + 成交量增加 | 量价齐升 | 健康上涨 | ★★★★★ |
| 价格上涨 + 成交量减少 | 量价背离 | 上涨乏力 | ★★★☆☆ |
| 价格下跌 + 成交量增加 | 放量下跌 | 恐慌抛售 | ★★★★☆ |
| 价格下跌 + 成交量减少 | 缩量下跌 | 下跌趋缓 | ★★★☆☆ |

## 二、关键量能形态

### 2.1 放量突破 (Volume Breakout)

```
形态特征:
价格:         阻力位
              │
         ┌────┴────┐  ← 突破K线
         │         │     (大阳线)
         └────┬────┘
              │
              
成交量:       │
              │    ┌───┐
              │    │   │  ← 放量
              └────┘   └───
              
含义: 大量资金推动价格突破关键位
操作: 突破确认后跟进
```

**突破有效性判断**:
- ✅ 成交量是均量的2倍以上
- ✅ 突破后3天内站稳
- ✅ 突破幅度 > 3%
- ❌ 假突破: 放量但未能站稳

### 2.2 缩量整理 (Consolidation)

```
形态特征:
价格:    ┌───┐   ┌───┐   ┌───┐
        │   │   │   │   │   │
        └───┘   └───┘   └───┘  ← 横盘整理
        
成交量:  │
        │    ┌─┐  ┌─┐  ┌─┐
        └────┘ └──┘ └──┘      ← 成交量萎缩
        
含义: 洗盘或蓄势，等待方向选择
操作: 观望，等待突破
```

**整理类型**:

| 类型 | 成交量 | 后续走势 |
|:-----|:-------|:---------|
| 上涨中继 | 缩量 | 向上突破概率大 |
| 下跌中继 | 缩量 | 向下突破概率大 |
| 顶部横盘 | 间歇放量 | 向下变盘概率大 |
| 底部横盘 | 间歇放量 | 向上变盘概率大 |

### 2.3 天量/地量

#### 天量 (Extreme High Volume)

```
特征: 成交量创近期新高(是均量3倍以上)

含义分析:
┌─────────────────────────────────────┐
│ 位置        │ 可能含义    │ 操作    │
├─────────────────────────────────────┤
│ 上涨末端    │ 主力出货    │ 卖出    │
│ 突破位置    │ 资金抢筹    │ 买入    │
│ 下跌中段    │ 恐慌抛售    │ 观望    │
│ 底部区域    │ 资金抄底    │ 关注    │
└─────────────────────────────────────┘
```

#### 地量 (Extreme Low Volume)

```
特征: 成交量创近期新低(是均量1/3以下)

含义分析:
┌─────────────────────────────────────┐
│ 位置        │ 可能含义    │ 操作    │
├─────────────────────────────────────┤
│ 下跌末端    │ 抛压耗尽    │ 准备买入 │
│ 上涨中段    │ 洗盘结束    │ 持有    │
│ 横盘末期    │ 变盘前奏    │ 等待    │
└─────────────────────────────────────┘

俗语: "地量见地价"
```

### 2.4 量价背离

#### 顶背离 (Bearish Divergence)

```
价格:    高点1        高点2 (更高)
         │            │
         │            │
         
成交量:  │            │
         │  ┌──┐      │  ┌──┐ (更低)
         └──┘  └──────┘  └──┘

含义: 价格上涨但买盘不足，可能见顶
操作: 警惕，考虑减仓
```

#### 底背离 (Bullish Divergence)

```
价格:    低点1        低点2 (更低)
         │            │
         │            │
         
成交量:  │            │
         │  ┌──┐      │  ┌──┐ (更低或持平)
         └──┘  └──────┘  └──┘

含义: 价格下跌但卖盘减少，可能见底
操作: 关注，准备抄底
```

## 三、不同周期的量能分析

### 3.1 分钟线量能

**特点**:
- 波动大，噪音多
- 对日内交易重要
- 需要结合盘口分析

**应用**:
```
5分钟放量突破 + 盘口大单买入 → 日内做多信号
15分钟缩量回调 + 支撑位 → 日内低吸机会
```

### 3.2 日线量能

**特点**:
- 趋势判断更可靠
- 反映主力资金行为
- 适合波段和趋势交易

**关键量能形态**:

| 形态 | 特征 | 意义 |
|:-----|:-----|:-----|
| 量增价涨 | 成交量逐步放大 | 健康上涨 |
| 量缩价涨 | 成交量萎缩 | 上涨乏力 |
| 量增价跌 | 放量下跌 | 恐慌抛售 |
| 量缩价跌 | 缩量下跌 | 抛压减轻 |

### 3.3 周线/月线量能

**特点**:
- 反映长期资金流向
- 信号更可靠但滞后
- 适合大周期趋势判断

## 四、量能与K线、MACD结合

### 4.1 强势买入信号

```
条件组合:
1. K线形态: 大阳线/锤子线/吞没形态
2. MACD: 金叉或零轴上方
3. 成交量: 放量(是均量2倍以上)

可靠性: ★★★★★
操作: 积极买入
```

### 4.2 弱势卖出信号

```
条件组合:
1. K线形态: 大阴线/流星线/吞没形态
2. MACD: 死叉或零轴下方
3. 成交量: 放量下跌

可靠性: ★★★★★
操作: 果断卖出
```

### 4.3 警示信号

```
条件组合:
1. K线形态: 新高但K线实体缩小
2. MACD: 可能背离
3. 成交量: 缩量或异常放量

可靠性: ★★★☆☆
操作: 警惕，减仓观望
```

## 五、量能指标

### 5.1 OBV (On Balance Volume)

**原理**: 用成交量验证趋势

```python
def calculate_obv(df):
    """
    计算OBV指标
    """
    obv = [0]
    
    for i in range(1, len(df)):
        if df['close'].iloc[i] > df['close'].iloc[i-1]:
            obv.append(obv[-1] + df['volume'].iloc[i])
        elif df['close'].iloc[i] < df['close'].iloc[i-1]:
            obv.append(obv[-1] - df['volume'].iloc[i])
        else:
            obv.append(obv[-1])
    
    df['obv'] = obv
    return df
```

**应用**:
- OBV上升 + 价格上升 = 趋势确认
- OBV背离 = 趋势可能反转

### 5.2 VWAP (Volume Weighted Average Price)

**原理**: 成交量加权平均价

```python
def calculate_vwap(df):
    """
    计算VWAP指标
    """
    typical_price = (df['high'] + df['low'] + df['close']) / 3
    df['vwap'] = (typical_price * df['volume']).cumsum() / df['volume'].cumsum()
    return df
```

**应用**:
- 价格在VWAP上方 → 多头主导
- 价格在VWAP下方 → 空头主导
- 日内交易常用参考

### 5.3 量比

**公式**:
```
量比 = 当日成交量 / 过去5日平均成交量
```

| 量比值 | 含义 | 操作 |
|:-------|:-----|:-----|
| > 2 | 显著放量 | 关注突破 |
| 1.5-2 | 温和放量 | 趋势延续 |
| 0.8-1.5 | 正常 | 观察 |
| 0.5-0.8 | 缩量 | 警惕 |
| < 0.5 | 严重缩量 | 观望 |

## 七、常见错误

### ❌ 错误1: 只看单日成交量
**问题**: 单日放量可能只是偶然
**解决**: 结合多日趋势和均线判断

### ❌ 错误2: 忽视流通盘大小
**问题**: 小盘股和大票的量不能简单比较
**解决**: 使用量比或换手率

### ❌ 错误3: 把对倒当真实成交
**问题**: 主力对倒制造成交量假象
**解决**: 结合盘口、K线形态综合判断
