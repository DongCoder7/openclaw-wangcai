# 量能分析 - 成交量与筹码分布

## 一、成交量的本质

### 1.1 什么是成交量

成交量（Volume）指在一定时间内成交的股票数量或金额。

**含义**：
- 市场参与度的度量
- 资金进出的痕迹
- 价格变动的确认或质疑

### 1.2 成交量的重要性

**量价关系核心原则**：

> "价是虚的，量是实的。无量上涨不可持续，巨量下跌必有蹊跷。"

**成交量验证价格**：
- 价涨量增 ✅ 上涨真实
- 价涨量缩 ⚠️ 上涨存疑
- 价跌量增 ❌ 恐慌抛售
- 价跌量缩 ✅ 抛压减轻

## 二、量价关系详解

### 2.1 上涨中的量价关系

| 量价组合 | 形态 | 含义 | 操作建议 |
|:---------|:-----|:-----|:---------|
| **价涨量增** | 阳线+放量 | 买盘积极，上涨健康 | 持有或加仓 |
| **价涨量平** | 阳线+持平 | 买盘一般，趋势延续 | 持有观望 |
| **价涨量缩** | 阳线+缩量 | 买盘不足，上涨乏力 | 警惕回调 |
| **价涨停量缩** | 涨停+无量 | 筹码锁定，继续看涨 | 持有 |

**典型场景**：

**底部放量上涨**：
```
价格
  │         ╭─╮
  │        ╱   ╲
  │       ╱     ╲     ← 放量突破
  │  ────╱───────╲────
  │      成交量柱状图
  │         ████
  │    ████ ████ ████
  └──────────────────────
含义：资金介入，底部确立
```

**顶部放量滞涨**：
```
价格
  │    ╭──╮
  │   ╱    ╲  ╭──╮
  │  ╱      ╲╱    ╲   ← 价格停滞
  │ ╱               ╲
  │───成交量───────────
  │ ██████████████████  ← 巨量
  │
  └──────────────────────
含义：出货嫌疑，警惕见顶
```

### 2.2 下跌中的量价关系

| 量价组合 | 形态 | 含义 | 操作建议 |
|:---------|:-----|:-----|:---------|
| **价跌量增** | 阴线+放量 | 恐慌抛售，可能见底 | 等待企稳 |
| **价跌量平** | 阴线+持平 | 抛压稳定，趋势延续 | 观望 |
| **价跌量缩** | 阴线+缩量 | 抛压减轻，接近底部 | 关注反弹 |
| **价跌停量增** | 跌停+放量 | 恐慌盘涌出 | 等待开板 |

**典型场景**：

**恐慌性放量下跌**（见底信号）：
```
价格
  │
  │    ╭──╮
  │   ╱    ╲
  │  ╱      ╲  ╭──╮
  │ ╱        ╲╱    ╲    ← 放量长阴
  │╱                  ╲
  │───成交量───────────
  │          ████████████  ← 天量
  │     ████
  └──────────────────────
含义：恐慌盘杀出，可能是最后一跌
```

**无量阴跌**（危险信号）：
```
价格
  │
  │    ╭─╮
  │   ╱   ╲  ╭─╮
  │  ╱     ╲╱   ╲  ╭─╮
  │ ╱              ╲╱  ╲
  │───成交量─────────────
  │     ██  ██  ██  ██   ← 持续缩量
  │
  └──────────────────────
含义：无人问津，阴跌不止
```

### 2.3 特殊量价形态

**天量天价**：
- 价格创新高，成交量也创新高
- 可能是顶部信号
- 后续如不能继续放量，可能见顶

**地量地价**：
- 价格创新低，成交量创阶段新低
- 可能是底部信号
- 市场交投清淡，抛压穷尽

**量价背离**：
- 价格创新高但成交量未创新高（顶背离）
- 价格创新低但成交量未创新低（底背离）
- 背离信号预示趋势可能反转

## 三、均量线分析

### 3.1 什么是均量线

均量线（Volume MA）是成交量的移动平均线。

**常用周期**：
- 5日均量（短期）
- 10日均量（中期）
- 20日均量（长期）

### 3.2 均量线的应用

**均量线金叉**：
```
成交量
  │
  │    ╱ 5日均量
  │   ╱
  │  ╱    ╱ 10日均量
  │ ╱    ╱
  │╱    ╱
  └───────────────────
      金叉
含义：量能转强，关注机会
```

**均量线死叉**：量能转弱，警惕风险

**均量线多头排列**：5日>10日>20日，量能充沛  
**均量线空头排列**：5日<10日<20日，量能萎缩

## 四、筹码分布分析

### 4.1 什么是筹码分布

筹码分布显示在不同价格区间的持仓成本分布。

**核心概念**：
- **密集区**：筹码集中，支撑/阻力强
- **发散区**：筹码分散，支撑/阻力弱
- **套牢盘**：高于现价的持仓（压力）
- **获利盘**：低于现价的持仓（支撑）

### 4.2 筹码分布形态

**单峰密集**：
```
筹码密度
  │        ╭──╮
  │       ╱    ╲
  │      ╱      ╲
  │     ╱   筹码  ╲
  │    ╱    密集区  ╲
  │   ╱              ╲
  └────────────────────────→ 价格
含义：成本集中，变盘在即
```

**双峰密集**：
- 两个筹码密集区
- 中间是筹码真空区
- 股价常在两峰之间震荡

**多峰分散**：
- 筹码分散在多个价位
- 多空分歧大
- 趋势不明朗

### 4.3 筹码分布的应用

**上峰不移，下跌不止**：
- 上方套牢盘多
- 每涨一点就有人解套卖出
- 上涨压力大

**下峰锁定，上涨不息**：
- 下方筹码稳定
- 主力未出货
- 上涨可持续

**筹码下移，主力出货**：
- 低位筹码减少
- 高位筹码增加
- 主力在派发

## 五、换手率分析

### 5.1 什么是换手率

换手率 = 成交量 / 流通股本 × 100%

**含义**：股票转手的频率，反映交投活跃度

### 5.2 换手率的判断标准

| 换手率 | 状态 | 含义 |
|:-------|:-----|:-----|
| < 1% | 低迷 | 无人问津，庄股或冷门股 |
| 1-3% | 正常 | 正常交易，随大盘波动 |
| 3-7% | 活跃 | 关注度高，可能有行情 |
| 7-10% | 高度活跃 | 强势股，主力运作 |
| > 10% | 异常活跃 | 可能是出货或换庄 |
| > 20% | 极高 | 次新股或重大消息 |

### 5.3 换手率的实战应用

**低位高换手**：
- 可能是主力建仓
- 关注后续走势确认

**高位高换手**：
- 可能是主力出货
- 警惕见顶风险

**涨停换手率低**：
- 筹码锁定良好
- 继续看涨

**涨停换手率高**：
- 多空分歧大
- 可能打开涨停

## 六、量能实战策略

### 6.1 放量突破策略

**买点**：
1. 股价突破关键阻力位
2. 成交量是近期平均的2倍以上
3. 突破后3天不回落

**止损**：
- 突破阳线最低点
- 或固定5-8%止损

### 6.2 缩量回调策略

**买点**：
1. 上升趋势中的回调
2. 成交量明显萎缩（较前期高点缩量50%+）
3. 在支撑位企稳

**逻辑**：缩量回调=洗盘而非出货

### 6.3 天量逃顶策略

**卖点**：
1. 股价处于高位
2. 放出近期天量
3. 价格滞涨或收长上影

**逻辑**：天量=主力出货

## 七、量能陷阱

### 7.1 常见量能陷阱

**对倒放量**：
- 主力自买自卖制造成交量
- 目的是吸引跟风盘
- 识别：放量但价格不动或异动

**缩量诱多**：
- 无量上涨后突然放量下跌
- 追高者被套

**放量滞涨**：
- 成交量大但价格不涨
- 出货信号

### 7.2 如何避免陷阱

1. **结合价格位置**：高位放量要警惕
2. **结合K线形态**：放量+长上影=危险
3. **结合筹码分布**：筹码上移=出货
4. **结合换手率**：异常高换手要警惕

## 八、总结

### 量能分析核心

1. **量是价的先行指标**：量在价先
2. **放量突破最可靠**：确认趋势延续
3. **缩量回调是机会**：洗盘而非出货
4. **天量天价要警惕**：可能是顶部
5. **筹码分布看主力**：锁定程度定方向
