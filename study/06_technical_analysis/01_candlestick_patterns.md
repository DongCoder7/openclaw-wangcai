# K线技术 - 形态识别与趋势判断

## 一、K线基础

### 1.1 K线的构成

K线（Candlestick）记录了一段时间内价格的四个关键数据：

```
         最高价
            │
     ┌──────┼──────┐
     │      │      │  ← 上影线
     │    ┌─┴─┐    │
     │    │   │    │  ← 实体（阳线红色/阴线绿色）
     │    └───┘    │
     │      │      │  ← 下影线
     └──────┼──────┘
            │
         最低价

开盘价 ──────→ 实体下沿（阳线）/ 实体上沿（阴线）
收盘价 ──────→ 实体上沿（阳线）/ 实体下沿（阴线）
```

**阳线**（红色）：收盘 > 开盘，上涨  
**阴线**（绿色）：收盘 < 开盘，下跌

### 1.2 K线实体大小含义

| 实体类型 | 特征 | 市场含义 |
|:---------|:-----|:---------|
| **大阳线** | 实体很长，涨幅大 | 买方强势，多头主导 |
| **大阴线** | 实体很长，跌幅大 | 卖方强势，空头主导 |
| **小阳线** | 实体短，微涨 | 买方略强，但力量有限 |
| **小阴线** | 实体短，微跌 | 卖方略强，但力量有限 |
| **十字星** | 实体极短 | 多空平衡，方向待选 |

### 1.3 影线含义

| 影线类型 | 市场含义 |
|:---------|:---------|
| **长上影线** | 上方压力大，冲高回落 |
| **长下影线** | 下方支撑强，探底回升 |
| **无影线（光头光脚）** | 单边走势极强 |

## 二、单根K线形态

### 2.1 重要单根K线

**锤头线（Hammer）**
```
形态特征：
    │
    │    ┌───┐
    │    │   │  ← 实体小（阳或阴）
    │    └───┘
    │      │
    │      │   ← 下影线长（实体2倍以上）
    │      │
    └──────┴────────
    
位置：下跌末期
含义：底部反转信号，买方开始反击
```

**吊颈线（Hanging Man）**
```
形态特征：同锤头线
位置：上涨末期
含义：顶部反转信号，警惕回调
```

**流星线（Shooting Star）**
```
形态特征：
    │      │
    │      │   ← 上影线长
    │      │
    │    ┌─┴─┐
    │    │   │  ← 实体小
    │    └───┘
    └──────────────

位置：上涨末期
含义：冲高回落，顶部信号
```

**倒锤头（Inverted Hammer）**
```
形态特征：同流星线
位置：下跌末期
含义：底部试探，可能反弹
```

### 2.2 十字星（Doji）

**普通十字星**：
```
    │
    │    ─┬─
    │     │
    │    ─┴─
    └────────
含义：多空平衡，趋势可能反转或中继
```

**长腿十字星**（上下影线都长）：
- 多空激烈争夺，方向待选

**蜻蜓十字星**（只有下影线）：
- 下方支撑强

**墓碑十字星**（只有上影线）：
- 上方压力大

## 三、K线组合形态

### 3.1 反转形态

**吞没形态（Engulfing）**

**看涨吞没**：
```
下跌中：        上涨中（看涨吞没）：
    │    ┌─┐        │    ┌─────┐
    │    │ │        │    │     │
    │  ┌─┴─┴─┐      │  ┌─┤     │
    │  │     │      │  │ │     │
    └──────────      └────────────
    阴线后大阳线完全包住前者
    
含义：买方力量压倒卖方，趋势可能反转向上
```

**看跌吞没**：阳线后大阴线完全包住前者，趋势可能反转向下

**晨星（Morning Star）**

```
下跌末期：
    │    ┌─┐      │      ┌───┐
    │    │ │      │      │   │
    │  ┌─┴─┴─┐  ┌─┼─┐  ┌─┴───┘
    │  │     │  │ │ │  │
    └────────────────────────────
      大阴线    十字星    大阳线
      
含义：下跌→犹豫→上涨，底部反转信号
```

**暮星（Evening Star）**：上涨末期的反转形态，与晨星相反

**穿刺线（Piercing Line）**
```
下跌中：
    │    ┌───┐
    │    │   │
    │  ┌─┼───┤
    │  │ │   │
    └───────────
      阴线后阳线收盘插入阴线实体一半以上
      
含义：买方反击，可能反弹
```

**乌云盖顶（Dark Cloud Cover）**：上涨中的看跌形态，与穿刺线相反

### 3.2 中继形态

**红三兵（Three White Soldiers）**
```
    │        ┌─┐
    │    ┌─┐ │ │
    │  ┌─┤ │ │ │
    │  │ │ │ │ │
    └─────────────
      连续三根阳线，逐步抬高
      
含义：上涨势头强劲，可能继续
```

**三只乌鸦（Three Black Crows）**：连续三根阴线，看跌

**上升三法（Rising Three Methods）**
```
上涨中：
    │  ┌─────┐
    │  │     │
    │  │ ┌─┐ │
    │  │ │ │ │
    │  │ ├─┤ │
    │  │ │ │ │
    │  │ └─┘ │
    │  │     │
    │  └─────┘
    └───────────
      大阳线后3根小阴线（不跌破第一根开盘）
      然后再一根大阳线
      
含义：上涨中继，洗盘后继续涨
```

## 四、趋势判断

### 4.1 趋势的定义

**上升趋势**：高点越来越高，低点也越来越高
```
价格
  │      ╭─╮
  │     ╱   ╲    ╭─╮
  │    ╱     ╲  ╱   ╲  ╭─╮
  │   ╱       ╲╱     ╲╱   ╲
  └────────────────────────────→ 时间
         低点抬高    高点抬高
```

**下降趋势**：高点越来越低，低点也越来越低

**横盘趋势**：高点和低点都在一个区间内

### 4.2 趋势线画法

**上升趋势线**：连接两个以上的低点
```
    │      ╱
    │     ╱  ← 上升趋势线
    │    ╱
    │   ╱
    └────────────────
```

**下降趋势线**：连接两个以上的高点

**有效突破**：
- 收盘价突破趋势线
- 突破时成交量放大
- 突破后3天不回到趋势线内

### 4.3 支撑与阻力

**支撑位**：价格下跌时可能止跌的位置
- 前期低点
- 密集成交区
- 整数关口
- 均线位置

**阻力位**：价格上涨时可能遇阻的位置
- 前期高点
- 密集成交区
- 整数关口
- 均线位置

**支撑与阻力的转换**：
```
         阻力（前高）
    │      ╭──╮
    │     ╱    ╲
    │    ╱      ╲    ╭──╮
    │   ╱   支撑 ───→│  │
    │  ╱ （前阻力）   └──┘
    └─────────────────────
```
突破阻力后，阻力变支撑；跌破支撑后，支撑变阻力。

## 五、经典形态

### 5.1 头肩顶/底

**头肩顶（顶部反转）**：
```
    │      ╭─╮
    │     ╱   ╲    ╭─╮  ← 右肩
    │    ╱     ╲  ╱   ╲
    │   ╱  头   ╲╱     ╲    ← 头（最高）
    │  ╱    ↑   ╱╲      ╲
    │ ╱   左肩 ╱  ╲  右肩 ╲
    │╱        ╱    ╲      ╲
    └──────────────────────────
              颈线（支撑线）
              跌破颈线 = 确认反转
```

**头肩底（底部反转）**：与头肩顶相反

### 5.2 双顶/双底

**双顶（M头）**：
```
    │      ╭──╮    ╭──╮
    │     ╱    ╲  ╱    ╲
    │    ╱      ╲╱      ╲
    │   ╱                 ╲
    └─────────────────────────
              颈线
```

**双底（W底）**：与双顶相反

### 5.3 三角形整理

**对称三角形**：
```
    │  ╭─╮
    │ ╱   ╲  ╭─╮
    │╱      ╲╱   ╲  ╭─╮
    │               ╲╱
    └─────────────────────
      高点降低，低点抬高
      突破方向决定后续走势
```

**上升三角形**：水平阻力 + 上升支撑 → 向上突破概率大  
**下降三角形**：下降阻力 + 水平支撑 → 向下突破概率大

## 六、K线实战技巧

### 6.1 多周期验证

**原则**：大周期定方向，小周期找买卖点

| 周期 | 用途 | 操作 |
|:-----|:-----|:-----|
| 日线/周线 | 判断大趋势 | 确定多空方向 |
| 60分钟 | 判断中期趋势 | 寻找入场点 |
| 15分钟 | 判断短期趋势 | 精确买卖点 |

**示例**：
- 周线上升趋势 + 日线回调到位 = 买入机会
- 周线下降趋势 + 日线反弹 = 卖出机会

### 6.2 量价配合

**上涨时的量价关系**：
- 价涨量增 ✅ 健康上涨
- 价涨量缩 ⚠️ 上涨乏力
- 价涨量平 ➡️ 震荡整理

**下跌时的量价关系**：
- 价跌量增 ❌ 恐慌抛售
- 价跌量缩 ✅ 抛压减轻
- 价跌量平 ➡️ 阴跌不止

### 6.3 使用原则

1. **形态需要确认**：单根K线不如组合可靠
2. **结合趋势**：顺势操作成功率更高
3. **注意位置**：同样形态在不同位置意义不同
4. **设置止损**：形态失败及时离场

### 6.4 常见误区

❌ **误区1**：只看形态不看位置  
✅ 正确：高位锤头≠低位锤头

❌ **误区2**：形态必须100%标准  
✅ 正确：允许一定变形，核心要素具备即可

❌ **误区3**：形态一定有效  
✅ 正确：形态是概率，非绝对，需止损

❌ **误区4**：只看K线不看量能  
✅ 正确：量价配合才能确认

## 七、总结

### K线分析核心

1. **单根K线看力量**：实体大小、影线长短
2. **组合形态看转折**：吞没、晨星、三兵等
3. **趋势判断看高低点**：上升通道、下降通道
4. **支撑阻力看关键位**：前高前低、密集区
5. **形态确认看量能**：放量突破才有效

### K线分析 checklist

- [ ] 确定当前趋势方向
- [ ] 识别关键支撑阻力位
- [ ] 观察K线形态（单根+组合）
- [ ] 验证成交量配合
- [ ] 多周期共振确认
- [ ] 设置止损位
