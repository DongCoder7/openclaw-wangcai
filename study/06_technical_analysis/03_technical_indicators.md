# 技术指标 - MACD/KDJ/RSI/布林带

## 一、MACD指标

### 1.1 MACD的构成

MACD（Moving Average Convergence Divergence）由三部分组成：

| 组成部分 | 说明 | 计算 |
|:---------|:-----|:-----|
| **DIF（快线）** | 短期与长期EMA差 | EMA(12) - EMA(26) |
| **DEA（慢线）** | DIF的平滑 | DIF的9日EMA |
| **MACD柱** | DIF与DEA差 | (DIF - DEA) × 2 |

### 1.2 MACD基本用法

**金叉与死叉**：

```
DIF上穿DEA = 金叉（买入信号）
DIF下穿DEA = 死叉（卖出信号）
```

**零轴的意义**：
- DIF > 0：多头市场
- DIF < 0：空头市场
- 零轴上方金叉：强势上涨信号
- 零轴下方金叉：弱势反弹信号

### 1.3 MACD实战信号

**1. 零轴上方金叉**（最强买入信号）
```
MACD
  │      DIF
  │     ╱
  │    ╱    DEA
  │   ╱    ╱
  │  ╱    ╱
  │ ╱    ╱   ← 金叉
  │╱____╱
  └──────────────
  零轴─────────────
含义：多头市场中的强势买点
```

**2. 底背离**（底部反转信号）
```
价格
  │
  │    ╭──╮    ╭──╮
  │   ╱    ╲  ╱    ╲  ← 价格新低
  │  ╱      ╲╱      ╲
  │╱                   ╲
  │
MACD │
  │    ╭──╮
  │   ╱    ╲  ╭──╮
  │  ╱      ╲╱    ╲   ← DIF未新低（背离）
  │╱
  └───────────────────
含义：下跌动能衰竭，可能反弹
```

**3. 顶背离**（顶部反转信号）
- 价格创新高，DIF未创新高
- 上涨动能衰竭，可能见顶

### 1.4 MACD使用要点

**优点**：
- 趋势跟踪效果好
- 信号相对可靠
- 适用于中长线

**缺点**：
- 滞后性明显
- 震荡市频繁假信号

**最佳参数**：
- 默认(12,26,9)适用于日线
- 短线可调整为(6,13,5)
- 长线可调整为(24,52,18)

## 二、KDJ指标

### 2.1 KDJ的构成

KDJ（随机指标）由三条线组成：

| 线 | 说明 | 计算 |
|:--|:-----|:-----|
| **K线** | 快线 | RSV的3日平滑 |
| **D线** | 慢线 | K的3日平滑 |
| **J线** | 敏感线 | 3K - 2D |

**RSV**（未成熟随机值）：
```
RSV = (收盘价 - N日内最低价) / (N日内最高价 - N日内最低价) × 100
```

### 2.2 KDJ的取值范围

| 区间 | 状态 | 含义 |
|:-----|:-----|:-----|
| > 80 | 超买区 | 可能回调 |
| 20-80 | 正常区 | 趋势延续 |
| < 20 | 超卖区 | 可能反弹 |

### 2.3 KDJ实战信号

**1. 金叉买入**
```
KDJ
100│
 80│    K线
    │   ╱
 50│  ╱  D线
    │ ╱  ╱
 20│╱___╱   ← 20以下金叉
    └────────────
含义：超卖区金叉，反弹概率大
```

**2. 死叉卖出**
- 80以上死叉，回调概率大

**3. J线极端值**
- J > 100：严重超买，短期见顶
- J < 0：严重超卖，短期见底

### 2.4 KDJ使用要点

**优点**：
- 灵敏，适合短线
- 超买超卖提示明确

**缺点**：
- 震荡市中假信号多
- 单边行情中持续超买/超卖

**使用技巧**：
- 20以下金叉买入，80以上死叉卖出
- J线<0是极佳买点，J线>100是极佳卖点
- 结合趋势使用效果更好

## 三、RSI指标

### 3.1 RSI的原理

RSI（Relative Strength Index）衡量价格上涨和下跌的相对强度。

**计算公式**：
```
RSI = 100 - 100/(1 + RS)
RS = N日内上涨幅度平均值 / N日内下跌幅度平均值
```

### 3.2 RSI的取值范围

| 区间 | 状态 | 操作 |
|:-----|:-----|:-----|
| > 80 | 严重超买 | 警惕回调 |
| 70-80 | 超买 | 考虑减仓 |
| 30-70 | 正常 | 持仓观望 |
| 20-30 | 超卖 | 考虑加仓 |
| < 20 | 严重超卖 | 警惕反弹 |

### 3.3 RSI实战信号

**1. 超买超卖**
```
RSI
100│────── 严重超买线
 80│────── 超买线
    │
 50│────── 中性线
    │
 20│────── 超卖线
  0│────── 严重超卖线
    └───────────
```

**2. 背离信号**
- 价格新低，RSI未新低 = 底背离（看涨）
- 价格新高，RSI未新高 = 顶背离（看跌）

**3. 中线穿越**
- RSI上穿50：转强信号
- RSI下穿50：转弱信号

### 3.4 RSI使用要点

**常用参数**：
- RSI(6)：短期，灵敏
- RSI(12)：中期，平衡
- RSI(24)：长期，稳定

**注意事项**：
- 单边行情中RSI可能持续超买/超卖
- 背离信号比超买超卖更可靠
- 结合价格形态使用

## 四、布林带（BOLL）

### 4.1 布林带的构成

| 组成部分 | 计算 |
|:---------|:-----|
| **中轨** | 20日移动平均线 |
| **上轨** | 中轨 + 2倍标准差 |
| **下轨** | 中轨 - 2倍标准差 |

### 4.2 布林带的基本用法

**轨道含义**：
- 价格通常在上下轨之间运行
- 触及上轨 = 相对高位
- 触及下轨 = 相对低位
- 突破轨道 = 趋势可能加速

**带宽变化**：
- 带宽收窄 = 变盘在即（波动率下降）
- 带宽扩大 = 趋势加速（波动率上升）

### 4.3 布林带实战信号

**1. 布林带收口**（变盘信号）
```
价格
  │  ╭──╮
  │ ╱    ╲  ╭──╮
  │╱      ╲╱    ╲  ╭──╮
  │                ╲╱  ╲
  │    布林带收窄 → 变盘
  └──────────────────────
含义：波动率降低，即将选择方向
```

**2. 布林带开口**（趋势加速）
```
价格
  │
  │    ╭──╮
  │   ╱    ╲
  │  ╱      ╲      ╭──╮
  │ ╱        ╲____╱    ╲___
  │╱  布林带开口 → 趋势加速
  └──────────────────────────
含义：趋势确立，顺势操作
```

**3. 触碰下轨买入**
- 价格在布林带下轨附近 = 相对低估
- 结合其他指标确认

**4. 触碰上轨卖出**
- 价格在布林带上轨附近 = 相对高估
- 结合其他指标确认

### 4.4 布林带使用要点

**优点**：
- 直观显示价格波动范围
- 自动适应波动率变化
- 支撑阻力作用明显

**缺点**：
- 单边行情中持续触碰轨道
- 参数固定可能不适应所有股票

**参数调整**：
- 高波动股票：可调整为(26, 2.5)
- 低波动股票：可调整为(10, 1.5)

## 五、均线系统

### 5.1 常用均线

| 均线 | 周期 | 用途 |
|:-----|:-----|:-----|
| MA5 | 5日 | 超短期趋势 |
| MA10 | 10日 | 短期趋势 |
| MA20 | 20日 | 中期趋势 |
| MA60 | 60日 | 中长期趋势 |
| MA120 | 120日 | 长期趋势 |
| MA250 | 250日 | 年线，牛熊分界 |

### 5.2 均线排列

**多头排列**（上涨趋势）：
```
价格
  │        ╱ MA5
  │       ╱
  │      ╱  MA10
  │     ╱
  │    ╱    MA20
  │   ╱
  │  ╱      MA60
  │╱
  └───────────────
含义：短期>中期>长期，强势上涨
```

**空头排列**（下跌趋势）：MA5<MA10<MA20<MA60

### 5.3 均线金叉死叉

**金叉买入**：
- MA5上穿MA10：短期金叉
- MA10上穿MA20：中期金叉
- MA20上穿MA60：长期金叉（牛熊转换）

**死叉卖出**：与金叉相反

## 六、指标组合使用

### 6.1 推荐组合

**组合1：趋势跟踪**
- MACD + 均线系统
- 适用：趋势明确的行情

**组合2：震荡操作**
- KDJ + RSI + 布林带
- 适用：震荡整理行情

**组合3：综合判断**
- MACD + KDJ + 成交量
- 适用：大多数行情

### 6.2 指标共振

**多重指标共振 = 高概率信号**

**买入共振**：
- MACD金叉
- KDJ 20以下金叉
- 价格触碰布林带下轨
- 成交量放大

**卖出共振**：
- MACD死叉
- KDJ 80以上死叉
- 价格触碰布林带上轨
- 成交量萎缩

### 6.3 指标冲突处理

当指标信号冲突时：
1. **看周期**：大周期指标优先
2. **看位置**：低位看涨，高位看跌
3. **看量能**：量价配合优先
4. **看趋势**：顺势操作

## 七、指标使用原则

### 7.1 核心原则

1. **指标是工具，不是圣杯**
   - 没有100%准确的指标
   - 指标只是概率优势

2. **多重确认**
   - 单指标信号不可靠
   - 2-3个指标共振才操作

3. **结合价格形态**
   - 指标+K线形态=更可靠
   - 指标+支撑阻力=更精确

4. **顺势操作**
   - 趋势向上时，以买入信号为主
   - 趋势向下时，以卖出信号为主

### 7.2 常见误区

❌ **过度依赖指标**：不看价格形态和基本面  
❌ **频繁调整参数**：追求完美参数不存在  
❌ **忽视市场环境**：震荡指标用在趋势市  
❌ **过度交易**：指标稍有变化就操作

### 四大指标速查表

| 指标 | 核心信号 | 适用场景 | 最佳参数 |

| **MACD** | 金叉/死叉、背离 | 趋势跟踪 | (12,26,9) |

| **RSI** | 30/70超买超卖 | 判断强弱 | (14) |

```
布林带看波动，收口变盘要警惕
均线看方向，多头排列持股好
```
