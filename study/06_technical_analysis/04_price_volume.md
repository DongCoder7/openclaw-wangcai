# 量价关系 - 背离识别与买卖点

## 一、量价背离的概念

### 1.1 什么是背离

背离（Divergence）指价格与指标（或成交量）方向不一致的现象。

**背离的本质**：价格趋势与内在动能的分离，预示趋势可能反转。

### 1.2 背离的类型

| 类型 | 价格 | 指标/量能 | 含义 |
|:-----|:-----|:----------|:-----|
| **顶背离** | 创新高 | 未创新高 | 上涨动能衰竭，可能见顶 |
| **底背离** | 创新低 | 未创新低 | 下跌动能衰竭，可能见底 |

---

## 二、量价背离详解

### 2.1 量价顶背离

**形态**：
```
价格
  │         ╭──╮
  │        ╱    ╲    ╭──╮
  │       ╱      ╲  ╱    ╲  ← 价格新高
  │      ╱        ╲╱      ╲
  │     ╱                  ╲
  │
成交量│
  │         ╭──╮
  │        ╱    ╲
  │       ╱      ╲    ╭──╮  ← 量能未新高（背离）
  │      ╱        ╲  ╱
  │     ╱          ╲╱
  └────────────────────────
含义：上涨缺乏量能支撑，警惕回调
```

**特征**：
- 价格创出新高
- 成交量较前期高点萎缩
- 买盘不足，上涨难以持续

**应对**：
- 减仓或获利了结
- 等待回调确认
- 不追涨

### 2.2 量价底背离

**形态**：
```
价格
  │    ╭──╮
  │   ╱    ╲    ╭──╮
  │  ╱      ╲  ╱    ╲  ← 价格新低
  │ ╱        ╲╱      ╲
  │╱                  ╲
  │
成交量│
  │    ╭──╮
  │   ╱    ╲
  │  ╱      ╲    ╭──╮  ← 量能未新低（背离）
  │ ╱        ╲  ╱
  │╱          ╲╱
  └────────────────────────
含义：抛压减轻，可能见底反弹
```

**特征**：
- 价格创出新低
- 成交量较前期低点萎缩
- 卖盘枯竭，下跌接近尾声

**应对**：
- 关注买入机会
- 等待确认信号（如放量阳线）
- 分批建仓

## 三、指标背离详解

### 3.1 MACD背离

**MACD顶背离**：
```
价格        MACD
  │         ╭──╮       DIF
  │        ╱    ╲     ╱
  │       ╱      ╲   ╱
  │      ╱   新高  ╲ ╱  未新高
  │     ╱          ╲╱
  │    ╱              ╲
  └────────────────────────
含义：上涨动能不足，可能回调
```

**MACD底背离**：价格新低，DIF未新低，看涨信号

**可靠性**：
- 周线级别背离 > 日线级别 > 小时级别
- 多次背离 > 单次背离

### 3.2 RSI背离

**RSI顶背离**：
- 价格创新高
- RSI(14)未创新高
- 上涨动能衰竭

**RSI底背离**：
- 价格创新低
- RSI(14)未创新低
- 下跌动能衰竭

**注意**：RSI在极端值（>80或<20）时的背离更可靠

## 四、背离的确认与操作

### 4.1 背离的确认信号

**顶背离确认**：
1. 价格跌破短期趋势线
2. 出现大阴线
3. 成交量放大（确认出货）
4. MACD死叉

**底背离确认**：
1. 价格突破短期下降趋势线
2. 出现大阳线
3. 成交量放大（确认资金入场）
4. MACD金叉

### 4.2 背离操作策略

**顶背离操作策略**：

| 阶段 | 操作 | 仓位 |
|:-----|:-----|:-----|
| 发现背离 | 减仓 | 减30-50% |
| 确认背离 | 清仓或大幅减仓 | 剩20%或清仓 |
| 跌破支撑 | 彻底离场 | 0% |

**底背离操作策略**：

| 阶段 | 操作 | 仓位 |
|:-----|:-----|:-----|
| 发现背离 | 关注，准备资金 | 0% |
| 确认背离 | 建仓 | 30-50% |
| 突破阻力 | 加仓 | 70-80% |

### 4.3 背离失败的应对

**背离不一定成功！**

**顶背离失败**：
- 价格继续上涨
- 量能重新放大
- 应对：认错买回，或等待下次机会

**底背离失败**：
- 价格继续下跌
- 应对：止损离场，等待下次背离

**止损设置**：
- 顶背离：买入阳线最低点
- 底背离：背离低点下方3-5%

## 五、买卖点的确定

### 5.1 买点选择

**第一类买点（趋势转折）**：
- 底背离确认后
- 放量突破下降趋势线
- 回踩支撑位不破

**第二类买点（趋势延续）**：
- 上升趋势中的回调
- 缩量回踩均线支撑
- 出现止跌K线形态

**买点确认信号**（满足2个以上）：
- [ ] 底背离
- [ ] 放量阳线
- [ ] MACD金叉
- [ ] KDJ 20以下金叉
- [ ] 触及布林带下轨
- [ ] 到达重要支撑位

### 5.2 卖点选择

**第一类卖点（趋势转折）**：
- 顶背离确认后
- 放量跌破上升趋势线
- 反弹阻力位不过

**第二类卖点（趋势延续中的调整）**：
- 急涨后的放量滞涨
- 触及重要阻力位
- 出现见顶K线形态

**卖点确认信号**（满足2个以上）：
- [ ] 顶背离
- [ ] 放量滞涨或放量下跌
- [ ] MACD死叉
- [ ] KDJ 80以上死叉
- [ ] 触及布林带上轨
- [ ] 到达重要阻力位

### 5.3 买卖点示例

**完整交易流程**：
```
买入
  │
  │    底背离发现 → 关注
  │         ↓
  │    放量阳线确认 → 建仓50%
  │         ↓
  │    回踩不破支撑 → 加仓30%
  │         ↓
  │    上涨趋势确立 → 持有
  │         ↓
  │    顶背离出现 → 减仓50%
  │         ↓
  │    跌破趋势线 → 清仓
  │
  └────────────────────────→ 时间
```

## 六、常见量价形态与买卖点

### 6.1 底部形态

**W底（双底）**：
```
价格
  │    ╭──╮    ╭──╮
  │   ╱    ╲  ╱    ╲
  │  ╱      ╲╱      ╲
  │ ╱        ↑       ╲
  │╱      买点        ╲
  └────────────────────────
买点：突破颈线
止损：第二底低点
```

**头肩底**：
- 左肩、头、右肩
- 买点：突破颈线
- 止损：右肩低点

**圆弧底**：
- 缓慢筑底过程
- 买点：放量突破平台
- 特点：成交量逐渐萎缩再放大

### 6.2 顶部形态

**M头（双顶）**：
- 两个相近的高点
- 卖点：跌破颈线
- 止损：第二顶高点

**头肩顶**：
- 左肩、头、右肩
- 卖点：跌破颈线
- 涨幅测量：头到颈线的距离

**圆弧顶**：
- 缓慢见顶过程
- 卖点：跌破重要支撑
- 特点：成交量逐步萎缩

### 6.3 持续形态

**上升三角形**：
```
价格
  │    ╭──╮
  │   ╱    ╲  ╭──╮
  │  ╱      ╲╱    ╲  ╭──╮
  │ ╱               ╲╱   ← 买点：突破上轨
  │╱_____________________
  │      支撑线
  └────────────────────────
```

**旗形整理**：
- 快速上涨后的小幅回调
- 买点：突破旗形上轨
- 特点：成交量萎缩后放大

## 七、综合交易系统

### 7.1 系统要素

一个完整的交易系统包含：

| 要素 | 说明 | 示例 |
|:-----|:-----|:-----|
| **选股** | 选择什么股票 | 趋势向上，量能健康 |
| **买点** | 何时买入 | 底背离+放量突破 |
| **仓位** | 买多少 | 首次50%，确认后加满 |
| **止损** | 何时离场（错误时） | 跌破支撑-5% |
| **止盈** | 何时离场（正确时） | 顶背离或目标位 |
| **跟踪** | 如何持仓 | 移动止损保护利润 |

### 7.2 交易规则示例

**买入规则**（必须全部满足）：
1. 股价处于上升趋势（MA20向上）
2. MACD在零轴上方或刚刚金叉
3. 出现底背离或缩量回调
4. KDJ在50以下金叉
5. 成交量较前期萎缩后再放大

**卖出规则**（满足1个即卖出）：
1. 出现顶背离
2. 跌破上升趋势线
3. 放量长阴线
4. 触及目标价位
5. 触发止损

### 7.3 仓位管理

**分批建仓**：
- 首次建仓：30-50%
- 确认信号：加仓至70-80%
- 趋势确立：满仓

**分批减仓**：
- 首次减仓：30-50%（顶背离）
- 确认转折：减仓至20-30%
- 跌破支撑：清仓

**风险控制**：
- 单笔最大亏损：总资金2%
- 单只股票最大仓位：总资金20%
- 总仓位控制：牛市80%，熊市30%

## 八、总结

### 核心要点

1. **背离是预警信号**：顶背离看跌，底背离看涨
2. **背离需要确认**：单根K线确认，不急于操作
3. **多重信号共振**：背离+形态+指标=高胜率
4. **严格止损**：背离可能失败，保护本金第一
5. **顺势操作**：大趋势向上，以做多为主

### 交易 checklist

**买入前检查**：
- [ ] 大趋势方向向上
- [ ] 出现底背离信号
- [ ] 有确认K线形态
- [ ] 成交量配合
- [ ] 止损位明确

**卖出前检查**：
- [ ] 是否出现顶背离
- [ ] 是否跌破关键支撑
- [ ] 成交量是否异常
- [ ] 是否达到目标位
- [ ] 是否触发止损

### 量价关系口诀

```
量价背离要警惕，顶背离卖底背离买
放量突破是机会，缩量回调可等待
形态确认再动手，严格止损保本金
顺势操作胜率高，复利增长是王道
```
