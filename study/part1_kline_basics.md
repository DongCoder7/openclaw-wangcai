## 1. K线四要素 (10分钟)

### 1.1 基础构成

```
        上影线 (最高价)
           │
         ┌─┴─┐ ← 实体 (开盘-收盘)
         │   │
         └─┬─┘
           │
        下影线 (最低价)
```

### 1.2 颜色含义

| 颜色 | 条件 | 含义 |
|:-----|:-----|:-----|
| 🟢 绿色/白色 | 收盘 > 开盘 | 买方强，看涨 |
| 🔴 红色/黑色 | 收盘 < 开盘 | 卖方强，看跌 |

### 1.3 实体大小

| 大小 | 判断标准 | 市场含义 |
|:-----|:---------|:---------|
| 大实体 | >价格范围70% | 趋势强烈 |
| 小实体 | <价格范围30% | 多空平衡 |
| 十字星 | 开盘≈收盘 | 犹豫不决 |

## 2. 单根K线形态 (20分钟)

### 2.1 大阳线 (强势看涨)

```
特征: 长实体，几乎无影线
形态:  ┌───────────┐
       │           │
       └───────────┘
信号: 🟢 强烈买入压力
位置: 任何位置都看涨
```

### 2.2 大阴线 (强势看跌)

```
特征: 长实体，几乎无影线
形态:  ┌───────────┐
       │           │
       └───────────┘
信号: 🔴 强烈卖出压力
位置: 任何位置都看跌
```

### 2.3 十字星 (犹豫不决)

```
特征: 实体极小，影线可长可短
形态:     │
        ┌─┴─┐
        │   │
        └─┬─┘
          │
信号: ⚪ 多空平衡
位置: 趋势末端可能反转
```

### 2.4 锤子线 (看涨反转)

```
特征: 小实体+长下影线(>2倍实体)
形态:        ┌───┐
            │   │
            └───┘
              │
              │
              │
信号: 🟢 下方支撑强劲
位置: 下跌趋势末端
```

**记住**: 长下影线 = 下方有支撑，买盘涌入

### 2.5 流星线 (看跌反转)

```
特征: 小实体+长上影线(>2倍实体)
形态:        │
             │
             │
          ┌──┴──┐
          │     │
          └─────┘
信号: 🔴 上方阻力强劲
位置: 上涨趋势末端
```

**记住**: 长上影线 = 上方有阻力，卖盘涌出

## 3. 组合形态 (15分钟)

### 3.1 看涨吞没 (Bullish Engulfing)

```
第一根: 小阴线
第二根: 大阳线，完全包住第一根

形态:        ┌───────────┐
         ┌───┤           │
         │   │           │
         └───┤           │
             └───────────┘

信号: 🟢🟢 强烈看涨反转
位置: 下跌后出现
```

### 3.2 看跌吞没 (Bearish Engulfing)

```
第一根: 小阳线
第二根: 大阴线，完全包住第一根

形态:    ┌───────────┐
        │           ├───┐
        │           │   │
        │           ├───┘
        └───────────┘

信号: 🔴🔴 强烈看跌反转
位置: 上涨后出现
```

### 3.3 早晨之星 (Morning Star)

```
三根K线:
1. 大阴线
2. 小实体(星线) - 多空平衡
3. 大阳线

形态:    ┌─────┐          ┌─────────┐
        │     │    ┌─┐   │         │
        │     │    │ │   │         │
        │     │    └─┘   │         │
        └─────┘          └─────────┘

信号: 🟢🟢 底部反转
位置: 下跌趋势末端
```

### 3.4 黄昏之星 (Evening Star)

```
三根K线:
1. 大阳线
2. 小实体(星线)
3. 大阴线

形态:    ┌─────────┐          ┌─────┐
        │         │    ┌─┐   │     │
        │         │    │ │   │     │
        │         │    └─┘   │     │
        └─────────┘          └─────┘

信号: 🔴🔴 顶部反转
位置: 上涨趋势末端
```

## 5. 实战判断流程

```
第1步: 看实体
  ├─ 大实体 → 趋势强烈
  └─ 小实体 → 多空平衡

第2步: 看影线
  ├─ 长下影 → 有支撑
  ├─ 长上影 → 有阻力
  └─ 无影线 → 趋势坚决

第3步: 看位置
  ├─ 趋势末端 + 反转形态 → 可能反转
  └─ 趋势中段 + 持续形态 → 趋势延续

第4步: 看组合
  └─ 多根K线组合 → 信号更强
```

## 6. 练习任务 (15分钟)

### 练习1: 形态识别 (5分钟)
目标: 在以下描述中判断K线形态

1. 收盘远高于开盘，实体很长，几乎没影线 → **大阳线**
2. 开盘收盘几乎相同，上下影线较长 → **十字星**
3. 小实体，长下影线，出现在下跌后 → **锤子线**
4. 小阴线后大阳线，阳线包住阴线 → **看涨吞没**
5. 上涨后出现，三根K线：大阳-小实体-大阴 → **黄昏之星**

### 练习2: 快速判断 (5分钟)
看到以下K线，你的第一反应是？

| 场景 | 你的判断 | 参考答案 |
|:-----|:---------|:---------|
| 大跌后出现锤子线 | ? | 可能反弹，关注买入 |
| 大涨后出现流星线 | ? | 可能见顶，考虑卖出 |
| 高位出现看跌吞没 | ? | 强烈卖出信号 |
| 低位出现看涨吞没 | ? | 强烈买入信号 |
| 横盘出现十字星 | ? | 继续观望，等待方向 |
