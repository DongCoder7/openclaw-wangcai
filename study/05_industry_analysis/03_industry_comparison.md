# 行业比较与轮动逻辑

## 一、行业比较框架

### 1.1 比较维度

| 维度 | 指标 | 说明 |
|:-----|:-----|:-----|
| **成长性** | 收入增速、利润增速 | 未来空间 |
| **盈利性** | ROE、毛利率、净利率 | 赚钱能力 |
| **估值** | PE、PB、PEG | 贵还是便宜 |
| **确定性** | 现金流质量、订单可见性 | 风险程度 |
| **催化剂** | 政策支持、技术突破 | 短期驱动 |

## 二、行业轮动逻辑

### 2.1 经济周期与行业轮动

**美林投资时钟**：

```
        经济增长
           │
    复苏期 │ 过热期
   ────────┼────────→
    衰退期 │ 滞胀期
           │
        经济下行
```

| 阶段 | 特征 | 领涨行业 |
|:-----|:-----|:---------|
| **复苏期** | 经济↑+通胀↓ | 金融、地产、可选消费 |
| **过热期** | 经济↑+通胀↑ | 资源、能源、原材料 |
| **滞胀期** | 经济↓+通胀↑ | 现金、防御性消费 |
| **衰退期** | 经济↓+通胀↓ | 债券、医药、公用事业 |

### 2.2 A股行业轮动特点

**季节性轮动**：
- **春季（1-3月）**：政策预期，科技股活跃
- **夏季（4-6月）**：业绩验证，消费股稳健
- **秋季（7-9月）**：中报行情，成长股占优
- **冬季（10-12月）**：估值切换，低估值修复

**事件驱动轮动**：
- 两会：政策受益板块
- 财报季：业绩超预期板块
- 美联储议息：利率敏感板块

## 三、行业配置策略

### 3.1 核心-卫星策略

```
组合结构：
核心仓位（60-70%）：长期持有优质行业
卫星仓位（30-40%）：轮动配置景气行业
```

**核心行业选择**：
- 长期成长确定性高
- 护城河深厚
- 如：消费、医药、科技龙头

**卫星行业选择**：
- 当前景气度高
- 催化剂明确
- 如：周期股、政策受益股

## 四、总结

### 行业投资 checklist

- [ ] 明确行业所处生命周期阶段
- [ ] 分析产业链位置和护城河
- [ ] 评估当前景气度和估值
- [ ] 识别短期催化剂
- [ ] 确定配置权重（核心/卫星）

### 行业轮动原则

1. **顺应趋势**：不要对抗经济周期
2. **提前布局**：在催化剂兑现前介入
3. **及时止盈**：轮动有周期，不可恋战
4. **核心坚守**：长期优质行业不丢底仓
