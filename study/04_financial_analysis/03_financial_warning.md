# 财务预警 - 舞弊识别与风险信号

## 一、为什么需要财务预警

### 1.1 财务造假的影响

- **瑞幸咖啡**：财务造假导致股价暴跌80%，退市
- **康美药业**：虚增货币资金887亿，最终被ST
- **乐视网**：持续造假，最终退市，投资者血本无归

### 1.2 财务预警的目标

1. **识别财务造假**：发现虚增收入、利润
2. **发现经营风险**：预警流动性危机、偿债风险
3. **评估持续经营能力**：判断公司是否可能破产

## 二、常见的财务舞弊手段

### 2.1 收入造假

**手段1：虚构收入**
- 伪造销售合同
- 与关联方进行虚假交易
- 提前确认收入（还未交付就确认）

**识别信号**：
- 收入增长远超行业平均
- 应收账款/收入比大幅上升
- 现金流/利润比持续偏低

**手段2：隐瞒收入**
- 体外循环（收入不入账）
- 关联交易非关联化

### 2.2 利润造假

**手段1：虚增利润**
- 少计提折旧摊销
- 少计提坏账准备
- 资本化费用（应费用化的支出转为资产）

**手段2：调节利润**
- 跨期调节（今年多计提，明年转回）
- 资产减值调节（需要业绩时减值，不需要时转回）

### 2.3 资产造假

**手段1：虚增资产**
- 虚增存货（盘点困难）
- 虚增在建工程（难以核实）
- 虚增货币资金（康美药业案例）

**手段2：隐瞒负债**
- 表外融资（不体现在报表的负债）
- 明股实债（实际是债务，但记为股权）

## 三、财务预警指标体系

### 3.1 盈利能力预警指标

| 指标 | 计算公式 | 预警信号 |
|:-----|:---------|:---------|
| **毛利率异常** | (收入-成本)/收入 | 大幅波动或高于同行过多 |
| **净利率下滑** | 净利润/收入 | 连续3年下滑 |
| **ROE下降** | 净利润/净资产 | 从高位快速下降 |

### 3.2 现金流预警指标

| 指标 | 计算公式 | 预警信号 |
|:-----|:---------|:---------|
| **经营现金流/净利润** | 经营现金流÷净利润 | < 0.5且持续多年 |
| **自由现金流** | 经营现金流-资本支出 | 连续多年为负 |
| **现金周期** | 存货周转+应收周转-应付周转 | 明显变长 |

**核心逻辑**：
- 赚利润不等于赚现金
- 长期利润有现金支撑才真实

### 3.3 资产质量预警指标

| 指标 | 计算公式 | 预警信号 |
|:-----|:---------|:---------|
| **应收账款/收入** | 应收账款÷营业收入 | 持续上升 |
| **存货/收入** | 存货÷营业收入 | 明显高于同行 |
| **商誉/净资产** | 商誉÷净资产 | > 30%风险大 |
| **固定资产周转率** | 收入÷固定资产 | 持续下降 |

### 3.4 偿债风险预警指标

| 指标 | 计算公式 | 危险阈值 |
|:-----|:---------|:---------|
| **资产负债率** | 总负债÷总资产 | > 70%高危 |
| **流动比率** | 流动资产÷流动负债 | < 1.0危险 |
| **速动比率** | (流动资产-存货)÷流动负债 | < 0.5危险 |
| **利息保障倍数** | EBIT÷利息支出 | < 1.0无法付息 |

## 四、财务舞弊识别模型

### 4.1 M-score模型（ Beneish模型）

由印第安纳大学教授Messod Beneish提出，通过8个财务比率识别盈利操纵。

**计算公式**：
```
M = -4.84 + 0.92×DSRI + 0.528×GMI + 0.404×AQI + 0.892×SGI + 0.115×DEPI - 0.172×SGAI + 4.679×TATA - 0.327×LVGI
```

**判断标准**：
- M > -1.78：可能存在盈利操纵
- M < -1.78：操纵可能性较低

**8个指标含义**：

| 指标 | 含义 | 操纵信号 |
|:-----|:-----|:---------|
| DSRI | 应收账款指数 | 应收账款增长>收入增长 |
| GMI | 毛利率指数 | 毛利率下降 |
| AQI | 资产质量指数 | 非实物资产增加 |
| SGI | 销售增长指数 | 收入异常高增长 |
| DEPI | 折旧指数 | 折旧率下降 |
| SGAI | 销售管理费用指数 | 费用率上升 |
| TATA | 应计项目/总资产 | 应计项目异常增加 |
| LVGI | 杠杆指数 | 杠杆率上升 |

### 4.2 Z-score模型（Altman模型）

由Edward Altman提出，用于预测企业破产概率。

**计算公式**：
```
Z = 1.2×X1 + 1.4×X2 + 3.3×X3 + 0.6×X4 + 1.0×X5
```

**变量定义**：
- X1 = 营运资本/总资产
- X2 = 留存收益/总资产
- X3 = EBIT/总资产
- X4 = 市值/负债账面价值
- X5 = 收入/总资产

**判断标准**：
- Z > 2.99：财务健康，破产风险低
- 1.81 < Z < 2.99：灰色地带，需警惕
- Z < 1.81：破产风险高

## 五、实用识别技巧

### 5.1 勾稽关系检查

**核心原理**：财务报表之间存在逻辑关系，造假会破坏这种关系。

**检查点1：利润表与现金流量表**
```
净利润 vs 经营现金流
- 正常情况下两者应大致同步
- 净利润↑但经营现金流↓：警惕利润造假
```

**检查点2：资产负债表与现金流量表**
```
现金变动 = 经营现金流 + 投资现金流 + 筹资现金流
- 两边应该相等
- 不等说明数据有问题
```

**检查点3：收入与应收账款**
```
收入增长 vs 应收账款增长
- 应收增长 > 收入增长：可能虚增收入
- 应收账款周转天数变长：回款困难或造假
```

### 5.2 异常信号清单

**必须警惕的信号**：

1. **高管频繁减持**：内部人不看好
2. **审计机构更换**：可能拒绝配合
3. **CFO频繁离职**：财务压力大
4. **关联交易激增**：利益输送风险
5. **在建工程长期挂账**：可能虚增资产
6. **存货盘点困难**：可能虚增存货
7. **预付款项异常**：可能资金占用
8. **其他应收款大增**：可能资金占用或造假

### 5.3 行业对比法

**与同行比较异常指标**：

| 比较维度 | 异常信号 |
|:---------|:---------|
| 毛利率 | 远高于/远低于同行 |
| 费用率 | 远低于同行（可能少计费用） |
| 应收周转 | 远低于同行（回款慢或造假） |
| 存货周转 | 远低于同行（滞销或虚增） |
| 资产负债率 | 远高于同行 |

## 六、经典案例分析

### 6.1 瑞幸咖啡造假案

**造假手段**：
- 虚构商品券业务
- 伪造银行流水
- 虚增收入约22亿元

**识别信号**：
- 单店收入远超星巴克
- 现金流与利润严重不匹配
- 增长速度违背商业逻辑

### 6.2 康美药业造假案

**造假手段**：
- 虚增货币资金887亿元
- 虚增收入约300亿元
- 虚增利润约40亿元

**识别信号**：
- 货币资金大存大贷（有钱还大量借款）
- 利息收入远低于应有水平
- 存货异常增长

### 6.3 乐视网造假案

**造假手段**：
- 虚增收入
- 关联交易非关联化
- 体外循环

**识别信号**：
- 关联交易占比过高
- 应收账款增长异常
- 持续亏损但收入高增长

## 七、自我保护清单

### 7.1 投资前的检查

**必查项目**：
- [ ] 最近3年审计意见（非标意见要警惕）
- [ ] 高管是否有被调查记录
- [ ] 应收账款/收入比是否持续上升
- [ ] 经营现金流是否持续低于净利润
- [ ] 毛利率是否大幅波动
- [ ] 存货增长是否远超收入增长
- [ ] 商誉是否过高
- [ ] 资产负债率是否过高

### 7.2 持有中的监控

**定期监控指标**：
- 季度财报勾稽关系
- 应收账款周转天数变化
- 经营现金流变化
- 关联交易变化
- 高管增减持动态

### 7.3 发现问题的应对

**发现预警信号时**：
1. **减仓**：先降低风险敞口
2. **深入研究**：核实信号真实性
3. **咨询专业意见**：审计、投行观点
4. **果断退出**：确认造假后不留恋

## 八、总结

### 财务预警的核心原则

1. **利润≠现金**：有利润没现金要警惕
2. **异常即信号**：与历史/同行差异大要深究
3. **关注质量**：资产质量比数量重要
4. **持续跟踪**：财务风险是动态变化的

### 快速识别口诀

```
收入要看现金收，利润要看质量优
应收账款别太高，存货积压要担忧
关联交易要小心，大存大贷是警报
审计意见要细看，高管离职要留神
现金流是生命线，持续为负要远离
```

### 安全投资原则

1. **不懂不做**：看不懂的财报不投
2. **疑罪从有**：有疑点宁可错过
3. **分散投资**：不要把鸡蛋放一个篮子
4. **设置止损**：发现风险及时退出
