# 估值模型 - PE/PB/PS/DCF方法详解

## 一、估值的本质

### 1.1 什么是估值

估值（Valuation）是确定一项资产**内在价值**的过程。对于股票而言，估值帮助我们判断：

- 当前股价是**贵**还是**便宜**？
- 合理的买入价格是多少？
- 未来上涨空间有多大？

### 1.2 估值的核心逻辑

所有估值方法都基于一个基本等式：

```
价值 = 未来现金流的折现值
```

**关键要素**：
- **未来现金流**：公司能赚多少钱？
- **折现率**：考虑到风险，未来钱的价值要打多少折扣？
- **时间**：钱是什么时候收到的？

## 二、相对估值法（倍数法）

相对估值法是最常用的估值方法，通过与同行业公司或历史数据比较来判断贵贱。

### 2.1 PE（市盈率）

**公式**：
```
PE = 股价 / 每股收益 = 总市值 / 净利润
```

**含义**：投资者愿意为每1元盈利支付多少钱

**示例**：
- 茅台股价1700元，每股收益70元 → PE = 1700/70 = **24倍**
- 意味着投资茅台需要24年回本（假设盈利不变）

**应用场景**：
| 情况 | 适用性 |
|:-----|:-------|
| 盈利稳定的公司 | ✅ 最适合 |
| 亏损公司 | ❌ 不适用（PE为负或无意义） |
| 周期股 | ⚠️ 需谨慎（盈利波动大） |
| 高成长公司 | ⚠️ 需结合PEG |

**如何判断PE高低**：

| PE区间 | 一般判断 | 注意事项 |
|:------:|:---------|:---------|
| < 10倍 | 低估或价值陷阱 | 检查盈利质量 |
| 10-20倍 | 合理区间 | 需看行业平均 |
| 20-30倍 | 偏高 | 需看成长性 |
| > 30倍 | 高估值 | 需看增长持续性 |
| > 50倍 | 泡沫风险 | 谨慎参与 |

**PE的变形**：

1. **静态PE** = 股价 / 上年度每股收益
2. **动态PE（TTM）** = 股价 / 最近4个季度每股收益
3. **预测PE** = 股价 / 预测明年每股收益

**推荐**：使用**TTM PE**（滚动市盈率），更能反映当前盈利状况。

### 2.2 PB（市净率）

**公式**：
```
PB = 股价 / 每股净资产 = 总市值 / 净资产
```

**含义**：投资者愿意为每1元净资产支付多少钱

**应用场景**：
| 情况 | 适用性 |
|:-----|:-------|
| 银行、保险等金融机构 | ✅ 最适合 |
| 重资产行业（地产、钢铁） | ✅ 适合 |
| 轻资产公司（互联网、软件） | ⚠️ 不太适用 |
| 亏损公司 | ✅ 可用 |

**如何判断PB高低**：

| PB区间 | 一般判断 |
|:------:|:---------|
| < 1倍 | 跌破净资产，可能低估 |
| 1-2倍 | 合理偏低 |
| 2-3倍 | 合理区间 |
| > 5倍 | 偏高 |
| > 10倍 | 轻资产高成长公司 |

**巴菲特的PB观点**：
> "以低于净资产的价格买入优质公司，是最安全的投资。"

**注意事项**：
- **账面价值≠真实价值**：净资产可能高估（商誉）或低估（土地）
- **ROE与PB的关系**：高ROE公司 deserve 高PB
- **破净（PB<1）≠便宜**：可能是资产质量差或盈利差

### 2.3 PS（市销率）

**公式**：
```
PS = 总市值 / 营业收入 = 股价 / 每股营收
```

**含义**：投资者愿意为每1元收入支付多少钱

**应用场景**：
| 情况 | 适用性 |
|:-----|:-------|
| 亏损但高增长公司 | ✅ 最适合 |
| 互联网/科技公司 | ✅ 适合 |
| 成熟稳定公司 | ⚠️ 不如PE |
| 强周期公司 | ⚠️ 收入波动大 |

**示例**：
- 亚马逊早期长期亏损，但收入高速增长 → 用PS估值
- 拼多多早期亏损 → 用PS估值

**PS的优缺点**：
- ✅ **优点**：不受盈利波动影响，适合成长早期公司
- ❌ **缺点**：不反映盈利能力，收入高≠赚钱

### 2.4 EV/EBITDA（企业价值倍数）

**公式**：
```
EV = 市值 + 总负债 - 现金 （企业价值）
EBITDA = 息税折旧摊销前利润
EV/EBITDA = 企业价值 / EBITDA
```

**含义**：不考虑资本结构和税收折旧因素，纯粹看经营价值

**应用场景**：
| 情况 | 适用性 |
|:-----|:-------|
| 高负债公司 | ✅ 比PE更准确 |
| 不同国家公司比较 | ✅ 不受税率影响 |
| 重资产折旧大 | ✅ 不受折旧政策影响 |
| 并购估值 | ✅ 常用指标 |

**与PE的区别**：
- PE看股东回报，EV/EBITDA看企业整体回报
- 高负债公司PE可能被低估，EV/EBITDA更准确

## 三、PEG（市盈率相对盈利增长比率）

### 3.1 公式

```
PEG = PE / 盈利增长率（G）
```

**示例**：
- 公司A：PE=30倍，增长率=30% → PEG = 1.0
- 公司B：PE=20倍，增长率=10% → PEG = 2.0

**结论**：虽然A的PE更高，但考虑增长后更便宜（PEG更低）

### 3.2 判断标准

| PEG区间 | 判断 |
|:-------:|:-----|
| < 0.5 | 严重低估 |
| 0.5-1.0 | 低估 |
| 1.0 | 合理估值 |
| 1.0-1.5 | 偏高 |
| > 2.0 | 高估 |

### 3.3 注意事项

1. **G的选取**：通常用未来3-5年的**预期**增长率
2. **增长率的持续性**：高增长能否持续？
3. **PEG不适用**：增长率为负或极不稳定时

## 四、绝对估值法：DCF（现金流折现）

### 4.1 DCF的核心思想

公司的价值等于其未来能产生的一切现金流的**现值之和**。

```
公司价值 = Σ [未来第t年的自由现金流 / (1+折现率)^t]
```

### 4.2 关键概念

**自由现金流（FCF）**：
- 公司经营活动产生的现金
- 减去维持运营必需的资本支出
- 可以自由分配给股东和债权人的现金

**公式**：
```
FCF = 息税前利润(EBIT) × (1-税率) + 折旧摊销 - 资本支出 - 营运资本增加
```

**折现率（WACC）**：
- 加权平均资本成本（Weighted Average Cost of Capital）
- 反映资金的时间价值和风险
- 风险越高，折现率越高，未来现金流的现值越低

### 4.3 DCF计算步骤

**步骤1：预测未来5年自由现金流**

假设某公司未来5年FCF预测（亿元）：
| 年份 | 1 | 2 | 3 | 4 | 5 |
|:-----|:---:|:---:|:---:|:---:|:---:|
| FCF | 10 | 12 | 14 | 16 | 18 |

**步骤2：确定折现率（WACC）**

假设WACC = 10%

**步骤3：计算预测期现值**

```
PV1 = 10 / (1+10%)^1 = 9.09
PV2 = 12 / (1+10%)^2 = 9.92
PV3 = 14 / (1+10%)^3 = 10.52
PV4 = 16 / (1+10%)^4 = 10.93
PV5 = 18 / (1+10%)^5 = 11.18

预测期现值合计 = 51.64亿元
```

**步骤4：计算终值（永续价值）**

假设第5年后公司永续增长率为3%：

```
终值 = FCF5 × (1+g) / (WACC - g)
     = 18 × 1.03 / (10% - 3%)
     = 18.54 / 7%
     = 264.86亿元

终值现值 = 264.86 / (1+10%)^5 = 164.41亿元
```

**步骤5：计算企业价值**

```
企业价值 = 预测期现值 + 终值现值
        = 51.64 + 164.41
        = 216.05亿元
```

**步骤6：计算每股价值**

```
每股价值 = (企业价值 - 净负债) / 总股本
```

### 4.4 DCF的优缺点

**优点**：
- ✅ 理论基础扎实，考虑时间价值和风险
- ✅ 不受市场情绪影响
- ✅ 可以清晰看到估值驱动因素

**缺点**：
- ❌ 参数敏感：增长率、折现率小幅变化导致估值大幅变化
- ❌ 预测困难：未来现金流难以准确预测
- ❌ 复杂度高：计算复杂，需要大量假设

**适用场景**：
- 现金流稳定可预测的公司（公用事业、消费）
- 不适合：初创公司、周期股、转型期公司

## 五、不同行业的估值偏好

### 5.1 行业估值方法对照表

| 行业 | 首选估值 | 次选估值 | 原因 |
|:-----|:---------|:---------|:-----|
| **银行** | PB | PE | 高杠杆，看净资产 |
| **保险** | P/EV | PB | 内含价值法 |
| **券商** | PB | PE | 周期性强，资产驱动 |
| **地产** | NAV | PB | 净资产价值法 |
| **白酒** | PE | PEG | 盈利稳定，看增长 |
| **医药** | PE/PEG | PS | 研发管线估值 |
| **互联网** | PS | PEG | 早期看收入 |
| **半导体** | PEG | PS | 高成长 |
| **新能源** | PEG | PS | 高成长 |
| **公用事业** | DCF | PE | 现金流稳定 |

### 5.2 成长股 vs 价值股的估值差异

| 维度 | 成长股 | 价值股 |
|:-----|:-------|:-------|
| 估值重点 | PEG、PS | PE、PB |
| 关注指标 | 收入增速、市占率 | ROE、分红率 |
| 合理PE | 30-50倍 | 10-20倍 |
| 风险 | 增长不达预期 | 价值陷阱 |

## 六、估值实战技巧

### 6.1 多重估值法

**不要只用一种估值方法！**

建议组合使用：
1. **PE看盈利**：当前盈利水平
2. **PB看资产**：安全边际
3. **PEG看成长**：成长性是否合理
4. **DCF看内在价值**：长期价值锚点

### 6.2 历史分位数法

将当前估值与历史估值比较：

| 分位数 | 判断 | 操作建议 |
|:------:|:-----|:---------|
| < 20% | 历史低估区间 | 积极买入 |
| 20-40% | 偏低 | 逐步建仓 |
| 40-60% | 合理 | 持有观望 |
| 60-80% | 偏高 | 减仓止盈 |
| > 80% | 历史高位 | 警惕风险 |

**示例**：
- 茅台当前PE 25倍，历史分位数15% → 低估
- 茅台当前PE 45倍，历史分位数85% → 高估

### 6.3 同业比较法

与同行业公司比较估值水平：

| 公司 | PE | PB | ROE | 判断 |
|:-----|:--:|:--:|:---:|:-----|
| 茅台 | 25 | 8 | 30% | 龙头溢价合理 |
| 五粮液 | 20 | 6 | 25% | 相对低估 |
| 泸州老窖 | 18 | 5 | 22% | 估值偏低 |

**注意**：比较时要考虑公司质地差异，龙头 deserve 溢价。

## 七、估值陷阱与误区

### 7.1 常见估值陷阱

**陷阱1：低PE陷阱**
```
公司A：PE=5倍，看似便宜
→ 但盈利持续下滑，明年可能亏损
→ 价值陷阱！
```

**陷阱2：高成长陷阱**
```
公司B：增长率50%，PEG=1，看似合理
→ 但高增长不可持续，竞争加剧
→ 增长断崖风险！
```

**陷阱3：跨行业比较**
```
银行PE=6倍，科技股PE=40倍
→ 不能直接说银行更便宜
→ 不同行业估值中枢不同
```

### 7.2 估值的正确打开方式

1. **先看生意模式**：好公司 deserve 高估值
2. **再看成长性**：成长是估值的核心变量
3. **最后看估值**：在好公司和好价格之间平衡
4. **安全边际**：估值是艺术，留足安全边际

## 八、巴菲特的估值智慧

### 8.1 巴菲特的估值原则

1. **以企业所有者角度估值**：不要只看股价
2. **关注内在价值**：价值=未来现金流的折现
3. **能力圈原则**：只估值自己能理解的公司
4. **安全边际**：买入价格要有折扣

### 8.2 巴菲特的估值方法

**简单但有效的公式**：
```
合理市值 = 可理解的未来5-10年盈利能力 × 合理PE
```

巴菲特不追求精确计算，而是追求**模糊的正确**：
- 不需要算出公司值100亿还是110亿
- 只需要判断是明显低估（值50亿）还是高估（值200亿）

## 九、总结

### 核心估值方法速查表

| 方法 | 公式 | 适用场景 | 核心关注 |
|:-----|:-----|:---------|:---------|
| **PE** | 股价/每股收益 | 盈利稳定公司 | 盈利质量 |
| **PB** | 股价/每股净资产 | 银行、重资产 | 资产质量 |
| **PS** | 市值/收入 | 高成长公司 | 收入增长 |
| **PEG** | PE/增长率 | 成长股 | 增长持续性 |
| **EV/EBITDA** | 企业价值/EBITDA | 高负债、跨国比较 | 经营效率 |
| **DCF** | 现金流折现 | 现金流稳定公司 | 预测准确性 |

### 估值的本质

估值不是精确的数学计算，而是：
1. **定性的生意判断**：这门生意值多少钱？
2. **定量的比较分析**：相对历史/同行贵还是便宜？
3. **概率的期望计算**：在不同情景下的价值区间

**记住**：宁要模糊的正确，不要精确的错误。
