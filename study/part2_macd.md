## 1. MACD原理 (10分钟)

### 1.1 一句话理解

**MACD = 快速均线 - 慢速均线**

反映价格趋势的动量和方向变化。

### 1.2 计算公式

```
MACD线 = 12日EMA - 26日EMA
信号线 = MACD线的9日EMA
柱状图 = MACD线 - 信号线
```

**不用死记公式，记住**: MACD衡量的是短期趋势相对长期趋势的变化。

### 1.3 图表展示

```
价格          MACD区域
 │            │
 │            │  ┌──┐  ← MACD线(蓝)
 │            │  │  │     │
 │            └──┘  └────┘
 │                  ↑ 信号线(红/橙)
 │
 └──────────────

零轴 ──────────────────── (0线)
        │
        │  多头区域 (MACD > 0)
        │
        │  空头区域 (MACD < 0)
```

## 2. 金叉与死叉 (15分钟)

### 2.1 金叉 (买入信号)

```
定义: MACD线从下向上穿越信号线
形态: 
        MACD
         \\   /
          \\ /
           X  ← 交叉点(买入)
          / \\
         /   \\\_________ Signal
```

**三种金叉，信号强度不同**:

| 类型 | 位置 | 强度 | 操作 |
|:-----|:-----|:-----|:-----|
| 零轴上方金叉 | MACD > 0 | ⭐⭐⭐ | 上涨中继，可加仓 |
| 零轴附近金叉 | MACD ≈ 0 | ⭐⭐⭐⭐ | 趋势转折，积极买入 |
| 零轴下方金叉 | MACD < 0 | ⭐⭐⭐⭐⭐ | 底部反转，强烈买入 |

**口诀**: "零上金叉强，零下金叉更强"

### 2.2 死叉 (卖出信号)

```
定义: MACD线从上向下穿越信号线
形态:
        MACD
        \\\\   /
         \\\\ /
          X  ← 交叉点(卖出)
         / \\\\
        /   \\\\\
_______/         Signal
```

**三种死叉，信号强度不同**:

| 类型 | 位置 | 强度 | 操作 |
|:-----|:-----|:-----|:-----|
| 零轴上方死叉 | MACD > 0 | ⭐⭐⭐⭐⭐ | 顶部反转，强烈卖出 |
| 零轴附近死叉 | MACD ≈ 0 | ⭐⭐⭐⭐ | 趋势转折，积极卖出 |
| 零轴下方死叉 | MACD < 0 | ⭐⭐⭐ | 下跌中继，继续观望 |

**口诀**: "零上死叉狠，零下死叉更狠"

### 2.3 零轴的意义

```
           零轴 (0线)
             │
  MACD > 0   │   MACD < 0
  ───────────┼───────────
   多头市场   │   空头市场
   逢低做多   │   逢高做空
```

| 位置 | 趋势 | 策略 |
|:-----|:-----|:-----|
| MACD > 0 | 多头 | 只做多，不做空 |
| MACD < 0 | 空头 | 只做空，不做多 |
| 上穿零轴 | 转多 | 买入信号 |
| 下穿零轴 | 转空 | 卖出信号 |

## 3. 背离识别 (15分钟)

### 3.1 什么是背离？

**定义**: 价格走势与指标走势不一致

**核心逻辑**: 价格创新高/新低，但动能(MACD)没有跟上，趋势可能反转。

### 3.2 顶背离 (看跌)

```
价格走势:     高点1        高点2 (更高)
              │            │
              │            │
              
MACD走势:     高点A        高点B (更低)
              │            │
              └────────────┘
              
信号: 🔴 价格涨不动了，可能见顶
操作: 考虑减仓或卖出
```

**识别要点**:
- 价格创近期新高
- MACD没有创新高，反而走低
- 通常发生在上涨趋势末端

### 3.3 底背离 (看涨)

```
价格走势:     低点1        低点2 (更低)
              │            │
              │            │
              
MACD走势:     低点A        低点B (更高)
              │            │
              └────────────┘
              
信号: 🟢 价格跌不动了，可能见底
操作: 考虑建仓或买入
```

**识别要点**:
- 价格创近期新低
- MACD没有创新低，反而走高
- 通常发生在下跌趋势末端

### 3.4 背离可靠性

| 背离次数 | 可靠性 | 说明 |
|:---------|:-------|:-----|
| 单次背离 | ⭐⭐⭐ | 可能反转，需确认 |
| 双重背离 | ⭐⭐⭐⭐ | 更可能反转 |
| 三重背离 | ⭐⭐⭐⭐⭐ | 非常可能反转 |

**确认信号**:
- 背离 + K线反转形态 = 更可靠
- 背离 + 放量 = 更可靠
- 多周期同时背离 = 最可靠

## 4. 柱状图 (Histogram)

```
柱状图 = MACD线 - 信号线
```

**柱状图意义**:

| 状态 | 含义 | 操作 |
|:-----|:-----|:-----|
| 柱状图 > 0 | MACD > 信号，多头动能 | 持有多单 |
| 柱状图 < 0 | MACD < 信号，空头动能 | 持有空单 |
| 由负转正 | 动能转多 | 买入信号 |
| 由正转负 | 动能转空 | 卖出信号 |
| 正值缩小 | 多头动能减弱 | 警惕回调 |
| 负值缩小 | 空头动能减弱 | 关注反弹 |

## 5. MACD实战策略

### 策略1: 金叉买入法

**条件**:
1. MACD在零轴下方金叉 (底部区域)
2. K线出现锤子线或看涨形态
3. 成交量放大

**操作**: 买入
**止损**: 金叉低点下方
**止盈**: 前高或1:2盈亏比

### 策略2: 背离交易法

**条件**:
1. 价格创新高，MACD未新高 (顶背离)
2. 出现流星线或看跌吞没

**操作**: 卖出
**止损**: 新高上方
**止盈**: 下方支撑

### 策略3: 零轴趋势法

**条件**:
1. MACD持续在零轴上方
2. 每次回调到零轴附近金叉

**操作**: 逢低买入
**适用**: 趋势行情

判断以下是否为背离:

| 场景 | 价格 | MACD | 是否背离 | 类型 |
|:-----|:-----|:-----|:---------|:-----|
| 1 | 新高 | 未新高 | 是 | 顶背离 |
| 2 | 新低 | 未新低 | 是 | 底背离 |
| 3 | 新高 | 新高 | 否 | - |

### 练习3: 综合分析 (10分钟)

**场景**: 某股票日线数据
- 价格: 连续下跌后企稳
- K线: 出现锤子线
- MACD: 在-0.8位置金叉
- 成交量: 较前日放大50%

**问题**:
1. MACD信号是什么？ → **零轴下方金叉，强烈买入信号**
2. K线形态是什么？ → **锤子线，看涨反转**
3. 量价配合如何？ → **放量，确认信号**
4. 综合判断？ → **强烈买入信号，多因素共振**
