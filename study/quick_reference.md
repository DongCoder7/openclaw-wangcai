## ⏰ 多周期速查

### 三周期搭配

| 风格 | 趋势 | 结构 | 入场 |
|:-----|:-----|:-----|:-----|
| 日内 | 60分钟 | 15分钟 | 5分钟 |
| 短线 | 日线 | 60分钟 | 15分钟 |
| 波段 | 周线 | 日线 | 60分钟 |

### 分析流程

```
1. 日线定方向 → 只做大周期方向
2. 60分钟找结构 → 确定支撑阻力
3. 15分钟找入场 → K线+MACD+量能
```

### 共振强度

| 周期数 | 强度 | 仓位 |
|:-------|:----:|:-----|
| 三周期同向 | ⭐⭐⭐⭐⭐ | 重仓(60%) |
| 两周期同向 | ⭐⭐⭐ | 轻仓(30%) |
| 周期矛盾 | ⭐⭐ | 观望 |

## 🎯 买入检查清单

- [ ] 日线趋势向上 (价格在MA20上方)
- [ ] 日线MACD > 0
- [ ] 60分钟回调到位/企稳
- [ ] 15分钟出现看涨K线形态
- [ ] 15分钟MACD金叉
- [ ] 成交量放量 (量比>1.5)
- [ ] 风险收益比 ≥ 1:2

**满足5项以上**: 可以买入
**满足全部7项**: 重仓买入

---

## 🛑 卖出检查清单

- [ ] 日线趋势向下 (价格在MA20下方)
- [ ] 日线MACD < 0
- [ ] 60分钟反弹受阻
- [ ] 15分钟出现看跌K线形态
- [ ] 15分钟MACD死叉
- [ ] 成交量放量下跌

**满足4项以上**: 考虑卖出
**满足全部6项**: 果断卖出

## 💡 核心口诀

### K线
```
大阳线 → 强势上涨
大阴线 → 强势下跌
十字星 → 犹豫不决
锤子线 → 跌无可跌
流星线 → 涨到尽头
吞没形态 → 反转为王
```

### MACD
```
零上金叉强，零下金叉更强
零上死叉狠，零下死叉更狠
顶背离看跌，底背离看涨
```

### 量价
```
量价齐升是真涨
量价背离要警惕
放量突破可跟进
缩量上涨有风险
```

### 多周期
```
大周期定方向
中周期找结构
小周期找入场
多周期共振信号强
```
