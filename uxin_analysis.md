# 优迅股份（688807.SH）深度分析 - 重要澄清

**分析时间**: 2026-04-18  
**核心结论**: **优迅股份是光通信"电芯片"龙头，但不是DSP芯片公司！**

## 一、重要概念澄清

### 1.1 光通信电芯片分类
| 芯片类型 | 功能 | 代表厂商 | 价值占比 |
|:---|:---|:---|:---:|
| **DSP芯片（数字信号处理）** | 信号补偿、编码解码、PAM4算法处理 | Broadcom、Marvell | 30-40% |
| **TIA（跨阻放大器）** | 将光电流信号转换为电压信号并放大 | 优迅股份、Semtech | 5-8% |
| **LDD（激光驱动器）** | 驱动激光器，控制发光功率 | 优迅股份、Semtech | 3-5% |
| **LA（限幅放大器）** | 信号整形，保证幅度稳定 | 优迅股份 | 2-3% |
| **CDR（时钟数据恢复）** | 提取时钟信号，重定时整形 | 优迅股份 | 3-5% |
| **收发合一芯片** | 集成LDD+LA+CDR+DDM等功能 | 优迅股份 | 10-15% |

### 1.2 关键区别
> **DSP芯片** = 数字信号处理器（算法处理核心，技术壁垒最高）  
> **电芯片（TIA/LDD/LA/CDR）** = 模拟/混合信号芯片（光电转换辅助）

**优迅股份做的是后者（电芯片），不是DSP芯片！**
